# Profile Settings

Customize your Bayit+ experience with profile settings.

## Managing Profiles

### Creating a Profile

1. Open Bayit+ app
2. Tap your profile icon
3. Tap **Add Profile**
4. Enter profile name
5. Choose an avatar
6. Set as Kids profile if needed
7. Save

### Editing a Profile

1. Tap profile icon
2. Tap **Manage Profiles**
3. Select profile to edit
4. Modify name, avatar, or settings
5. Save changes

### Deleting a Profile

1. Go to **Manage Profiles**
2. Select profile to delete
3. Tap **Delete Profile**
4. Confirm deletion

**Note:** Deleting a profile removes all preferences and watch history.

### Profile Limits

| Plan | Max Profiles |
|------|--------------|
| Basic | 2 |
| Standard | 4 |
| Premium | 5 |
| Family | 6 |

## Profile Customization

### Avatar Options

Choose from:
- Pre-set character avatars
- Color backgrounds
- Custom image upload (Premium)
- Animated avatars (Premium)

### Display Name

- Maximum 20 characters
- Special characters allowed
- Emojis supported
- Visible to family members

### Language Settings

Set per-profile language:

1. Open profile settings
2. Tap **Language**
3. Choose:
   - **App Language**: Interface language
   - **Content Language**: Default audio/subtitles
4. Save

Available: English, Hebrew, Spanish

## Preferences

### Content Preferences

Help us recommend better content:

1. Go to **Profile** > **Preferences**
2. Select favorite genres
3. Rate content you've watched
4. Mark content as "not interested"

### Maturity Ratings

Control content visibility:

1. Go to **Profile** > **Maturity Settings**
2. Set maximum rating:
   - Kids (G, TV-Y)
   - Teens (PG-13, TV-14)
   - Adults (R, TV-MA)
   - All content
3. Enter PIN to confirm

### Autoplay Settings

| Setting | Description |
|---------|-------------|
| Preview Autoplay | Videos autoplay while browsing |
| Next Episode | Automatically play next episode |
| Skip Intro | Auto-skip TV show intros |
| Credits Exit | Exit player at credits |

## Kids Profiles

### Creating a Kids Profile

1. Add new profile
2. Toggle **Kids Profile** on
3. Select age range
4. Choose avatar
5. Save

### Kids Profile Features

- Age-appropriate content only
- Simplified interface
- No access to settings
- No payment options
- Safe search enabled

### PIN Protection

Protect mature profiles:

1. Go to adult profile settings
2. Tap **Profile Lock**
3. Set 4-digit PIN
4. Required to enter profile

## Notifications

### Notification Types

Control which notifications you receive:

| Type | Description |
|------|-------------|
| New Releases | When new content arrives |
| Continue Watching | Reminders to resume |
| Recommendations | Personalized suggestions |
| Expiring Soon | Downloads about to expire |
| Account | Important account updates |

### Managing Notifications

1. Go to **Profile** > **Notifications**
2. Toggle each notification type
3. Set quiet hours if desired
4. Choose email vs push

## Privacy Settings

### Watch History

- View your complete history
- Remove individual items
- Clear all history
- Pause history (private mode)

### Data & Privacy

1. **Download Your Data**: Request all data
2. **Delete Account**: Permanent deletion
3. **Ad Personalization**: Toggle targeted ads
4. **Analytics**: Opt out of analytics

### Viewing Activity

View devices and sessions:

1. Go to **Settings** > **Security**
2. Tap **Recent Activity**
3. See all logged-in devices
4. Sign out specific devices
5. Sign out all devices

## Syncing Across Devices

### What Syncs

- Watch history
- My List
- Preferences
- Continue watching position
- Ratings and reviews

### Sync Issues

If not syncing:
1. Check internet connection
2. Verify same account on all devices
3. Force refresh (pull down)
4. Log out and back in

## Account Settings

### Email & Password

1. Go to **Settings** > **Account**
2. Update email address
3. Change password
4. Verify via email

### Phone Number

Add for account recovery:
1. Go to **Settings** > **Account**
2. Tap **Phone Number**
3. Enter and verify via SMS

### Linked Accounts

Manage social logins:
- Google
- Apple
- Facebook

Add or remove linked accounts in Security settings.

## Export Your Data

### Requesting Data Export

1. Go to **Settings** > **Privacy**
2. Tap **Download Your Data**
3. Confirm request
4. Receive email when ready
5. Download within 7 days

### Data Included

- Account information
- Watch history
- Preferences
- Payment history
- Support interactions
