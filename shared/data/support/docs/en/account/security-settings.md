# Security Settings

Protect your Bayit+ account with password management, two-factor authentication, and login monitoring.

## Password Management

### Changing Your Password

1. Go to **Settings** > **Account** > **Security**
2. Tap **Change Password**
3. Enter your current password
4. Enter your new password twice
5. Tap **Save Changes**

### Password Requirements

Your password must include:
- At least 8 characters
- One uppercase letter (A-Z)
- One lowercase letter (a-z)
- One number (0-9)
- One special character (!@#$%^&*)

### Password Best Practices

- Use a unique password not used elsewhere
- Consider using a password manager
- Never share your password with anyone
- Change your password every 6 months

## Two-Factor Authentication

### Enabling 2FA

1. Go to **Settings** > **Account** > **Security**
2. Tap **Two-Factor Authentication**
3. Choose your method:
   - **Authenticator App** (recommended)
   - **SMS Code**
4. Follow the setup instructions
5. Save your backup codes securely

### Using an Authenticator App

1. Download Google Authenticator or Authy
2. Scan the QR code shown in Bayit+
3. Enter the 6-digit code to verify
4. Store backup codes in a safe place

### Backup Codes

When you enable 2FA, you receive 10 backup codes:
- Each code can only be used once
- Store them securely offline
- Use them if you lose access to your 2FA device
- Generate new codes in Security settings if needed

## Login History

### Viewing Recent Logins

1. Go to **Settings** > **Account** > **Security**
2. Tap **Login History**
3. View all recent sign-ins with:
   - Date and time
   - Device type
   - Location (approximate)
   - IP address

### Identifying Suspicious Activity

Look for logins that:
- Occurred from unknown locations
- Used unfamiliar devices
- Happened at unusual times

### Responding to Unauthorized Access

1. Change your password immediately
2. Enable two-factor authentication
3. Sign out all devices (see below)
4. Review connected applications
5. Contact support if concerned

## Signing Out All Devices

### How to Sign Out Everywhere

1. Go to **Settings** > **Account** > **Security**
2. Tap **Sign Out All Devices**
3. Confirm your password
4. All sessions are terminated

This action:
- Signs out every device except your current one
- Requires re-authentication on all devices
- Does not affect your downloads or preferences

## Security Notifications

### Managing Alerts

1. Go to **Settings** > **Account** > **Security**
2. Tap **Security Notifications**
3. Choose which alerts to receive:
   - New device sign-ins
   - Password changes
   - 2FA changes
   - Subscription changes

### Notification Delivery

Security alerts are sent via:
- Email (always enabled)
- Push notification (optional)
- In-app notification (optional)

## Account Recovery

### Setting Up Recovery Options

1. Go to **Settings** > **Account** > **Security**
2. Tap **Recovery Options**
3. Add a recovery email address
4. Add a recovery phone number
5. Verify each method

### If You Lose Access

1. Use password reset via email
2. Use backup codes for 2FA
3. Contact support with identity verification
