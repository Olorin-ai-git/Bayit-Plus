# Community Directory

The Community Directory helps you find Jewish resources wherever you are, from synagogues to kosher restaurants.

## Finding Synagogues

Search for synagogues in your area:

1. Open Judaism section
2. Select Community Directory
3. Choose Synagogues
4. Browse results by location

Each listing includes address, denomination, service times, and contact information.

## Synagogue Filters

Refine your synagogue search:

- Filter by denomination (Orthodox, Conservative, Reform)
- Filter by nusach (Ashkenaz, Sefard, Edot HaMizrach)
- Show only those with daily minyanim
- Display wheelchair accessible locations
- Find those with youth programs

## Kosher Restaurants

Discover kosher dining options:

1. Select Restaurants in the directory
2. View nearby kosher establishments
3. Filter by cuisine type
4. Check kosher certification details

Listings include menus, hours, kashrut supervision, and user reviews.

## Jewish Organizations

Find community organizations:

- Jewish community centers
- Day schools and yeshivot
- Chesed organizations
- Eruv information
- Mikvah locations

## Travel Mode

When traveling, the directory adjusts automatically:

- Enable location services for automatic updates
- Search any city before your trip
- Save favorite locations for future visits
- Download directory data for offline access

## Adding Listings

Help expand the directory:

1. Tap Add Listing
2. Enter the establishment details
3. Include contact and kashrut information
4. Submit for verification

Community contributions are reviewed before publication.

## Saving Favorites

Save frequently visited locations:

- Tap the heart icon on any listing
- Access favorites from your profile
- Receive updates when information changes
- Share favorites with family and friends
