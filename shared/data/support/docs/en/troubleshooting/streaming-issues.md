# Troubleshooting Streaming Issues

Having trouble with playback? Follow these steps to resolve common streaming problems.

## Quick Fixes

Try these first before diving deeper:

1. **Check Internet Connection** - Ensure you're connected
2. **Restart the App** - Close completely and reopen
3. **Restart Your Device** - A fresh start often helps
4. **Update the App** - Get the latest version
5. **Try Different Content** - Check if issue is specific to one title

## Buffering Issues

### Symptoms
- Video pauses frequently
- "Loading" spinner appears
- Quality drops suddenly

### Solutions

#### Check Your Speed
Minimum requirements:
| Quality | Speed Required |
|---------|----------------|
| SD (480p) | 3 Mbps |
| HD (720p) | 5 Mbps |
| Full HD (1080p) | 10 Mbps |
| 4K Ultra HD | 25 Mbps |

Test your speed at speedtest.net.

#### Improve Connection
1. **Use Wi-Fi** - More stable than mobile data
2. **Move Closer** - Get nearer to your router
3. **Reduce Interference** - Away from microwaves, cordless phones
4. **Use 5GHz** - Faster than 2.4GHz if available
5. **Wired Connection** - Ethernet is most stable (for TV)

#### Reduce Quality
1. Go to **Settings** > **Playback**
2. Set quality to **Auto** or **Medium**
3. Disable HDR if enabled
4. Try again

#### Close Background Apps
Other apps may be using bandwidth:
1. Close browsers with streaming video
2. Pause downloads
3. Stop cloud backups temporarily
4. Disconnect other devices from network

## Video Won't Start

### Symptoms
- Black screen
- Endless loading
- Error messages

### Solutions

#### Check Your Subscription
1. Go to **Settings** > **Account**
2. Verify subscription is active
3. Check payment method is valid
4. Renew if expired

#### Clear App Cache
**iOS:**
1. Go to Settings > General > iPhone Storage
2. Find Bayit+ and tap
3. Tap "Offload App" then reinstall

**Android:**
1. Go to Settings > Apps > Bayit+
2. Tap Storage
3. Tap "Clear Cache"

**Web:**
1. Open browser settings
2. Clear cache and cookies
3. Restart browser

#### Check Device Compatibility
Ensure your device meets requirements:
- iOS 14.0 or later
- Android 8.0 or later
- tvOS 14.0 or later
- Chrome 90+, Safari 14+, Firefox 90+

## Poor Video Quality

### Symptoms
- Blurry or pixelated video
- Resolution lower than expected
- Quality doesn't improve over time

### Solutions

#### Manual Quality Selection
1. During playback, tap **Settings** icon
2. Select **Quality**
3. Choose **High** or **Ultra**
4. Wait for quality to improve

#### Check Display Settings
1. Ensure device resolution is set correctly
2. Enable HDR if supported
3. Check HDMI cable (for TV)
4. Update display drivers (for web)

#### Verify Content Quality
- Not all content is available in 4K
- Check title details for available quality
- Older content may be SD only

## Audio Problems

### No Sound
1. Check device volume
2. Check app volume
3. Try different content
4. Check mute isn't on
5. Check audio output device

### Audio Out of Sync
1. Pause and resume playback
2. Seek back a few seconds
3. Restart the app
4. Lower video quality

### Wrong Audio Track
1. Tap audio settings during playback
2. Select correct language
3. Check for descriptive audio setting

## Live TV Specific Issues

### Channel Won't Load
1. Check if channel is in your package
2. Try a different channel
3. Check for service outages
4. Refresh the channel guide

### Channel Freezing
1. Lower quality settings
2. Check internet stability
3. Try reloading the channel
4. Report ongoing issues

### Guide Not Updating
1. Pull down to refresh
2. Clear app cache
3. Check timezone settings
4. Reinstall app if persistent

## Download Issues

### Download Fails
1. Check storage space
2. Check internet connection
3. Verify content is downloadable
4. Check subscription allows downloads

### Download Won't Play
1. Check if download expired
2. Verify you're logged in
3. Re-download the content
4. Check device date/time

## Error Codes

| Code | Meaning | Solution |
|------|---------|----------|
| E001 | Network error | Check internet |
| E002 | Authentication failed | Re-login |
| E003 | Content unavailable | Try later |
| E004 | Device limit reached | Remove a device |
| E005 | Geo-restriction | Check your location |
| E006 | Payment required | Update billing |

## Still Having Issues?

If problems persist after trying these solutions:

1. **Check Service Status** - Visit our status page
2. **Use Voice Support** - Say "Hey Bayit, I need help"
3. **Create a Ticket** - Go to Support > Contact
4. **Include Details**:
   - Device type and model
   - App version
   - Internet speed
   - Error messages
   - Steps to reproduce

Our support team typically responds within 24 hours.
