# Error Codes Reference

This guide explains common Bayit+ error codes and their solutions.

## Network Errors (E001-E019)

### E001 - Network Connection Error

**Cause:** Unable to connect to Bayit+ servers.

**Solutions:**
1. Check your internet connection
2. Restart your router or modem
3. Try switching between Wi-Fi and mobile data
4. Disable VPN if active
5. Try again in a few minutes

### E002 - Connection Timeout

**Cause:** Server response took too long.

**Solutions:**
1. Check internet speed (minimum 3 Mbps required)
2. Move closer to your Wi-Fi router
3. Close other bandwidth-heavy applications
4. Restart the Bayit+ app
5. Try a wired connection if possible

### E003 - DNS Resolution Failed

**Cause:** Cannot resolve server address.

**Solutions:**
1. Restart your device
2. Reset network settings
3. Try using Google DNS (8.8.8.8) or Cloudflare (1.1.1.1)
4. Contact your internet provider if persistent

## Authentication Errors (E020-E039)

### E020 - Authentication Failed

**Cause:** Login credentials are incorrect or expired.

**Solutions:**
1. Verify email and password
2. Use "Forgot Password" to reset
3. Check Caps Lock is not on
4. Try logging in on the web

### E021 - Session Expired

**Cause:** Your login session has timed out.

**Solutions:**
1. Log in again with your credentials
2. Enable "Remember Me" for longer sessions
3. Check device date and time settings

### E022 - Account Locked

**Cause:** Too many failed login attempts.

**Solutions:**
1. Wait 30 minutes before trying again
2. Use "Forgot Password" to reset
3. Contact support if you did not attempt these logins

### E023 - Two-Factor Required

**Cause:** 2FA code needed but not provided.

**Solutions:**
1. Enter the code from your authenticator app
2. Check your SMS for the code
3. Use a backup code if needed
4. Contact support if locked out

## Subscription Errors (E040-E059)

### E040 - Subscription Required

**Cause:** No active subscription found.

**Solutions:**
1. Check subscription status in Account settings
2. Verify payment method is valid
3. Reactivate subscription if expired
4. Log out and log back in to refresh

### E041 - Payment Failed

**Cause:** Subscription payment could not be processed.

**Solutions:**
1. Update payment method
2. Check card has sufficient funds
3. Contact your bank if card is declined
4. Try alternative payment method

### E042 - Plan Limit Reached

**Cause:** Simultaneous stream limit exceeded.

**Solutions:**
1. Stop playback on another device
2. Check Settings > Devices for active streams
3. Upgrade plan for more simultaneous streams

## Content Errors (E060-E079)

### E060 - Content Unavailable

**Cause:** Requested content cannot be played.

**Solutions:**
1. Content may have been removed
2. Licensing may have changed
3. Check if content is available in your region
4. Try a different title

### E061 - Geo-Restriction

**Cause:** Content not available in your location.

**Solutions:**
1. Some content is region-locked
2. Check content availability for your region
3. Disable VPN if using one
4. Contact support for availability questions

### E062 - Playback Error

**Cause:** Video cannot be played.

**Solutions:**
1. Restart the app
2. Clear app cache
3. Update to latest app version
4. Try a different quality setting
5. Restart your device

### E063 - Download Not Permitted

**Cause:** Content cannot be downloaded.

**Solutions:**
1. Verify your plan supports downloads
2. Check if specific content allows downloads
3. Some content has download restrictions

## Device Errors (E080-E099)

### E080 - Device Limit Exceeded

**Cause:** Maximum registered devices reached.

**Solutions:**
1. Go to Settings > Devices
2. Remove devices you no longer use
3. Upgrade plan for more device slots

### E081 - Device Not Supported

**Cause:** Your device does not meet requirements.

**Solutions:**
1. Update device operating system
2. Check minimum requirements
3. Try using a supported device
4. Update the Bayit+ app

### E082 - HDCP Error

**Cause:** HDCP handshake failed with display.

**Solutions:**
1. Check HDMI cable connection
2. Try a different HDMI port
3. Use a certified HDMI cable
4. Restart both device and TV
5. Disable any HDMI splitters

### E083 - DRM Error

**Cause:** Digital rights management verification failed.

**Solutions:**
1. Update device software
2. Update Bayit+ app
3. Clear app cache
4. Restart device
5. Reinstall app if persistent

## Server Errors (E090-E099)

### E090 - Server Error

**Cause:** Problem with Bayit+ servers.

**Solutions:**
1. Check service status page
2. Wait a few minutes and retry
3. Server issues are usually resolved quickly

### E091 - Maintenance Mode

**Cause:** Bayit+ is undergoing scheduled maintenance.

**Solutions:**
1. Wait for maintenance to complete
2. Check status page for estimated completion
3. Follow social media for updates

### E099 - Unknown Error

**Cause:** An unexpected error occurred.

**Solutions:**
1. Restart the app
2. Update to latest version
3. Restart device
4. Contact support with error details

## Reporting Unresolved Errors

If your error persists:
1. Note the exact error code
2. Take a screenshot if possible
3. Go to **Settings** > **Help** > **Report Issue**
4. Include: error code, steps taken, device info
5. Support typically responds within 24 hours
