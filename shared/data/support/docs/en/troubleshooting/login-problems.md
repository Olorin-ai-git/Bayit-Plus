# Troubleshooting Login Problems

Can't access your account? Follow these steps to resolve login issues.

## Common Login Issues

### Forgot Password

1. Tap **Forgot Password** on login screen
2. Enter your email address
3. Check your inbox (and spam folder)
4. Click the reset link within 24 hours
5. Create a new password
6. Log in with new password

**Tips:**
- Password must be at least 8 characters
- Include uppercase, lowercase, and numbers
- Avoid common passwords
- Don't reuse passwords from other sites

### Wrong Password

If you're sure of your password:

1. Check Caps Lock is off
2. Check keyboard language
3. Try typing password in a text field first
4. Clear password field and retype slowly
5. Use password manager if available

### Account Locked

After multiple failed attempts, accounts are temporarily locked:

1. Wait 30 minutes
2. Try again with correct password
3. Use "Forgot Password" if unsure
4. Contact support if still locked

### Email Not Recognized

If "email not found" appears:

1. Check for typos in email
2. Try alternative email addresses
3. Check if you signed up with phone number
4. Try social login (Google, Apple)
5. Create a new account if needed

## Two-Factor Authentication

### Can't Access 2FA Code

**Authenticator App:**
1. Check correct time on phone
2. Look for Bayit+ entry in app
3. Use backup codes if available
4. Contact support with ID verification

**SMS Code:**
1. Check phone has signal
2. Wait up to 5 minutes
3. Tap "Resend Code"
4. Check all SMS inboxes
5. Try voice call option

### Lost Access to 2FA Device

1. Use one of your backup codes
2. Contact support with:
   - Account email
   - Last 4 digits of payment method
   - Recent viewing history
   - Government ID (if required)

### Disable 2FA

If you have access:
1. Go to **Settings** > **Security**
2. Tap **Two-Factor Authentication**
3. Enter current 2FA code
4. Confirm disable

## Social Login Issues

### Google Sign-In Not Working

1. Ensure Google account is active
2. Check Google app is updated
3. Try in web browser
4. Clear Google app cache
5. Re-authenticate with Google

### Apple Sign-In Not Working

1. Check Apple ID is active
2. Ensure Face ID/Touch ID works
3. Try password instead
4. Check iCloud settings
5. Sign in via web

### Connected Wrong Account

1. Log out completely
2. Clear app cache
3. Log in with correct method
4. Contact support if stuck

## Device Issues

### Session Expired

1. Normal security feature
2. Simply log in again
3. Enable "Remember Me"
4. Check device date/time

### Too Many Devices

You've reached your device limit:

1. Go to **Settings** > **Devices**
2. Remove devices you don't use
3. Or upgrade your plan for more

### Device Not Recognized

After reinstall or reset:
1. Log in normally
2. Verify via email if prompted
3. Complete device verification
4. This is a security feature

## Network Issues

### Login Times Out

1. Check internet connection
2. Try different network
3. Disable VPN temporarily
4. Try mobile data vs Wi-Fi
5. Wait and retry

### "Something Went Wrong"

1. Check service status page
2. Wait a few minutes
3. Restart the app
4. Check for app updates
5. Reinstall if persistent

## Account Security

### Suspicious Activity

If you notice unfamiliar activity:

1. Change password immediately
2. Enable two-factor authentication
3. Log out all devices
4. Review account activity
5. Contact support

### Account Compromised

1. Reset password immediately
2. Use unique, strong password
3. Check email for unauthorized changes
4. Review connected apps
5. Enable 2FA
6. Contact support

## Profile Issues

### Can't Switch Profiles

1. Ensure all profiles are set up
2. Check profile PIN if set
3. Close and reopen app
4. Log out and back in

### Profile Deleted

1. Profiles can't be recovered
2. Create a new profile
3. Preferences won't be restored
4. Watch history is lost

## Subscription Issues

### Subscription Not Showing

After successful payment:
1. Log out and back in
2. Wait 15 minutes for sync
3. Check payment confirmation email
4. Verify in app store subscriptions
5. Contact support with receipt

### Wrong Subscription Tier

1. Check account settings
2. Verify payment amount
3. Contact billing support
4. May take time to sync

## Contact Support

If you still can't log in:

1. **Use Voice Support** (from login screen)
   - Say "Hey Bayit, I can't log in"

2. **Email Support**
   - Include: email, device type, error messages
   - Expected response: 24 hours

3. **Create Ticket**
   - Access via web: bayitplus.com/support
   - No login required for login issues

**Information to Include:**
- Email address used
- Device type and model
- App version
- Screenshot of error
- Steps you've tried
