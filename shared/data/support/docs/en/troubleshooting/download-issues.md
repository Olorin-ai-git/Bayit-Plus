# Troubleshooting Download Issues

Having trouble downloading content for offline viewing? This guide helps resolve download failures, storage issues, and expired downloads.

## Download Failures

### Download Will Not Start

1. **Check internet connection** - Downloads require stable connection
2. **Verify subscription** - Your plan must support downloads
3. **Check content rights** - Not all titles are downloadable
4. **Restart the app** - Close completely and reopen

### Checking Download Availability

1. Go to the content details page
2. Look for the **Download** button
3. If grayed out, content is not downloadable
4. Some titles have regional restrictions

### Subscription Requirements

| Plan | Downloads Allowed |
|------|-------------------|
| Basic | No downloads |
| Standard | No downloads |
| Premium | Unlimited downloads |
| Family | Unlimited downloads |

Upgrade your plan if downloads are not available.

### Download Interrupted

If a download stops mid-progress:
1. Check Wi-Fi connection stability
2. Tap the download to resume
3. If stuck, cancel and restart
4. Move to stronger signal area

## Storage Issues

### Insufficient Storage Error

If you see "Not enough storage":

**iOS:**
1. Go to Settings > General > iPhone Storage
2. Review storage usage
3. Delete unused apps or content
4. Clear Bayit+ cache if needed

**Android:**
1. Go to Settings > Storage
2. Review storage breakdown
3. Clear cached data
4. Remove unused files or apps

### Checking Download Storage

1. Go to **Settings** > **Downloads**
2. View **Storage Used** by Bayit+
3. See breakdown by content type
4. Delete content to free space

### Managing Downloaded Content

1. Go to **Downloads** in the app
2. Review all downloaded titles
3. Swipe or tap to delete individual titles
4. Use **Delete All** to clear everything

### Download Quality and Size

Higher quality means larger file size:

| Quality | Approximate Size (per hour) |
|---------|----------------------------|
| Low | 300 MB |
| Medium | 700 MB |
| High | 1.5 GB |
| Ultra | 3 GB |

### Changing Download Quality

1. Go to **Settings** > **Downloads**
2. Select **Download Quality**
3. Choose preferred quality level
4. Lower quality saves storage space

### Download Location (Android)

Move downloads to SD card:
1. Go to **Settings** > **Downloads**
2. Tap **Download Location**
3. Select **SD Card** if available
4. New downloads save to SD card

## Expired Downloads

### Understanding Download Expiration

Downloaded content expires after:
- **30 days** from download date
- **48 hours** after first playback
- Or when subscription ends

### Checking Expiration Status

1. Go to **Downloads**
2. View expiration date under each title
3. Titles show "Expires in X days"
4. Expired content is removed automatically

### Renewing Expired Downloads

To watch expired content again:
1. Ensure internet connection
2. Delete the expired download
3. Re-download the content
4. New expiration period begins

### Downloads After Subscription Ends

If your subscription expires:
- All downloads become unplayable
- Content remains on device temporarily
- Reactivate subscription to restore access
- Or downloads will be automatically deleted

## Download Limits

### Maximum Downloads

Limits depend on your plan:
- **Premium:** 25 titles per device
- **Family:** 25 titles per device, per profile

### Device Limits for Downloads

Downloads are tied to specific devices:
- Cannot transfer downloads between devices
- Each device has its own limit
- Removing a device deletes its downloads

### Reaching Download Limit

If you cannot download more:
1. Go to **Downloads**
2. Delete titles you have watched
3. Delete titles you no longer need
4. Try downloading again

## Download Not Playing

### Downloaded Content Will Not Play

1. **Check expiration** - Content may have expired
2. **Verify account** - Ensure you are logged in
3. **Check date/time** - Device date must be accurate
4. **Restart app** - Close and reopen Bayit+

### Playback Error on Downloaded Content

1. Delete the problematic download
2. Re-download the content
3. Ensure sufficient storage
4. Try a different quality setting

### Wrong Device Date/Time

Downloads verify against device time:
1. Go to device Settings > Date & Time
2. Enable **Set Automatically**
3. Restart the Bayit+ app
4. Try playing content again

## Wi-Fi Only Downloads

### Enabling Wi-Fi Only Mode

To prevent mobile data usage:
1. Go to **Settings** > **Downloads**
2. Enable **Wi-Fi Only**
3. Downloads only proceed on Wi-Fi
4. Saves mobile data allowance

### Downloading on Mobile Data

If you need to download on mobile:
1. Go to **Settings** > **Downloads**
2. Disable **Wi-Fi Only**
3. Be aware of data usage
4. Lower quality to reduce data consumption

## Contact Support

For persistent download issues:
1. **In-App:** Settings > Help > Download Problem
2. **Voice:** "Hey Bayit, I cannot download"
3. Include: device type, content title, and error message
