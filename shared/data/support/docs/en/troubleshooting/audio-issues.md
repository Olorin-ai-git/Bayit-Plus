# Troubleshooting Audio Issues

Experiencing sound problems? Follow these steps to diagnose and fix audio issues on Bayit+.

## No Sound

### Quick Checks

1. **Check device volume** - Ensure it is not muted or at zero
2. **Check app volume** - In-player volume control may be lowered
3. **Check physical connections** - Headphones, speakers, or HDMI cables
4. **Try different content** - Verify the issue affects multiple titles

### Device-Specific Solutions

**Mobile (iOS/Android):**
1. Check silent mode is off
2. Press volume up button
3. Check Bluetooth audio is not connected elsewhere
4. Restart the app

**Smart TV:**
1. Check TV volume and mute status
2. Verify correct audio output (TV speakers vs soundbar)
3. Check HDMI-CEC audio settings
4. Test other apps for sound

**Web Browser:**
1. Check browser tab is not muted
2. Click inside the video player
3. Check system volume mixer
4. Try a different browser

### Audio Output Settings

1. Go to **Settings** > **Playback** > **Audio Output**
2. Verify correct output device is selected
3. Test with **Audio Test** button
4. Switch between stereo and surround options

## Audio Out of Sync

### Symptoms

- Voices do not match lip movements
- Sound arrives before or after video
- Delay increases over time

### Solutions

#### Quick Fixes
1. Pause and resume playback
2. Seek backward 10-15 seconds
3. Restart the app completely
4. Try a different title

#### Adjust Audio Delay

If your device supports audio sync adjustment:
1. During playback, tap **Settings** icon
2. Select **Audio Sync**
3. Adjust delay in milliseconds
4. Positive values delay audio
5. Negative values advance audio

#### Network-Related Sync Issues
1. Lower video quality to reduce buffering
2. Check your internet connection stability
3. Disconnect other devices using bandwidth
4. Try a wired connection if available

#### Device-Specific Fixes

**Smart TV with Soundbar:**
1. Check soundbar audio delay settings
2. Disable audio processing features
3. Use HDMI ARC connection if available
4. Update soundbar firmware

**Bluetooth Headphones:**
1. Bluetooth introduces natural delay
2. Use wired headphones for perfect sync
3. Check for low-latency codec support
4. Keep device close to Bluetooth source

## Surround Sound Problems

### Surround Sound Not Working

1. Go to **Settings** > **Playback** > **Audio**
2. Check **Surround Sound** is enabled
3. Verify your equipment supports surround
4. Check content includes surround audio track

### Verifying Equipment Support

| Audio Format | Requirement |
|--------------|-------------|
| Dolby Digital 5.1 | Compatible receiver or soundbar |
| Dolby Atmos | Atmos-enabled equipment |
| DTS | DTS-compatible receiver |

### Configuring Surround Sound

1. Ensure receiver is set to correct input
2. Check receiver decode mode (Auto recommended)
3. Verify all speaker connections
4. Run receiver's speaker calibration

### Content Availability

Not all content supports surround sound:
- Check title details for available audio formats
- Some titles only offer stereo
- Older content may have limited options

## Wrong Audio Track

### Changing Audio Language

1. During playback, tap **Audio** icon
2. Select preferred language
3. Audio switches immediately
4. Setting is saved for future playback

### Audio Description

If hearing narration of visual elements:
1. Tap **Audio** icon during playback
2. Disable **Audio Description**
3. Select standard audio track
4. Check accessibility settings

### Setting Default Audio

1. Go to **Settings** > **Playback** > **Audio**
2. Set **Preferred Audio Language**
3. Enable **Audio Description** default if needed
4. Settings apply to all future playback

## Crackling or Distorted Audio

### Causes and Solutions

1. **Lower volume** - Distortion at high levels
2. **Check connections** - Loose cables cause issues
3. **Update drivers** - For web browser playback
4. **Reduce quality** - Lower bandwidth helps

### Hardware Troubleshooting

1. Test with different headphones or speakers
2. Try a different audio output
3. Check for device software updates
4. Restart your audio equipment

## Contact Support

If audio issues persist:
1. **In-App:** Settings > Help > Audio Problem
2. **Voice:** "Hey Bayit, I have an audio problem"
3. Include: device type, content title, and issue description
