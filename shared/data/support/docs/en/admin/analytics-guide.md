# Analytics Guide

Bayit+ Analytics provides comprehensive insights into platform performance, user behavior, and content engagement. Use these tools to make data-driven decisions about your streaming service.

## Analytics Dashboard

Access analytics from the main navigation. The dashboard presents key metrics with customizable date ranges and comparison periods.

## Core Metrics

**Engagement Metrics**:
- Total hours watched
- Average session duration
- Content completion rates
- Peak viewing times

**User Metrics**:
- Daily and monthly active users
- New user signups
- Churn rate and retention
- Profile distribution

**Content Metrics**:
- Most watched titles
- Category performance
- Search trends
- Recommendation effectiveness

## Creating Custom Reports

Build reports tailored to your needs:

1. Navigate to **Analytics** > **Custom Reports**
2. Click **New Report**
3. Select metrics to include
4. Choose dimensions for grouping (date, category, device)
5. Apply filters to narrow the data
6. Save the report for future access

## Scheduled Reports

Automate report delivery:

1. Open any saved report
2. Click **Schedule**
3. Choose frequency (daily, weekly, monthly)
4. Enter recipient email addresses
5. Select format (PDF, CSV, Excel)

Reports are delivered automatically at the scheduled time.

## Data Export

Export raw data for external analysis:

| Export Type | Description |
|-------------|-------------|
| Viewing Data | Anonymized watch events |
| User Demographics | Aggregated user statistics |
| Content Performance | Title-level metrics |
| Search Queries | Trending search terms |

## Real-Time Analytics

Monitor live platform activity:

- Current concurrent viewers
- Active streams by content
- Geographic distribution
- Device type breakdown

Access real-time data from **Analytics** > **Real-Time**.

## Cohort Analysis

Track user groups over time to understand retention and behavior patterns. Create cohorts based on signup date, first content watched, or subscription tier.
