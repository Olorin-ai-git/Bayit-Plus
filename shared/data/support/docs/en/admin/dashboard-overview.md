# Admin Dashboard Overview

The Bayit+ Admin Dashboard provides a centralized hub for managing all aspects of your streaming platform. This guide covers navigation, key metrics, and essential features available to administrators.

## Accessing the Dashboard

Navigate to your organization's admin URL and sign in with your administrator credentials. Two-factor authentication is required for all admin accounts.

## Main Navigation

The left sidebar contains primary navigation sections:

- **Dashboard**: Overview and key metrics
- **Content**: Library management and uploads
- **Users**: Account and profile management
- **Analytics**: Reports and data exports
- **Campaigns**: Marketing and notifications
- **Support**: Ticket management
- **Settings**: Platform configuration

## Key Performance Metrics

The main dashboard displays real-time metrics:

| Metric | Description |
|--------|-------------|
| Active Users | Currently streaming viewers |
| Daily Active Users | Unique users in past 24 hours |
| Content Hours Watched | Total viewing time today |
| New Signups | Registrations in past 24 hours |
| Support Tickets | Open tickets requiring attention |

## Quick Actions

The dashboard header provides quick access to common tasks:

- **Add Content**: Start a new content upload
- **Create Campaign**: Launch a new notification campaign
- **View Reports**: Access analytics dashboard
- **System Status**: Check platform health

## Customizing Your View

Personalize your dashboard layout:

1. Click the **Customize** button in the top right
2. Drag widgets to rearrange their position
3. Toggle visibility for metrics you use most
4. Save your layout preferences

## Notification Center

The bell icon in the header shows recent system notifications including:

- Content processing completions
- User reports requiring review
- System alerts and maintenance notices
- Campaign performance milestones

## Keyboard Shortcuts

Navigate efficiently with keyboard shortcuts:

- **G + D**: Go to Dashboard
- **G + C**: Go to Content
- **G + U**: Go to Users
- **G + A**: Go to Analytics
- **?**: Show all shortcuts
