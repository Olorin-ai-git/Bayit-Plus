# Content Management

The Content Management system allows administrators to add, edit, organize, and remove media from the Bayit+ library. This guide covers the complete content lifecycle.

## Content Library Overview

Access the content library from **Content** > **Library**. The library displays all content with filtering options:

- **Type**: Movies, Series, Channels, Podcasts
- **Status**: Published, Draft, Processing, Archived
- **Rating**: Filter by age rating
- **Category**: Browse by genre or category

## Adding New Content

To add content to the library:

1. Navigate to **Content** > **Add New**
2. Select the content type (Movie, Series, Channel, Podcast)
3. Upload media files or provide streaming URLs
4. Complete the metadata form (title, description, cast, ratings)
5. Upload artwork (poster, backdrop, thumbnails)
6. Set availability and scheduling options
7. Submit for processing

## Editing Existing Content

Modify content details at any time:

1. Locate the content in the library
2. Click the **Edit** button
3. Update any metadata fields
4. Changes to non-media fields publish immediately
5. Media file changes trigger reprocessing

## Content Metadata

Required metadata for all content:

| Field | Description |
|-------|-------------|
| Title | Display name in all languages |
| Description | Synopsis shown on details page |
| Rating | Age rating (G, PG, PG-13, R) |
| Categories | Genre and topic classifications |
| Runtime | Duration in minutes |
| Release Year | Original release date |

## Scheduling Content

Control when content becomes available:

- **Immediate**: Publish as soon as processing completes
- **Scheduled**: Set a specific publish date and time
- **Windowed**: Define start and end availability dates

## Removing Content

To remove content from the platform:

1. Select the content in the library
2. Click **Actions** > **Archive**
3. Archived content is hidden but preserved
4. For permanent deletion, contact your system administrator

Archiving preserves viewing history and allows future restoration.
