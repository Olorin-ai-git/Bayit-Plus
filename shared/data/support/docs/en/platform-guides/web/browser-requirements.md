# Browser Requirements

Bayit+ is optimized for modern web browsers to deliver the best streaming experience with high-quality video playback and smooth navigation.

## Supported Browsers

| Browser | Minimum Version | Recommended Version |
|---------|-----------------|---------------------|
| Google Chrome | 90+ | Latest |
| Mozilla Firefox | 88+ | Latest |
| Apple Safari | 14+ | Latest |
| Microsoft Edge | 90+ | Latest |

## System Requirements

### Minimum Requirements
- **Processor:** Dual-core 2.0 GHz or equivalent
- **RAM:** 4 GB
- **Internet:** 5 Mbps for SD quality
- **Operating System:** Windows 10, macOS 10.14, or Ubuntu 20.04

### Recommended Requirements
- **Processor:** Quad-core 2.5 GHz or equivalent
- **RAM:** 8 GB or more
- **Internet:** 25 Mbps for 4K Ultra HD
- **Operating System:** Windows 11, macOS 12, or latest Ubuntu LTS

## Browser Settings

For optimal performance, ensure the following settings are enabled:

1. **JavaScript:** Must be enabled for the player to function
2. **Cookies:** Required for maintaining your session and preferences
3. **Hardware Acceleration:** Enable for smoother video playback
4. **Pop-up Blocker:** Add Bayit+ to your allowed sites list

## Troubleshooting Tips

- Clear your browser cache if you experience playback issues
- Disable browser extensions that may interfere with video streaming
- Ensure your browser is updated to the latest version
- Check that your graphics drivers are up to date

## Unsupported Browsers

Internet Explorer and older browser versions are not supported. For the best experience, please update to one of the supported browsers listed above.
