# User Profiles

Create personalized profiles for each member of your household. Each profile maintains its own preferences, watch history, and recommendations.

## Profile Limits by Plan

The number of profiles depends on your subscription:
- **Basic**: 1 profile
- **Standard**: Up to 3 profiles
- **Premium**: Up to 5 profiles
- **Family**: Up to 6 profiles

## Creating a Profile

1. Go to Settings and select "Manage Profiles"
2. Tap "Add Profile"
3. Enter a name for the profile
4. Choose an avatar from the available options
5. Set profile preferences (optional)
6. Save the new profile

## Profile Settings

Each profile can be customized with:

### Avatar
Select from a gallery of avatar images to personalize your profile icon.

### Language Preference
Set the preferred language for the interface and content recommendations.

### Maturity Rating
Control the maximum content rating visible on this profile.

### Autoplay
Enable or disable automatic playback of next episodes and previews.

### Subtitle Defaults
Set preferred subtitle language and display settings.

## Kids Profiles

Create dedicated profiles for children with enhanced safety features:
- Content limited to age-appropriate ratings
- Curated kids-only interface
- Restricted access to account settings
- Simplified navigation

To create a kids profile, toggle "Kids Profile" when setting up a new profile.

## Profile PIN

Protect profiles with a PIN code to prevent unauthorized access:
1. Select a profile from Manage Profiles
2. Enable "Profile Lock"
3. Create a 4-digit PIN
4. Confirm the PIN

## Switching Profiles

Change profiles at any time:
- Tap your profile icon in the app
- Select a different profile from the list
- Kids profiles may require parent PIN to exit

## Deleting a Profile

Remove profiles you no longer need:
1. Go to Manage Profiles
2. Select the profile to delete
3. Confirm deletion

The primary account profile cannot be deleted.
