# Electronic Program Guide

The Electronic Program Guide (EPG) shows you what is on live TV channels now and in the coming days. Plan your viewing and discover new content with the interactive guide.

## Accessing the EPG

1. Navigate to "Live TV" from the main menu
2. Select "Guide" or "Program Guide"
3. The full channel grid appears

## Navigating the Guide

### Channel List
Channels appear vertically on the left side. Scroll up and down to browse all available channels.

### Time Grid
Programs display horizontally in time slots. Scroll left and right to see past and upcoming shows.

### Date Selection
Use the date selector at the top to jump to a specific day. Most guides display up to 7 days of programming.

## Guide Features

### Now and Next
A quick view shows what is currently playing and coming up next on each channel.

### Program Details
Select any program in the grid to view:
- Title and episode information
- Synopsis and description
- Duration and air time
- Age rating
- Cast and crew (when available)

### Quick Actions
From the program details, you can:
- Watch live (if currently airing)
- Set a reminder
- Schedule a recording
- Add to watchlist

## Filtering Channels

Organize the guide to show only channels you care about:

### Favorite Channels
Mark channels as favorites to filter the guide to only your preferred channels.

### Channel Categories
Filter by channel type:
- Entertainment
- News
- Sports
- Kids
- Movies
- Music

## Reminders

Set reminders for upcoming programs:
1. Select a future program in the guide
2. Tap "Set Reminder"
3. Receive a notification when the program is about to start

Manage all reminders from the Settings menu.

## Recording from Guide

Schedule DVR recordings directly from the guide. Select any program and choose "Record" to save it for later viewing.
