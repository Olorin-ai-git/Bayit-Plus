# סקירת לוח הבקרה למנהלים

לוח הבקרה של Bayit+ מספק מרכז מאוחד לניהול כל ההיבטים של פלטפורמת הסטרימינג. מדריך זה מכסה ניווט, מדדים עיקריים ותכונות חיוניות הזמינות למנהלים.

## גישה ללוח הבקרה

נווטו לכתובת הניהול של הארגון שלכם והיכנסו עם פרטי המנהל. אימות דו-שלבי נדרש לכל חשבונות המנהלים.

## ניווט ראשי

סרגל הצד השמאלי מכיל את קטעי הניווט העיקריים:

- **לוח בקרה**: סקירה ומדדים עיקריים
- **תוכן**: ניהול ספרייה והעלאות
- **משתמשים**: ניהול חשבונות ופרופילים
- **אנליטיקה**: דוחות וייצוא נתונים
- **קמפיינים**: שיווק והתראות
- **תמיכה**: ניהול פניות
- **הגדרות**: תצורת הפלטפורמה

## מדדי ביצועים עיקריים

לוח הבקרה הראשי מציג מדדים בזמן אמת:

| מדד | תיאור |
|-----|-------|
| משתמשים פעילים | צופים בסטרימינג כעת |
| משתמשים פעילים יומיים | משתמשים ייחודיים ב-24 השעות האחרונות |
| שעות תוכן שנצפו | זמן צפייה כולל היום |
| הרשמות חדשות | רישומים ב-24 השעות האחרונות |
| פניות תמיכה | פניות פתוחות הדורשות טיפול |

## פעולות מהירות

כותרת לוח הבקרה מספקת גישה מהירה למשימות נפוצות:

- **הוסף תוכן**: התחל העלאת תוכן חדש
- **צור קמפיין**: השק קמפיין התראות חדש
- **צפה בדוחות**: גש ללוח האנליטיקה
- **סטטוס מערכת**: בדוק את תקינות הפלטפורמה

## התאמה אישית של התצוגה

התאימו אישית את פריסת לוח הבקרה:

1. לחצו על כפתור **התאמה אישית** בפינה הימנית העליונה
2. גררו ווידג'טים לסידור מחדש של מיקומם
3. הפעילו או כבו נראות למדדים שאתם משתמשים בהם ביותר
4. שמרו את העדפות הפריסה שלכם

## מרכז ההתראות

סמל הפעמון בכותרת מציג התראות מערכת אחרונות הכוללות:

- השלמות עיבוד תוכן
- דיווחי משתמשים הדורשים סקירה
- התראות מערכת והודעות תחזוקה
- אבני דרך בביצועי קמפיינים

## קיצורי מקלדת

נווטו ביעילות עם קיצורי מקלדת:

- **G + D**: עבור ללוח בקרה
- **G + C**: עבור לתוכן
- **G + U**: עבור למשתמשים
- **G + A**: עבור לאנליטיקה
- **?**: הצג את כל הקיצורים
