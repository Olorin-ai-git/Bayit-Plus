# נהלי העלאת וידאו

מדריך זה מכסה את תהליך העבודה המלא להעלאת תוכן וידאו ל-Bayit+, מהכנת קבצים דרך קידוד ועד להשלמת מטא-דאטה.

## הכנת קבצים

לפני ההעלאה, ודאו שקבצי המקור שלכם עומדים בדרישות הבאות:

| מפרט | דרישה |
|------|-------|
| פורמט | MP4, MOV, MKV או MXF |
| קודק וידאו | H.264, H.265 או ProRes |
| רזולוציה | מינימום 1920x1080 (עדיף 4K) |
| אודיו | AAC או PCM, סטריאו או 5.1 |
| קצב סיביות | מינימום 15 Mbps ל-HD |

## התחלת העלאה

1. נווטו אל **תוכן** > **העלאה**
2. גררו קבצים לאזור ההעלאה או לחצו לעיון
3. ניתן להעלות מספר קבצים בו-זמנית
4. קבצים גדולים תומכים בהעלאות ניתנות לחידוש

## התקדמות העלאה

עקבו אחר סטטוס ההעלאה מ-**תור ההעלאות**:

- **בהעלאה**: העברת קובץ בתהליך
- **בעיבוד**: קידוד ואריזה
- **מוכן**: זמין להזנת מטא-דאטה
- **פורסם**: פעיל בפלטפורמה

## תהליך הקידוד

לאחר ההעלאה, קבצים מקודדים אוטומטית:

- רמות איכות מרובות (SD, HD, 4K כאשר רלוונטי)
- אריזת HLS לסטרימינג אדפטיבי
- נרמול אודיו ותאימות עוצמה
- יצירת תמונות ממוזערות בפריימים מרכזיים

זמן העיבוד משתנה בהתאם למשך ואיכות המקור.

## הזנת מטא-דאטה

השלימו מטא-דאטה נדרשת לכל העלאה:

1. **מידע בסיסי**: כותרת, תיאור, שנת יציאה
2. **סיווג**: קטגוריות, ז'אנרים, תגיות
3. **דירוגים**: דירוג גיל ותיאורי תוכן
4. **קרדיטים**: שחקנים, צוות ופרטי הפקה
5. **גרפיקה**: תמונות פוסטר, רקע ולוגו

## דרישות גרפיקה

| סוג נכס | ממדים | פורמט |
|---------|-------|-------|
| פוסטר | 800x1200 פיקסלים | JPEG או PNG |
| רקע | 1920x1080 פיקסלים | JPEG או PNG |
| לוגו | 400x150 פיקסלים | PNG עם שקיפות |
| תמונה ממוזערת | 640x360 פיקסלים | JPEG |

## פרסום תוכן

לאחר השלמת המטא-דאטה:

1. לחצו על **תצוגה מקדימה** לצפייה בדף התוכן
2. ודאו שכל המידע מוצג נכון
3. בחרו תזמון פרסום (מיידי או מתוזמן)
4. לחצו על **פרסם** להפיכת התוכן לזמין

## העלאות מרובות

לספריות תוכן גדולות, השתמשו בתכונת ההעלאה המרובה:

1. הכינו קובץ CSV עם מטא-דאטה
2. העלו קבצי וידאו באמצעות SFTP
3. ייבאו את ה-CSV של המטא-דאטה
4. סקרו ופרסמו באצוות
