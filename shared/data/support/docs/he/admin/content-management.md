# ניהול תוכן

מערכת ניהול התוכן מאפשרת למנהלים להוסיף, לערוך, לארגן ולהסיר מדיה מספריית Bayit+. מדריך זה מכסה את מחזור החיים המלא של התוכן.

## סקירת ספריית התוכן

גשו לספריית התוכן מ-**תוכן** > **ספרייה**. הספרייה מציגה את כל התוכן עם אפשרויות סינון:

- **סוג**: סרטים, סדרות, ערוצים, פודקאסטים
- **סטטוס**: פורסם, טיוטה, בעיבוד, בארכיון
- **דירוג**: סינון לפי דירוג גיל
- **קטגוריה**: עיון לפי ז'אנר או קטגוריה

## הוספת תוכן חדש

להוספת תוכן לספרייה:

1. נווטו אל **תוכן** > **הוסף חדש**
2. בחרו את סוג התוכן (סרט, סדרה, ערוץ, פודקאסט)
3. העלו קבצי מדיה או ספקו כתובות URL לסטרימינג
4. מלאו את טופס המטא-דאטה (כותרת, תיאור, שחקנים, דירוגים)
5. העלו גרפיקה (פוסטר, רקע, תמונות ממוזערות)
6. הגדירו אפשרויות זמינות ותזמון
7. שלחו לעיבוד

## עריכת תוכן קיים

שנו פרטי תוכן בכל עת:

1. אתרו את התוכן בספרייה
2. לחצו על כפתור **עריכה**
3. עדכנו כל שדה מטא-דאטה
4. שינויים בשדות שאינם מדיה מתפרסמים מיד
5. שינויים בקבצי מדיה מפעילים עיבוד מחדש

## מטא-דאטה של תוכן

מטא-דאטה נדרשת לכל תוכן:

| שדה | תיאור |
|-----|-------|
| כותרת | שם תצוגה בכל השפות |
| תיאור | תקציר המוצג בדף הפרטים |
| דירוג | דירוג גיל (G, PG, PG-13, R) |
| קטגוריות | סיווגי ז'אנר ונושא |
| משך | אורך בדקות |
| שנת יציאה | תאריך יציאה מקורי |

## תזמון תוכן

שלטו מתי תוכן הופך לזמין:

- **מיידי**: פרסום ברגע שהעיבוד מסתיים
- **מתוזמן**: הגדירו תאריך ושעת פרסום ספציפיים
- **חלון זמן**: הגדירו תאריכי התחלה וסיום לזמינות

## הסרת תוכן

להסרת תוכן מהפלטפורמה:

1. בחרו את התוכן בספרייה
2. לחצו על **פעולות** > **ארכיון**
3. תוכן בארכיון מוסתר אך נשמר
4. למחיקה קבועה, פנו למנהל המערכת שלכם

ארכיון שומר על היסטוריית צפייה ומאפשר שחזור עתידי.
