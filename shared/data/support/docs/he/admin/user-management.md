# ניהול משתמשים

אזור ניהול המשתמשים מספק כלים לניהול חשבונות מנויים, תפקידי מנהלים והרשאות גישה ברחבי פלטפורמת Bayit+.

## ספריית משתמשים

גשו לספריית המשתמשים מ-**משתמשים** > **כל המשתמשים**. הספרייה תומכת ב:

- **חיפוש**: מצאו משתמשים לפי אימייל, שם או מזהה חשבון
- **מסננים**: פילוח לפי סטטוס מנוי, תאריך יצירה או פעילות
- **פעולות מרובות**: בחרו מספר משתמשים לפעולות אצווה
- **ייצוא**: הורידו רשימות משתמשים לניתוח חיצוני

## צפייה בפרטי משתמש

לחצו על כל משתמש לצפייה בפרופיל המלא שלו:

- מידע חשבון וסטטוס מנוי
- פרופילים שנוצרו תחת החשבון
- היסטוריית צפייה והעדפות
- היסטוריית פניות תמיכה
- מידע תשלום וחיוב

## פעולות חשבון

פעולות זמינות לחשבונות בודדים:

| פעולה | תיאור |
|-------|-------|
| איפוס סיסמה | שליחת אימייל לאיפוס סיסמה |
| השעיית חשבון | השבתת גישה זמנית |
| ביטול מנוי | סיום מנוי בתשלום |
| הענקת קרדיט | החלת קרדיט לחשבון |
| מיזוג חשבונות | איחוד חשבונות כפולים |

## תפקידי מנהלים

הקצו הרשאות מפורטות לחברי צוות הניהול:

- **מנהל ראשי**: גישה מלאה לפלטפורמה
- **מנהל תוכן**: ספריית תוכן והעלאות בלבד
- **נציג תמיכה**: חשבונות משתמשים ופניות תמיכה
- **מנתח**: אנליטיקה ודוחות (קריאה בלבד)
- **שיווק**: ניהול קמפיינים בלבד

## יצירת חשבונות מנהל

להוספת מנהל חדש:

1. נווטו אל **משתמשים** > **מנהלים**
2. לחצו על **הוסף מנהל**
3. הזינו כתובת אימייל ושם
4. בחרו את התפקיד המתאים
5. המנהל החדש מקבל אימייל הזמנה

## אבטחה ויומני ביקורת

כל פעולות הניהול נרשמות לאבטחה:

- שינויים בחשבונות משתמשים
- שינויי תוכן
- עדכוני הרשאות
- ניסיונות התחברות וסשנים

גשו ליומני ביקורת מ-**הגדרות** > **אבטחה** > **יומן ביקורת**.

## פעולות מרובות על משתמשים

בצעו פעולות על מספר משתמשים בו-זמנית:

1. השתמשו במסננים לפילוח משתמשי היעד
2. בחרו משתמשים עם תיבות סימון
3. בחרו פעולה מרובה מהתפריט הנפתח
4. אשרו את הפעולה עם קוד ה-PIN של המנהל
