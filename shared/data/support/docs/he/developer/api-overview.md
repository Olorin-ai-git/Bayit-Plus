# סקירת REST API

ה-API של Bayit+ מספק גישה תכנותית לפלטפורמת הסטרימינג. מדריך זה מכסה כתובות בסיס, גרסאות, פורמטי בקשות ומוסכמות כלליות הנהוגות בכל נקודות הקצה.

## כתובות בסיס

| סביבה | כתובת בסיס |
|-------|-----------|
| ייצור | `https://api.bayit.plus/v1` |
| Staging | `https://api.staging.bayit.plus/v1` |
| Sandbox | `https://api.sandbox.bayit.plus/v1` |

כל בקשות ה-API חייבות להשתמש ב-HTTPS. בקשות HTTP נדחות.

## גרסאות API

גרסת ה-API כלולה בנתיב ה-URL. הגרסה היציבה הנוכחית היא `v1`. כאשר מוכנסים שינויים שוברים, גרסה חדשה משוחררת תוך שמירה על תמיכה בגרסאות קודמות במהלך תקופת הוצאה משימוש.

מחזור חיי גרסה:
- **נוכחית**: בפיתוח פעיל ובתמיכה מלאה
- **מיושנת**: נתמכת אך ללא תכונות חדשות; מומלץ לעבור
- **הוסרה**: אינה זמינה עוד

## פורמט בקשות

כל הבקשות צריכות לכלול את הכותרות הבאות:

```
Content-Type: application/json
Accept: application/json
Authorization: Bearer {access_token}
```

גוף הבקשות חייב להיות JSON תקין. ערכי תאריך משתמשים בפורמט ISO 8601.

## פורמט תגובות

תגובות מוצלחות מחזירות JSON עם מבנה עקבי:

```json
{
  "data": { },
  "meta": {
    "request_id": "req_abc123",
    "timestamp": "2024-01-15T10:30:00Z"
  }
}
```

## עימוד

נקודות קצה של רשימות תומכות בעימוד באמצעות פרמטרי שאילתה:

| פרמטר | תיאור | ברירת מחדל |
|-------|-------|------------|
| `page` | מספר עמוד (מתחיל מ-1) | 1 |
| `per_page` | פריטים לעמוד | 20 |
| `sort` | שדה מיון | משתנה |
| `order` | כיוון מיון (asc/desc) | desc |

תגובות עם עימוד כוללות מטא-דאטה לניווט:

```json
{
  "data": [ ],
  "pagination": {
    "current_page": 1,
    "per_page": 20,
    "total_items": 150,
    "total_pages": 8
  }
}
```

## סינון

רוב נקודות הקצה של רשימות תומכות בסינון באמצעות פרמטרי שאילתה. תחביר הסינון משתנה לפי סוג שדה:

- **התאמה מדויקת**: `?status=published`
- **ערכים מרובים**: `?category=drama,comedy`
- **טווח**: `?release_year_gte=2020&release_year_lte=2024`
- **חיפוש**: `?q=search+term`

## זמינות SDK

ערכות SDK רשמיות זמינות עבור:

- JavaScript/TypeScript
- Python
- Swift
- Kotlin

ערכות ה-SDK מטפלות באימות, עימוד וטיפול בשגיאות אוטומטית.
