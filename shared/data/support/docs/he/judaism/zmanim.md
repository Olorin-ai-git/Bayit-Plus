# זמנים

Bayit+ מספקת זמנים הלכתיים מדויקים בהתאם למיקומכם לשמירה יומית ותפילה.

## צפייה בזמנים יומיים

לראות את הזמנים של היום:

1. פתחו את סעיף היהדות
2. זמנים מופיעים במסך הראשי
3. לחצו לצפייה ברשימה המלאה של הזמנים

הזמנים מתעדכנים אוטומטית בהתאם למיקום הנוכחי שלכם.

## זמנים זמינים

הזמנים ההלכתיים הבאים מוצגים:

- עלות השחר
- טלית ותפילין מוקדם
- נץ החמה (זריחה)
- סוף זמן קריאת שמע (מגן אברהם)
- סוף זמן קריאת שמע (גר"א)
- סוף זמן תפילת שחרית
- חצות (צהריים)
- מנחה גדולה
- מנחה קטנה
- פלג המנחה
- שקיעה
- צאת הכוכבים
- חצות לילה

## זמני שבת

זמנים מיוחדים לשבת כוללים:

- הדלקת נרות (18 דקות לפני השקיעה כברירת מחדל)
- צאת שבת (לאחר שלושה כוכבים)
- זמן רבינו תם

התאימו דקות הדלקת נרות בהגדרות שלכם בהתאם למנהג.

## הגדרות מיקום

הגדירו את המיקום שלכם לזמנים מדויקים:

1. עברו להגדרות
2. בחרו מיקום
3. בחרו זיהוי אוטומטי או הזינו ידנית
4. בחרו את הגובה שלכם אם ידוע

גובה משפיע על חישובי זמנים, במיוחד באזורים הרריים.

## התראות זמנים

הגדירו התראות לזמנים חשובים:

- תזכורות הדלקת נרות
- התראות זמן מנחה
- אזהרות שקיעה
- התראות מותאמות לכל זמן

הגדירו העדפות התראות בהגדרות היהדות.

## שיטות חישוב

בחרו את שיטת החישוב המועדפת:

- גר"א (הגאון מווילנא)
- מגן אברהם
- רבינו תם
- אפשרויות דקות קבועות

קהילות שונות נוהגות לפי שיטות שונות, ו-Bayit+ מתאימה לכל הדעות העיקריות.

## תצוגת ווידג'ט

הוסיפו זמנים למסך הבית לעיון מהיר ללא פתיחת האפליקציה.
