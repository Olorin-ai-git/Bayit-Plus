# פתרון בעיות סטרימינג

יש לכם בעיה בהפעלה? עקבו אחר השלבים הבאים לפתרון בעיות סטרימינג נפוצות.

## תיקונים מהירים

נסו קודם לפני שמעמיקים:

1. **בדקו חיבור אינטרנט** - ודאו שאתם מחוברים
2. **הפעילו מחדש את האפליקציה** - סגרו לגמרי ופתחו מחדש
3. **הפעילו מחדש את המכשיר** - התחלה מחדש לעתים עוזרת
4. **עדכנו את האפליקציה** - השיגו את הגרסה האחרונה
5. **נסו תוכן אחר** - בדקו אם הבעיה ספציפית לכותר אחד

## בעיות באפרינג

### תסמינים
- הווידאו נעצר לעתים קרובות
- ספינר "טעינה" מופיע
- האיכות יורדת פתאום

### פתרונות

#### בדקו את המהירות שלכם
דרישות מינימום:
| איכות | מהירות נדרשת |
|-------|--------------|
| SD (480p) | 3 Mbps |
| HD (720p) | 5 Mbps |
| Full HD (1080p) | 10 Mbps |
| 4K Ultra HD | 25 Mbps |

בדקו את המהירות ב-speedtest.net.

#### שפרו חיבור
1. **השתמשו ב-Wi-Fi** - יציב יותר מנתונים סלולריים
2. **התקרבו** - קרוב יותר לנתב
3. **הפחיתו הפרעות** - הרחק ממיקרוגלים, טלפונים אלחוטיים
4. **השתמשו ב-5GHz** - מהיר יותר מ-2.4GHz אם זמין
5. **חיבור קווי** - אתרנט הכי יציב (לטלוויזיה)

#### הפחיתו איכות
1. גשו ל**הגדרות** > **הפעלה**
2. הגדירו איכות ל**אוטומטי** או **בינוני**
3. כבו HDR אם מופעל
4. נסו שוב

#### סגרו אפליקציות רקע
אפליקציות אחרות עשויות להשתמש ברוחב פס:
1. סגרו דפדפנים עם וידאו
2. השהו הורדות
3. עצרו גיבויי ענן זמנית
4. נתקו מכשירים אחרים מהרשת

## וידאו לא מתחיל

### תסמינים
- מסך שחור
- טעינה אינסופית
- הודעות שגיאה

### פתרונות

#### בדקו את המנוי
1. גשו ל**הגדרות** > **חשבון**
2. ודאו שהמנוי פעיל
3. בדקו שאמצעי התשלום תקף
4. חדשו אם פג תוקף

#### נקו מטמון אפליקציה
**iOS:**
1. גשו להגדרות > כללי > אחסון iPhone
2. מצאו Bayit+ ולחצו
3. לחצו "פרוק אפליקציה" והתקינו מחדש

**Android:**
1. גשו להגדרות > אפליקציות > Bayit+
2. לחצו אחסון
3. לחצו "נקה מטמון"

**דפדפן:**
1. פתחו הגדרות דפדפן
2. נקו מטמון ועוגיות
3. הפעילו מחדש דפדפן

#### בדקו תאימות מכשיר
ודאו שהמכשיר עומד בדרישות:
- iOS 14.0 ומעלה
- Android 8.0 ומעלה
- tvOS 14.0 ומעלה
- Chrome 90+, Safari 14+, Firefox 90+

## איכות וידאו ירודה

### תסמינים
- וידאו מטושטש או מפוקסל
- רזולוציה נמוכה מהצפוי
- האיכות לא משתפרת לאורך זמן

### פתרונות

#### בחירת איכות ידנית
1. במהלך ההפעלה, לחצו על סמל **הגדרות**
2. בחרו **איכות**
3. בחרו **גבוה** או **אולטרה**
4. חכו שהאיכות תשתפר

#### בדקו הגדרות תצוגה
1. ודאו שרזולוציית המכשיר מוגדרת נכון
2. הפעילו HDR אם נתמך
3. בדקו כבל HDMI (לטלוויזיה)
4. עדכנו דרייברי תצוגה (לדפדפן)

#### אמתו איכות תוכן
- לא כל התוכן זמין ב-4K
- בדקו פרטי הכותר לאיכות זמינה
- תוכן ישן יותר עשוי להיות SD בלבד

## בעיות אודיו

### אין קול
1. בדקו עוצמת המכשיר
2. בדקו עוצמת האפליקציה
3. נסו תוכן אחר
4. בדקו שהשתקה לא פעילה
5. בדקו התקן פלט אודיו

### אודיו לא מסונכרן
1. השהו והמשיכו הפעלה
2. קפצו אחורה כמה שניות
3. הפעילו מחדש את האפליקציה
4. הורידו איכות וידאו

### פס קול שגוי
1. לחצו על הגדרות אודיו במהלך ההפעלה
2. בחרו שפה נכונה
3. בדקו הגדרת תיאור קולי

## בעיות טלוויזיה חיה

### ערוץ לא נטען
1. בדקו אם הערוץ בחבילה שלכם
2. נסו ערוץ אחר
3. בדקו הפסקות שירות
4. רעננו את מדריך הערוצים

### ערוץ קופא
1. הורידו הגדרות איכות
2. בדקו יציבות אינטרנט
3. נסו לטעון מחדש את הערוץ
4. דווחו על בעיות מתמשכות

### מדריך לא מתעדכן
1. משכו למטה לרענון
2. נקו מטמון אפליקציה
3. בדקו הגדרות אזור זמן
4. התקינו מחדש אם מתמיד

## בעיות הורדה

### הורדה נכשלת
1. בדקו שטח אחסון
2. בדקו חיבור אינטרנט
3. ודאו שהתוכן ניתן להורדה
4. בדקו שהמנוי מאפשר הורדות

### הורדה לא מופעלת
1. בדקו אם ההורדה פגה
2. ודאו שאתם מחוברים
3. הורידו מחדש את התוכן
4. בדקו תאריך/שעה של המכשיר

## עדיין יש בעיות?

אם הבעיות נמשכות אחרי ניסיון הפתרונות:

1. **בדקו סטטוס שירות** - בקרו בעמוד הסטטוס שלנו
2. **השתמשו בתמיכה קולית** - אמרו "היי בית, אני צריך עזרה"
3. **צרו פנייה** - גשו לתמיכה > יצירת קשר
4. **כללו פרטים**:
   - סוג ודגם מכשיר
   - גרסת אפליקציה
   - מהירות אינטרנט
   - הודעות שגיאה
   - שלבים לשחזור

צוות התמיכה שלנו מגיב בדרך כלל תוך 24 שעות.
