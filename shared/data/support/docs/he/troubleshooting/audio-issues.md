# פתרון בעיות שמע

נתקלים בבעיות עם השמע ב-Bayit+? מדריך זה מכסה בעיות שמע נפוצות ואת הפתרונות שלהן.

## אין שמע

### בדיקות ראשוניות

1. **ודאו שהעוצמה לא מושתקת**: בדקו את סמל ההשתקה בנגן ובמכשיר
2. **העלו את עוצמת הקול**: בדקו את עוצמת הקול באפליקציה ובמכשיר
3. **בדקו את חיבור הרמקולים**: אם משתמשים ברמקולים חיצוניים
4. **נתקו אוזניות**: לפעמים המכשיר חושב שאוזניות מחוברות

### פתרונות נוספים

- הפעילו מחדש את האפליקציה
- בדקו את הגדרות השמע של המכשיר
- נסו תוכן אחר לבדיקה
- הפעילו מחדש את המכשיר

## שמע לא מסונכרן עם וידאו

### אם השמע מקדים או מאחר

1. השהו והמשיכו את הניגון
2. הריצו אחורה או קדימה כמה שניות
3. סגרו ופתחו את האפליקציה
4. הורידו את איכות הוידאו

### בעיות סנכרון ספציפיות למכשיר

**טלוויזיה:**
- בדקו את הגדרות השהיית השמע בטלוויזיה
- נסו כבל HDMI אחר
- השביתו עיבוד שמע מתקדם

**מכשיר נייד:**
- ודאו שהאפליקציה מעודכנת
- נקו את מטמון האפליקציה
- הפעילו מחדש את המכשיר

## איכות שמע ירודה

### שמע מעוות או משובש

1. בדקו את חיבור האינטרנט
2. הורידו את איכות הסטרימינג
3. סגרו אפליקציות רקע
4. נסו חיבור קווי במקום Wi-Fi

### שמע חלש או שקט

1. העלו את העוצמה בכל הרמות (אפליקציה, מכשיר, רמקול)
2. הפעילו נורמליזציית שמע בהגדרות
3. בדקו את הגדרות השמע של המכשיר

## בעיות עם אוזניות

### אוזניות Bluetooth

1. נתקו והתאימו מחדש את האוזניות
2. ודאו שהאוזניות טעונות
3. בדקו שאין התקן אחר מחובר
4. עדכנו את קושחת האוזניות

### אוזניות קוויות

1. בדקו את חיבור הכבל
2. נסו אוזניות אחרות
3. נקו את שקע האוזניות

## שמע היקפי ו-Dolby

### Dolby Atmos לא עובד

1. ודאו שהתוכן תומך ב-Dolby Atmos
2. בדקו שהמכשיר והרמקולים תומכים
3. הפעילו Dolby Atmos בהגדרות האפליקציה
4. בדקו את הגדרות השמע של המכשיר

### בחירת רצועת שמע

1. לחצו על סמל השמע בנגן
2. בחרו את השפה או הפורמט המועדף
3. חלק מהתוכן מציע מספר אפשרויות שמע

## פנייה לתמיכה

אם הבעיה נמשכת:
1. עברו להגדרות ובחרו עזרה
2. בחרו בעיית שמע
3. ציינו את סוג המכשיר ואת הבעיה
4. צוות התמיכה יחזור אליכם
