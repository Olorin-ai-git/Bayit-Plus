# פתרון בעיות שמע

חווים בעיות שמע? עקבו אחר השלבים הבאים לאבחון ותיקון בעיות אודיו ב-Bayit+.

## אין שמע

### בדיקות מהירות

1. **בדקו עוצמת שמע במכשיר** - ודאו שאינו מושתק או באפס
2. **בדקו עוצמת שמע באפליקציה** - פקד עוצמת השמע בנגן עלול להיות מונמך
3. **בדקו חיבורים פיזיים** - אוזניות, רמקולים או כבלי HDMI
4. **נסו תוכן אחר** - ודאו שהבעיה משפיעה על מספר כותרים

### פתרונות ספציפיים למכשיר

**מובייל (iOS/Android):**
1. בדקו שמצב שקט כבוי
2. לחצו על כפתור הגברת עוצמה
3. בדקו ש-Bluetooth אודיו לא מחובר למקום אחר
4. הפעילו מחדש את האפליקציה

**טלוויזיה חכמה:**
1. בדקו עוצמת שמע וסטטוס השתקה בטלוויזיה
2. ודאו פלט אודיו נכון (רמקולי TV לעומת סאונדבר)
3. בדקו הגדרות אודיו HDMI-CEC
4. בדקו שמע באפליקציות אחרות

**דפדפן אינטרנט:**
1. בדקו שלשונית הדפדפן לא מושתקת
2. לחצו בתוך נגן הווידאו
3. בדקו מיקסר עוצמת המערכת
4. נסו דפדפן אחר

### הגדרות פלט אודיו

1. עברו ל**הגדרות** > **הפעלה** > **פלט אודיו**
2. ודאו שמכשיר הפלט הנכון נבחר
3. בדקו עם כפתור **בדיקת אודיו**
4. עברו בין אפשרויות סטריאו וסראונד

## שמע לא מסונכרן

### תסמינים

- קולות לא תואמים לתנועות שפתיים
- שמע מגיע לפני או אחרי הווידאו
- השהייה מתגברת עם הזמן

### פתרונות

#### תיקונים מהירים
1. השהו והמשיכו הפעלה
2. דלגו אחורה 10-15 שניות
3. הפעילו מחדש את האפליקציה לחלוטין
4. נסו כותר אחר

#### כיוון השהיית אודיו

אם המכשיר שלכם תומך בכיוון סנכרון אודיו:
1. בזמן הפעלה, לחצו על סמל **הגדרות**
2. בחרו **סנכרון אודיו**
3. כוונו השהייה במילישניות
4. ערכים חיוביים מעכבים אודיו
5. ערכים שליליים מקדימים אודיו

#### בעיות סנכרון הקשורות לרשת
1. הורידו איכות וידאו להפחתת באפרינג
2. בדקו יציבות חיבור האינטרנט
3. נתקו מכשירים אחרים שצורכים רוחב פס
4. נסו חיבור קווי אם זמין

#### תיקונים ספציפיים למכשיר

**טלוויזיה חכמה עם סאונדבר:**
1. בדקו הגדרות השהיית אודיו בסאונדבר
2. השביתו תכונות עיבוד אודיו
3. השתמשו בחיבור HDMI ARC אם זמין
4. עדכנו קושחת סאונדבר

**אוזניות Bluetooth:**
1. Bluetooth מציג השהייה טבעית
2. השתמשו באוזניות קוויות לסנכרון מושלם
3. בדקו תמיכה בקודק השהייה נמוכה
4. שמרו על המכשיר קרוב למקור ה-Bluetooth

## בעיות סראונד

### סראונד לא עובד

1. עברו ל**הגדרות** > **הפעלה** > **אודיו**
2. בדקו ש**סראונד** מופעל
3. ודאו שהציוד שלכם תומך בסראונד
4. בדקו שהתוכן כולל רצועת סראונד

### אימות תמיכת ציוד

| פורמט אודיו | דרישה |
|------------|--------|
| Dolby Digital 5.1 | רסיבר או סאונדבר תואם |
| Dolby Atmos | ציוד התומך ב-Atmos |
| DTS | רסיבר תואם DTS |

### הגדרת סראונד

1. ודאו שהרסיבר מוגדר לקלט הנכון
2. בדקו מצב פענוח ברסיבר (מומלץ Auto)
3. ודאו את כל חיבורי הרמקולים
4. הפעילו כיול רמקולים של הרסיבר

### זמינות תוכן

לא כל התוכן תומך בסראונד:
- בדקו פרטי כותר לפורמטי אודיו זמינים
- חלק מהכותרים מציעים רק סטריאו
- לתוכן ישן יותר עשויות להיות אפשרויות מוגבלות

## רצועת אודיו שגויה

### שינוי שפת אודיו

1. בזמן הפעלה, לחצו על סמל **אודיו**
2. בחרו שפה מועדפת
3. האודיו עובר מיד
4. ההגדרה נשמרת להפעלות עתידיות

### תיאור קולי

אם שומעים קריינות של אלמנטים ויזואליים:
1. לחצו על סמל **אודיו** בזמן הפעלה
2. השביתו **תיאור קולי**
3. בחרו רצועת אודיו רגילה
4. בדקו הגדרות נגישות

### הגדרת אודיו ברירת מחדל

1. עברו ל**הגדרות** > **הפעלה** > **אודיו**
2. הגדירו **שפת אודיו מועדפת**
3. הפעילו ברירת מחדל **תיאור קולי** אם נדרש
4. ההגדרות חלות על כל הפעלה עתידית

## אודיו מרשרש או מעוות

### סיבות ופתרונות

1. **הנמיכו עוצמה** - עיוות ברמות גבוהות
2. **בדקו חיבורים** - כבלים רופפים גורמים לבעיות
3. **עדכנו דרייברים** - להפעלה בדפדפן אינטרנט
4. **הורידו איכות** - רוחב פס נמוך יותר עוזר

### פתרון בעיות חומרה

1. בדקו עם אוזניות או רמקולים שונים
2. נסו פלט אודיו אחר
3. בדקו עדכוני תוכנה למכשיר
4. הפעילו מחדש את ציוד האודיו

## פניה לתמיכה

אם בעיות האודיו נמשכות:
1. **באפליקציה:** הגדרות > עזרה > בעיית אודיו
2. **קולית:** "היי בית, יש לי בעיית אודיו"
3. כללו: סוג מכשיר, שם תוכן ותיאור הבעיה
