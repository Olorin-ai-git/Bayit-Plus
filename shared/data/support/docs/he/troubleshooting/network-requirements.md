# דרישות רשת

ודאו שחיבור האינטרנט שלכם עומד בדרישות לביצועי סטרימינג אופטימליים ב-Bayit+.

## דרישות מהירות אינטרנט

### מהירות מינימלית לפי איכות

| איכות וידאו | מהירות הורדה נדרשת |
|-------------|-------------------|
| SD (480p) | 3 Mbps |
| HD (720p) | 5 Mbps |
| Full HD (1080p) | 10 Mbps |
| 4K Ultra HD | 25 Mbps |
| 4K עם HDR | 40 Mbps |

### בדיקת מהירות

1. בקרו ב-speedtest.net או fast.com
2. הריצו בדיקת מהירות
3. רשמו את מהירות ההורדה
4. השוו מול הדרישות
5. בדקו בשעות שונות ביום

### המלצות מהירות

לחוויה הטובה ביותר, אנו ממליצים:
- **צופה יחיד**: פי 1.5 מהמהירות המינימלית
- **מספר צופים**: חברו את הדרישות יחד
- **דוגמה**: שני זרמים ב-4K דורשים 50+ Mbps

## אופטימיזציית Wi-Fi

### מיקום הנתב

1. מקמו את הנתב במיקום מרכזי
2. הרימו את הנתב מהרצפה
3. הרחיקו מקירות ועצמים מתכתיים
4. הימנעו ממיקום ליד מיקרוגל או טלפונים אלחוטיים

### תדרי Wi-Fi

| תדר | מתאים ל |
|------|---------|
| 2.4 GHz | טווח ארוך יותר, יותר הפרעות |
| 5 GHz | מהירות גבוהה יותר, טווח קצר יותר |

**המלצה:** השתמשו ב-5 GHz לסטרימינג כשאפשר.

### הפחתת הפרעות

1. שנו ערוץ Wi-Fi בהגדרות הנתב
2. עדכנו קושחת הנתב
3. הסירו מכשירים מיותרים מהרשת
4. שקלו Wi-Fi mesh לבתים גדולים

### חיבור קווי

לסטרימינג יציב בטלוויזיה:
1. חברו כבל Ethernet למכשיר הסטרימינג
2. חברו את הקצה השני לנתב
3. חיבור קווי מבטל בעיות Wi-Fi
4. מומלץ לסטרימינג 4K

## הגדרות Firewall

### פורטים נדרשים

Bayit+ דורשת פורטים אלו פתוחים:

| פורט | פרוטוקול | מטרה |
|------|---------|-------|
| 80 | TCP | תעבורת HTTP |
| 443 | TCP | תעבורת HTTPS |
| 53 | UDP | פתרון DNS |

## בעיות VPN

### תאימות VPN

שימוש ב-VPN עלול לגרום לבעיות:
- הפחתת מהירות סטרימינג
- התנגשויות הגבלה גיאוגרפית
- ניתוקים במהלך ניגון

### הגדרות VPN מומלצות

אם חייבים להשתמש ב-VPN:
1. התחברו לשרת במדינה שלכם
2. השתמשו בשרת מותאם לסטרימינג
3. הפעילו split tunneling עבור Bayit+
4. השתמשו בפרוטוקול UDP למהירות טובה יותר

## סטרימינג בנתונים סלולריים

### צריכת נתונים משוערת

| איכות | נתונים לשעה |
|--------|-------------|
| נמוכה | 0.3 GB |
| בינונית | 0.7 GB |
| גבוהה | 1.5 GB |
| אולטרה | 3 GB |

### הפחתת צריכת נתונים

1. עברו להגדרות ובחרו ניגון
2. הגדירו איכות נתונים סלולריים לנמוכה או בינונית
3. הפעילו Wi-Fi בלבד להורדות
4. השתמשו בתוכן שהורד כשאפשר
