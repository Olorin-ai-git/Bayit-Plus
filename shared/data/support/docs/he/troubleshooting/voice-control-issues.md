# פתרון בעיות שליטה קולית

נתקלים בבעיות עם פקודות קול? מדריך זה עוזר לפתור בעיות הרשאות מיקרופון, זיהוי קולי ובעיות מילת השכמה.

## הרשאות מיקרופון

### בדיקת גישה למיקרופון

שליטה קולית דורשת הרשאת מיקרופון לפעול.

**iOS:**
1. פתחו הגדרות המכשיר
2. גללו למטה ל-Bayit+
3. הפעילו את מתג המיקרופון
4. חזרו ל-Bayit+ ונסו שוב

**Android:**
1. פתחו הגדרות המכשיר
2. עברו לאפליקציות ו-Bayit+
3. לחצו על הרשאות
4. הפעילו מיקרופון
5. חזרו ל-Bayit+ ונסו שוב

**דפדפן אינטרנט:**
1. לחצו על סמל המנעול בשורת הכתובת
2. מצאו הרשאת מיקרופון
3. הגדירו ל-אפשר
4. רעננו את הדף

### שגיאת הרשאה נדחתה

אם רואים "גישה למיקרופון נדחתה":
1. עקבו אחר הצעדים לעיל לפי המכשיר
2. הפעילו מחדש את אפליקציית Bayit+
3. נסו שליטה קולית שוב
4. אשרו הרשאה כשמתבקשים

### מיקרופון בשימוש ע"י אפליקציה אחרת

אם אפליקציה אחרת משתמשת במיקרופון:
1. סגרו אפליקציות תומכות קול אחרות
2. סיימו כל שיחה פעילה
3. עצרו כל הקלטה בתהליך
4. חזרו ל-Bayit+ ונסו שוב

## בעיות זיהוי קולי

### קול לא מזוהה

אם האפליקציה לא מבינה את הפקודות שלכם:

1. **דברו בבירור**: השתמשו בקצב דיבור רגיל
2. **הפחיתו רעשי רקע**: עברו לאזור שקט יותר
3. **התקרבו**: דברו לכיוון המיקרופון
4. **בדקו מיקרופון**: ודאו שהוא לא חסום

### שיפור דיוק זיהוי

1. עברו להגדרות ובחרו שליטה קולית ושפה
2. בחרו את השפה המועדפת
3. הפעילו אימון קולי אם זמין
4. השלימו את מפגש האימון

### שפות נתמכות

שליטה קולית תומכת ב:
- אנגלית (ארה"ב, בריטניה, אוסטרליה)
- עברית
- ספרדית (לטינית, אירופאית)

בחרו את השפה שלכם בהגדרות ושליטה קולית.

### פקודות קוליות נפוצות

| פקודה | פעולה |
|--------|--------|
| "הפעל [כותר]" | מתחיל להפעיל תוכן |
| "השהה" | משהה ניגון נוכחי |
| "המשך" | ממשיך ניגון |
| "חזור 30 שניות" | מריץ אחורה |
| "דלג קדימה" | קופץ קדימה |
| "הפעל כתוביות" | מפעיל כתוביות |
| "חפש [שאילתה]" | מחפש תוכן |
| "פתח הגדרות" | עובר להגדרות |

## בעיות מילת השכמה

### מילת השכמה לא מזוהה

מילת השכמה "היי בית" מפעילה שליטה קולית.

אם מילת השכמה לא מזוהה:
1. עברו להגדרות ובחרו שליטה קולית
2. ודאו שזיהוי מילת השכמה מופעל
3. בדקו שהרשאת מיקרופון ניתנה
4. ודאו שהמכשיר לא במצב חיסכון בצריכה

### התאמת רגישות מילת השכמה

1. עברו להגדרות ובחרו שליטה קולית ומילת השכמה
2. התאימו רמת רגישות:
   - **נמוכה**: פחות הפעלות שגויות
   - **בינונית**: מאוזן (מומלץ)
   - **גבוהה**: יותר רספונסיבי

### הפעלות שגויות

אם שליטה קולית מופעלת בטעות:
1. הורידו רגישות מילת השכמה
2. השביתו מילת השכמה והשתמשו בהפעלת כפתור
3. בדקו מילים בעלות צליל דומה בסביבה

### שינוי שיטת הפעלה

אם אתם מעדיפים לא להשתמש במילת השכמה:
1. עברו להגדרות ובחרו שליטה קולית
2. השביתו זיהוי מילת השכמה
3. השתמשו בכפתור המיקרופון במקום
4. לחצו והחזיקו כדי לדבר פקודות

## פרטיות ונתוני קול

### מדיניות הקלטת קול

- נתוני קול מעובדים בזמן אמת
- פקודות לא נשמרות לצמיתות
- אין שיתוף עם צד שלישי
- השביתו היסטוריית קול בהגדרות פרטיות

### השבתת תכונות קול

1. עברו להגדרות ובחרו שליטה קולית
2. כבו את שליטה קולית
3. תכונות קול מושבתות לחלוטין
4. השתמשו בפקדים ידניים בלבד

## פנייה לתמיכה

לבעיות שליטה קולית מתמשכות:
1. באפליקציה: הגדרות ועזרה ובעיית שליטה קולית
2. כללו: סוג מכשיר, פקודה שניסיתם, והודעת שגיאה
