# פתרון בעיות שליטה קולית

נתקלים בבעיות עם פקודות קוליות? מדריך זה עוזר לפתור בעיות הרשאות מיקרופון, זיהוי קולי ומילת השכמה.

## הרשאות מיקרופון

### בדיקת גישה למיקרופון

שליטה קולית דורשת הרשאת מיקרופון כדי לפעול.

**iOS:**
1. פתחו הגדרות במכשיר
2. גללו למטה ל-Bayit+
3. הפעילו מתג **מיקרופון**
4. חזרו ל-Bayit+ ונסו שוב

**Android:**
1. פתחו הגדרות במכשיר
2. עברו לאפליקציות > Bayit+
3. לחצו על הרשאות
4. הפעילו **מיקרופון**
5. חזרו ל-Bayit+ ונסו שוב

**דפדפן אינטרנט:**
1. לחצו על סמל המנעול בשורת הכתובת
2. מצאו הרשאת מיקרופון
3. הגדירו ל**אפשר**
4. רעננו את הדף

### שגיאת גישה למיקרופון נדחתה

אם אתם רואים "גישה למיקרופון נדחתה":
1. עקבו אחר הצעדים למעלה למכשיר שלכם
2. הפעילו מחדש את אפליקציית Bayit+
3. נסו שליטה קולית שוב
4. הענקו הרשאה כשתתבקשו

### מיקרופון בשימוש על ידי אפליקציה אחרת

אם אפליקציה אחרת משתמשת במיקרופון:
1. סגרו אפליקציות אחרות עם שליטה קולית
2. סיימו כל שיחה פעילה
3. עצרו כל הקלטה בתהליך
4. חזרו ל-Bayit+ ונסו שוב

## בעיות זיהוי קולי

### קול לא מזוהה

אם האפליקציה לא מבינה את הפקודות:

1. **דברו בבירור** - השתמשו בקצב דיבור רגיל
2. **הפחיתו רעשי רקע** - עברו לאזור שקט יותר
3. **התקרבו** - דברו לכיוון המיקרופון
4. **בדקו מיקרופון** - ודאו שלא חסום

### שיפור דיוק הזיהוי

1. עברו ל**הגדרות** > **שליטה קולית** > **שפה**
2. בחרו את השפה המועדפת
3. הפעילו **אימון קולי** אם זמין
4. השלימו את מפגש האימון

### שפות נתמכות

שליטה קולית תומכת ב:
- אנגלית (אמריקאית, בריטית, אוסטרלית)
- עברית
- ספרדית (אמריקה הלטינית, אירופית)

בחרו את השפה בהגדרות > שליטה קולית.

### פקודות קוליות נפוצות

| פקודה | פעולה |
|-------|-------|
| "הפעל [כותר]" | מתחיל להפעיל תוכן |
| "עצור" | משהה הפעלה נוכחית |
| "המשך" | ממשיך הפעלה |
| "חזור 30 שניות" | מריץ הפעלה אחורה |
| "קפוץ קדימה" | קופץ קדימה |
| "הפעל כתוביות" | מפעיל כתוביות |
| "חפש [שאילתה]" | מחפש תוכן |
| "פתח הגדרות" | עובר להגדרות |

## בעיות מילת השכמה

### מילת השכמה לא מזוהה

מילת ההשכמה "היי בית" מפעילה שליטה קולית.

אם מילת ההשכמה לא מזוהה:
1. עברו ל**הגדרות** > **שליטה קולית**
2. ודאו ש**זיהוי מילת השכמה** מופעל
3. בדקו שהרשאת מיקרופון ניתנה
4. ודאו שהמכשיר לא במצב חיסכון בחשמל

### כיוון רגישות מילת השכמה

1. עברו ל**הגדרות** > **שליטה קולית** > **מילת השכמה**
2. כוונו רמת **רגישות**:
   - **נמוכה** - פחות הפעלות שווא
   - **בינונית** - מאוזן (מומלץ)
   - **גבוהה** - יותר תגובתי

### הפעלות שווא

אם שליטה קולית מופעלת בטעות:
1. הורידו רגישות מילת השכמה
2. השביתו מילת השכמה והשתמשו בהפעלה בכפתור
3. בדקו מילים דומות בסביבה

### שינוי שיטת הפעלה

אם אתם מעדיפים לא להשתמש במילת ההשכמה:
1. עברו ל**הגדרות** > **שליטה קולית**
2. השביתו **זיהוי מילת השכמה**
3. השתמשו בכפתור המיקרופון במקום
4. לחצו והחזיקו כדי לומר פקודות

## בעיות ספציפיות למכשיר

### שליטה קולית בטלוויזיה חכמה

**Apple TV:**
1. החזיקו את כפתור Siri בשלט
2. אמרו את הפקודה
3. מילת השכמה לא נתמכת ב-Apple TV

**Android TV:**
1. לחצו על כפתור המיקרופון בשלט
2. או אמרו "Hey Google, פתח את Bayit+"
3. השתמשו באינטגרציית Google Assistant

**Amazon Fire TV:**
1. החזיקו את כפתור Alexa בשלט
2. אמרו "Alexa, פתח את Bayit+"
3. השתמשו ב-Alexa לפקודות קוליות

### שליטה קולית במובייל

1. לחצו על סמל המיקרופון באפליקציה
2. או השתמשו במילת ההשכמה כשהאפליקציה פתוחה
3. האזנת רקע דורשת שהאפליקציה תהיה פעילה

## צעדי פתרון בעיות

### שליטה קולית לא עובדת בכלל

1. הפעילו מחדש את אפליקציית Bayit+
2. בדקו הרשאות מיקרופון
3. ודאו חיבור אינטרנט
4. עדכנו את האפליקציה לגרסה האחרונה
5. הפעילו מחדש את המכשיר

### שליטה קולית עבדה בעבר

1. בדקו שינויי הרשאות אחרונים
2. ודאו שאין אפליקציות חדשות חוסמות מיקרופון
3. בדקו עדכוני מערכת
4. נקו מטמון אפליקציה ונסו שוב

### בדיקת המיקרופון

1. עברו ל**הגדרות** > **שליטה קולית**
2. לחצו על **בדוק מיקרופון**
3. דברו כשתתבקשו
4. ודאו שהקול שלכם מזוהה
5. כוונו רגישות מיקרופון אם נדרש

## פרטיות ונתוני קול

### מדיניות הקלטת קול

- נתוני קול מעובדים בזמן אמת
- פקודות לא נשמרות לצמיתות
- אין שיתוף נתוני קול עם צד שלישי
- השביתו היסטוריית קול בהגדרות פרטיות

### השבתת תכונות קול

1. עברו ל**הגדרות** > **שליטה קולית**
2. כבו את מתג **שליטה קולית**
3. תכונות קול מושבתות לחלוטין
4. השתמשו בפקדים ידניים בלבד

## פניה לתמיכה

לבעיות שליטה קולית מתמשכות:
1. **באפליקציה:** הגדרות > עזרה > בעיית שליטה קולית
2. כללו: סוג מכשיר, פקודה שניסיתם והודעת שגיאה
