# מדריך קודי שגיאה

מדריך זה מסביר קודי שגיאה נפוצים ב-Bayit+ ואת הפתרונות שלהם.

## שגיאות רשת (E001-E019)

### E001 - שגיאת חיבור רשת

**סיבה:** לא ניתן להתחבר לשרתי Bayit+.

**פתרונות:**
1. בדקו את חיבור האינטרנט
2. הפעילו מחדש את הנתב
3. נסו להחליף בין Wi-Fi לנתונים סלולריים
4. השביתו VPN אם פעיל
5. נסו שוב בעוד כמה דקות

### E002 - תם הזמן לחיבור

**סיבה:** תגובת השרת ארכה זמן רב מדי.

**פתרונות:**
1. בדקו את מהירות האינטרנט (מינימום 3 Mbps נדרש)
2. התקרבו לנתב
3. סגרו אפליקציות אחרות שמשתמשות ברוחב פס
4. הפעילו מחדש את האפליקציה

### E003 - שגיאת DNS

**סיבה:** לא ניתן לפתור את כתובת השרת.

**פתרונות:**
1. הפעילו מחדש את המכשיר
2. אפסו הגדרות רשת
3. נסו DNS של Google (8.8.8.8) או Cloudflare (1.1.1.1)

## שגיאות אימות (E020-E039)

### E020 - אימות נכשל

**סיבה:** פרטי התחברות שגויים או פגי תוקף.

**פתרונות:**
1. אמתו את האימייל והסיסמה
2. השתמשו ב"שכחתי סיסמה" לאיפוס
3. בדקו ש-Caps Lock כבוי

### E021 - פג תוקף ההתחברות

**סיבה:** ההתחברות פגה תוקף.

**פתרונות:**
1. התחברו מחדש
2. הפעילו "זכור אותי" להתחברות ארוכה יותר

### E022 - חשבון נעול

**סיבה:** ניסיונות התחברות כושלים רבים מדי.

**פתרונות:**
1. המתינו 30 דקות לפני ניסיון נוסף
2. השתמשו באיפוס סיסמה
3. פנו לתמיכה אם לא ניסיתם להתחבר

## שגיאות מנוי (E040-E059)

### E040 - נדרש מנוי

**סיבה:** לא נמצא מנוי פעיל.

**פתרונות:**
1. בדקו את מצב המנוי בהגדרות
2. אמתו שאמצעי התשלום תקף
3. הפעילו מחדש את המנוי

### E041 - תשלום נכשל

**סיבה:** לא ניתן לעבד את התשלום.

**פתרונות:**
1. עדכנו את אמצעי התשלום
2. בדקו שיש כסף בחשבון
3. פנו לבנק

### E042 - הגעתם למגבלת התוכנית

**סיבה:** חריגה ממגבלת צפיות בו-זמנית.

**פתרונות:**
1. עצרו ניגון במכשיר אחר
2. בדקו מכשירים פעילים בהגדרות
3. שדרגו תוכנית

## שגיאות תוכן (E060-E079)

### E060 - תוכן לא זמין

**סיבה:** התוכן המבוקש אינו זמין.

**פתרונות:**
1. התוכן עשוי להיות הוסר
2. הרישוי עשוי להשתנות
3. נסו תוכן אחר

### E061 - הגבלה גיאוגרפית

**סיבה:** התוכן לא זמין במיקומכם.

**פתרונות:**
1. חלק מהתוכן מוגבל לאזורים מסוימים
2. השביתו VPN אם משתמשים

### E062 - שגיאת ניגון

**סיבה:** הוידאו אינו יכול להתנגן.

**פתרונות:**
1. הפעילו מחדש את האפליקציה
2. נקו מטמון
3. עדכנו לגרסה האחרונה
4. הפעילו מחדש את המכשיר

## דיווח על שגיאות

אם השגיאה נמשכת:
1. רשמו את קוד השגיאה המדויק
2. צלמו מסך אם אפשר
3. עברו להגדרות ובחרו עזרה ודווחו על בעיה
4. כללו: קוד שגיאה, צעדים שננקטו, פרטי מכשיר
