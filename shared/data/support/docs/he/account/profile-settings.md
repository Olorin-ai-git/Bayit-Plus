# הגדרות פרופיל

התאימו את חוויית ה-Bayit+ שלכם עם הגדרות פרופיל.

## ניהול פרופילים

### יצירת פרופיל

1. פתחו את אפליקציית Bayit+
2. לחצו על סמל הפרופיל שלכם
3. לחצו **הוסף פרופיל**
4. הזינו שם פרופיל
5. בחרו אווטאר
6. הגדירו כפרופיל ילדים אם צריך
7. שמרו

### עריכת פרופיל

1. לחצו על סמל הפרופיל
2. לחצו **נהל פרופילים**
3. בחרו פרופיל לעריכה
4. שנו שם, אווטאר או הגדרות
5. שמרו שינויים

### מחיקת פרופיל

1. גשו ל**נהל פרופילים**
2. בחרו פרופיל למחיקה
3. לחצו **מחק פרופיל**
4. אשרו מחיקה

**הערה:** מחיקת פרופיל מסירה את כל ההעדפות והיסטוריית הצפייה.

### מגבלות פרופילים

| תוכנית | מקסימום פרופילים |
|--------|------------------|
| בסיסי | 2 |
| סטנדרט | 4 |
| פרימיום | 5 |
| משפחה | 6 |

## התאמה אישית של פרופיל

### אפשרויות אווטאר

בחרו מ:
- אווטארי דמויות מוגדרים מראש
- רקעים צבעוניים
- העלאת תמונה מותאמת (פרימיום)
- אווטארים מונפשים (פרימיום)

### שם תצוגה

- מקסימום 20 תווים
- תווים מיוחדים מותרים
- אימוג'י נתמכים
- גלוי לבני המשפחה

### הגדרות שפה

הגדירו שפה לכל פרופיל:

1. פתחו הגדרות פרופיל
2. לחצו **שפה**
3. בחרו:
   - **שפת אפליקציה**: שפת הממשק
   - **שפת תוכן**: ברירת מחדל לאודיו/כתוביות
4. שמרו

זמין: אנגלית, עברית, ספרדית

## העדפות

### העדפות תוכן

עזרו לנו להמליץ על תוכן טוב יותר:

1. גשו ל**פרופיל** > **העדפות**
2. בחרו ז'אנרים אהובים
3. דרגו תוכן שצפיתם
4. סמנו תוכן כ"לא מעניין"

### דירוגי בגרות

שליטה בנראות תוכן:

1. גשו ל**פרופיל** > **הגדרות בגרות**
2. הגדירו דירוג מקסימלי:
   - ילדים (G, TV-Y)
   - נוער (PG-13, TV-14)
   - מבוגרים (R, TV-MA)
   - כל התוכן
3. הזינו PIN לאישור

### הגדרות הפעלה אוטומטית

| הגדרה | תיאור |
|-------|-------|
| הפעלה אוטומטית של תצוגה מקדימה | וידאו מופעל אוטומטית בזמן דפדוף |
| פרק הבא | הפעלה אוטומטית של הפרק הבא |
| דלג על פתיחה | דילוג אוטומטי על פתיחות סדרות |
| יציאה בקרדיטים | יציאה מהנגן בקרדיטים |

## פרופילי ילדים

### יצירת פרופיל ילדים

1. הוסיפו פרופיל חדש
2. הפעילו **פרופיל ילדים**
3. בחרו טווח גילאים
4. בחרו אווטאר
5. שמרו

### תכונות פרופיל ילדים

- תוכן מותאם גיל בלבד
- ממשק מפושט
- אין גישה להגדרות
- אין אפשרויות תשלום
- חיפוש בטוח מופעל

### הגנת PIN

הגנו על פרופילים בוגרים:

1. גשו להגדרות פרופיל בוגר
2. לחצו **נעילת פרופיל**
3. הגדירו PIN בן 4 ספרות
4. נדרש לכניסה לפרופיל

## הגדרות הפעלה

### איכות וידאו

העדפות איכות לכל פרופיל:

| הגדרה | הכי טוב ל |
|-------|----------|
| אוטומטי | מתאים לחיבור |
| נמוך | חיסכון בנתונים |
| בינוני | מאוזן |
| גבוה | תמונה מיטבית |
| אולטרה | מסכי 4K |

### הגדרות אודיו

- שפת אודיו ברירת מחדל
- הפעלה/כיבוי סראונד
- נורמליזציית עוצמה
- ברירת מחדל לתיאור קולי

### הגדרות כתוביות

- שפת כתוביות ברירת מחדל
- סגנון כתוביות:
  - גודל גופן (קטן/בינוני/גדול)
  - צבע גופן
  - אטימות רקע
  - מיקום

## התראות

### סוגי התראות

שליטה באילו התראות לקבל:

| סוג | תיאור |
|-----|-------|
| חדשות | כשתוכן חדש מגיע |
| המשך צפייה | תזכורות להמשיך |
| המלצות | הצעות מותאמות אישית |
| פג בקרוב | הורדות שעומדות לפוג |
| חשבון | עדכוני חשבון חשובים |

### ניהול התראות

1. גשו ל**פרופיל** > **התראות**
2. החליפו כל סוג התראה
3. הגדירו שעות שקט אם רצוי
4. בחרו אימייל לעומת פוש

## הגדרות פרטיות

### היסטוריית צפייה

- צפו בהיסטוריה המלאה שלכם
- הסירו פריטים בודדים
- נקו את כל ההיסטוריה
- השהו היסטוריה (מצב פרטי)

### נתונים ופרטיות

1. **הורידו את הנתונים שלכם**: בקשו את כל הנתונים
2. **מחקו חשבון**: מחיקה קבועה
3. **התאמה אישית של פרסומות**: החליפו פרסומות ממוקדות
4. **אנליטיקה**: בטלו השתתפות באנליטיקה

### פעילות צפייה

צפו במכשירים ובהפעלות:

1. גשו ל**הגדרות** > **אבטחה**
2. לחצו **פעילות אחרונה**
3. צפו בכל המכשירים המחוברים
4. התנתקו ממכשירים ספציפיים
5. התנתקו מכל המכשירים

## סנכרון בין מכשירים

### מה מסתנכרן

- היסטוריית צפייה
- הרשימה שלי
- העדפות
- מיקום המשך צפייה
- דירוגים וביקורות

### בעיות סנכרון

אם לא מסתנכרן:
1. בדקו חיבור אינטרנט
2. ודאו אותו חשבון בכל המכשירים
3. כפו רענון (משכו למטה)
4. התנתקו והתחברו מחדש

## הגדרות חשבון

### אימייל וסיסמה

1. גשו ל**הגדרות** > **חשבון**
2. עדכנו כתובת אימייל
3. שנו סיסמה
4. אמתו באימייל

### מספר טלפון

הוסיפו לשחזור חשבון:
1. גשו ל**הגדרות** > **חשבון**
2. לחצו **מספר טלפון**
3. הזינו ואמתו ב-SMS

### חשבונות מקושרים

נהלו התחברויות חברתיות:
- גוגל
- אפל
- פייסבוק

הוסיפו או הסירו חשבונות מקושרים בהגדרות אבטחה.

## נגישות

### ויזואלי

- מצב ניגודיות גבוהה
- טקסט גדול יותר
- תמיכה בקורא מסך
- תנועה מופחתת

### אודיו

- תיאורים קוליים
- ברירת מחדל לכתוביות סגורות
- אודיו מונו
- שליטה קולית

### ניווט

- מחווני פוקוס
- ניווט מקלדת
- פקודות קוליות
- תמיכה בבקרת מתג

מצאו את כל האפשרויות ב**הגדרות** > **נגישות**.

## ייצוא הנתונים שלכם

### בקשת ייצוא נתונים

1. גשו ל**הגדרות** > **פרטיות**
2. לחצו **הורידו את הנתונים שלכם**
3. אשרו בקשה
4. קבלו אימייל כשמוכן
5. הורידו תוך 7 ימים

### נתונים כלולים

- מידע חשבון
- היסטוריית צפייה
- העדפות
- היסטוריית תשלומים
- אינטראקציות תמיכה
