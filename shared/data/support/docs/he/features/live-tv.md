# מדריך טלוויזיה חיה

צפו במאות ערוצים חיים עם Bayit+ Live TV.

## גישה לטלוויזיה חיה

1. פתחו את אפליקציית Bayit+
2. נווטו ללשונית **חי**
3. דפדפו או חפשו ערוצים
4. לחצו על ערוץ להתחיל לצפות

## מדריך ערוצים

מדריך הערוצים מציג מה משודר עכשיו ותוכניות קרובות.

### שימוש במדריך
- **תוכניות נוכחיות** - צפו מה משודר עכשיו בכל הערוצים
- **תצוגת לוח זמנים** - צפו בתוכניות קרובות ל-7 ימים הבאים
- **סינון לפי קטגוריה** - ספורט, חדשות, סרטים, בידור, ילדים
- **חיפוש ערוצים** - מצאו ערוצים ספציפיים לפי שם או מספר

### ניווט במדריך
- החליקו שמאלה/ימינה לנוע בזמן
- החליקו למעלה/למטה לדפדף בערוצים
- לחצו על תוכנית לפרטים
- לחצו ארוכות להגדיר תזכורת

## קטגוריות ערוצים

### חדשות
- CNN, BBC, Sky News
- ערוצי חדשות מקומיים
- חדשות עסקיות וכלכליות

### ספורט
- ESPN, Fox Sports
- משחקים ואירועים חיים
- חדשות וניתוח ספורט

### בידור
- סרטים, סדרות, ריאליטי
- ערוצי מוזיקה
- קומדיה ודרמה

### ילדים
- ערוצי קריקטורות
- תוכן חינוכי
- תכנות מותאם גיל

### בינלאומי
- ערוצים מכל העולם
- תוכן רב-לשוני
- תכנות תרבותי

## תכונות טלוויזיה חיה

### תמונה בתוך תמונה
צפו בטלוויזיה חיה בזמן דפדוף בתוכן אחר:
1. התחילו לצפות בערוץ
2. לחצו על כפתור PiP
3. המשיכו לדפדף באפליקציה

### נגן מיני
נגן קטן מופיע כשאתם מנווטים משם, מאפשר להמשיך לצפות.

### ערוצים מועדפים
סמנו ערוצים אהובים לגישה מהירה:
1. פתחו פרטי ערוץ
2. לחצו על סמל הכוכב
3. גשו למועדפים מלשונית המועדפים

### הקלטת ערוצים (פרימיום)
הקליטו טלוויזיה חיה לצפייה מאוחר יותר:
1. פתחו את מדריך התוכניות
2. מצאו את התוכנית שברצונכם להקליט
3. לחצו **הקלט**
4. גשו להקלטות מהספרייה

## הגדרות איכות

התאימו איכות סטרימינג בהתאם לחיבור:

| הגדרה | רזולוציה | צריכת נתונים |
|-------|----------|---------------|
| אוטומטי | מתאים | משתנה |
| נמוך | 480p | ~0.5 GB/שעה |
| בינוני | 720p | ~1.5 GB/שעה |
| גבוה | 1080p | ~3 GB/שעה |
| אולטרה | 4K | ~7 GB/שעה |

## בקרת הורים

הגבילו גישה לערוצים או דירוגי תוכן מסוימים:

1. גשו ל**הגדרות** > **בקרת הורים**
2. הגדירו קוד PIN
3. בחרו ערוצים או דירוגים מוגבלים
4. שמרו את ההגדרות

## פתרון בעיות

### ערוץ לא נטען
1. בדקו את חיבור האינטרנט
2. נסו להוריד את הגדרת האיכות
3. סגרו ופתחו מחדש את האפליקציה
4. הפעילו מחדש את המכשיר

### איכות ירודה
1. בדקו את מהירות האינטרנט
2. התחברו ל-Wi-Fi אם על נתונים סלולריים
3. הפחיתו איכות בהגדרות
4. התקרבו לנתב

### מדריך לא מתעדכן
1. משכו למטה לרענון
2. בדקו חיבור אינטרנט
3. נקו מטמון אפליקציה
4. עדכנו את האפליקציה

ראו [בעיות סטרימינג](/troubleshooting/streaming-issues) לעזרה נוספת.
