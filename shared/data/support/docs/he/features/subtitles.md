# כתוביות ונגישות

Bayit+ מספקת אפשרויות כתוביות מקיפות ותכונות נגישות כדי להבטיח שכולם יוכלו ליהנות מהתוכן שלנו.

## שפות כתוביות זמינות

כתוביות זמינות במספר שפות בהתאם לתוכן. אפשרויות נפוצות כוללות:
- אנגלית
- עברית
- ספרדית
- ערבית
- רוסית
- צרפתית

## הפעלת כתוביות

### בזמן ניגון
1. לחצו על המסך להצגת פקדי ניגון
2. בחרו את סמל הכתוביות (CC או בועת דיבור)
3. בחרו את השפה המועדפת
4. כתוביות מופיעות מיידית

### הגדרות ברירת מחדל
הגדירו שפת כתוביות ברירת מחדל בהגדרות הפרופיל:
1. עברו להגדרות ובחרו את הפרופיל שלכם
2. בחרו "העדפות כתוביות"
3. בחרו את השפה המועדפת
4. הפעילו "הצג תמיד כתוביות" אם רצוי

## התאמה אישית של כתוביות

התאימו את מראה הכתוביות לשיפור הקריאות:

### גודל טקסט
בחרו מטקסט קטן, בינוני, גדול או גדול במיוחד.

### סגנון גופן
בחרו מאפשרויות גופן זמינות כולל סגנונות sans-serif ו-serif.

### צבע טקסט
בחרו לבן, צהוב או צבעים אחרים בניגודיות גבוהה.

### רקע
הוסיפו רקע מאחורי הטקסט לנראות טובה יותר:
- ללא רקע
- שקוף למחצה
- רקע מלא

### סגנון קצוות
הוסיפו קו מתאר או אפקטי צל לקצוות הטקסט.

## כתוביות סגורות

חלק מהתוכן מציע כתוביות סגורות (CC) הכוללות תיאורי שמע שאינם דיאלוג כמו אפקטים קוליים ורמזי מוזיקה. חפשו את מחוון ה-CC בזמן עיון.

## תיאור שמע

כותרים נבחרים כוללים רצועות תיאור שמע המספרות אלמנטים חזותיים לצופים עיוורים או עם ראייה לקויה. רצועות זמינות מופיעות בתפריט בחירת השמע בזמן ניגון.

## סיוע לשמיעה

הפעילו התראות חזותיות לצופים חירשים או כבדי שמיעה. תכונה זו מספקת התראות על המסך לרמזי שמע חשובים.

## הגדרות נגישות

גשו לכל אפשרויות הנגישות מההגדרות:
1. עברו להגדרות
2. בחרו "נגישות"
3. הגדירו העדפות כתוביות, כיתוב ותיאור שמע
4. שינויים חלים על הפרופיל שלכם בכל המכשירים
