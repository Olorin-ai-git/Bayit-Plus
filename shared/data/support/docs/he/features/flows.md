# זרימות תוכן

זרימות הן פלייליסטים אצורים ואוספים נושאיים שתוכננו לשפר את חוויית הצפייה שלכם. צוות העריכה שלנו יוצר זרימות סביב נושאים, מצבי רוח, אירועים וטרנדים.

## מהן זרימות

זרימות משלבות תוכן קשור למסעות צפייה מגובשים. במקום לבחור כותרים בודדים, תוכלו לשקוע בזרימה ולתת לתוכן לנגן ברציפות.

דוגמאות לזרימות כוללות:
- קולנוע קלאסי ביום ראשון
- מרתון פשעים אמיתיים
- קומדיות מרגשות
- דוקומנטריים זוכי פרסים
- מיוחדי חגים עונתיים

## איתור זרימות

### זרימות מומלצות
גלו זרימות מודגשות במסך הבית בסעיף "זרימות". אלה מתעדכנות באופן קבוע בהתאם לאירועים נוכחיים, עונות וטרנדים.

### עיון בכל הזרימות
גשו לספריית הזרימות המלאה:
1. נווטו ל"זרימות" מהתפריט הראשי
2. עיינו לפי קטגוריה או נושא
3. צפו בתיאורי הזרימות והתוכן הכלול

### חיפוש זרימות
השתמשו בפונקציית החיפוש למציאת זרימות לפי שם או נושא.

## הפעלת זרימה

1. בחרו זרימה לצפייה בתכניה
2. לחצו על "הפעל זרימה" להתחיל מההתחלה
3. התוכן מתנגן ברציפות דרך האוסף
4. דלגו קדימה כדי לעבור לפריט הבא בזרימה

## פקדי ניגון זרימה

בזמן צפייה בזרימה:
- צפו בתור התוכן הבא
- דלגו לכל פריט בזרימה
- הסירו פריטים שכבר צפיתם בהם
- ערבבו את הסדר
- שמרו את הזרימה להמשך

## יצירת זרימות אישיות

מנויי פרימיום ומשפחתי יכולים ליצור זרימות מותאמות אישית:
1. עברו לספרייה שלי ובחרו "הזרימות שלי"
2. לחצו על "צור זרימה"
3. תנו שם לזרימה
4. הוסיפו סרטים, פרקים או קליפים
5. סדרו את הסדר
6. שמרו והפעילו בכל עת

## שיתוף זרימות

שתפו את הזרימות האהובות עליכם עם חברים ומשפחה:
1. פתחו את הזרימה שברצונכם לשתף
2. לחצו על סמל השיתוף
3. שלחו באמצעות הודעה, אימייל או רשתות חברתיות
4. נמענים עם חשבונות Bayit+ יכולים לצפות ולהפעיל את הזרימה המשותפת

## המלצות זרימות

Bayit+ מציעה זרימות חדשות בהתבסס על היסטוריית הצפייה והעדפותיכם. מצאו המלצות זרימות מותאמות אישית בסעיף "בשבילך".
