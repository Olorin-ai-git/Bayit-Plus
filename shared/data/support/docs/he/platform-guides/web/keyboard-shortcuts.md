# קיצורי מקלדת

שלטו בנגן האינטרנט של Bayit+ עם קיצורי מקלדת אלה לניווט יעיל ושליטה בהפעלה.

## פקדי הפעלה

| קיצור | פעולה |
|-------|-------|
| `רווח` או `K` | הפעלה / השהייה |
| `M` | השתקה / ביטול השתקה |
| `F` | מעבר למסך מלא |
| `Esc` | יציאה ממסך מלא |

## דילוג וניווט

| קיצור | פעולה |
|-------|-------|
| `חץ שמאלה` | אחורה 10 שניות |
| `חץ ימינה` | קדימה 10 שניות |
| `J` | אחורה 10 שניות |
| `L` | קדימה 10 שניות |
| `Home` | מעבר להתחלה |
| `End` | מעבר לסוף |
| `0-9` | קפיצה ל-0%-90% מהסרטון |

## פקדי עוצמת שמע

| קיצור | פעולה |
|-------|-------|
| `חץ למעלה` | הגברת עוצמה ב-5% |
| `חץ למטה` | הנמכת עוצמה ב-5% |
| `M` | מעבר בין השתקה לביטולה |

## כתוביות והגדרות

| קיצור | פעולה |
|-------|-------|
| `C` | הפעלה/כיבוי כתוביות |
| `S` | פתיחת תפריט הגדרות |

## מהירות הפעלה

| קיצור | פעולה |
|-------|-------|
| `Shift + >` | הגברת מהירות הפעלה |
| `Shift + <` | הפחתת מהירות הפעלה |

## טיפים לחוויה טובה יותר

- קיצורים פועלים כאשר נגן הווידאו בפוקוס
- לחצו על אזור הנגן תחילה אם קיצורים לא מגיבים
- השתמשו ב-`Tab` לניווט בין פקדי הנגן
- לחצו על `?` כדי לראות את שכבת הקיצורים בזמן צפייה

## נגישות

כל קיצורי המקלדת מתוכננים לעבוד עם קוראי מסך. הנגן תומך בתכונות נגישות סטנדרטיות כולל מחווני פוקוס ותגיות ARIA לכל הפקדים.
