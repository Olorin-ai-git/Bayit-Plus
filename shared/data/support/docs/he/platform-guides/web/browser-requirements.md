# דרישות דפדפן

Bayit+ מותאמת לדפדפני אינטרנט מודרניים כדי לספק את חוויית הסטרימינג הטובה ביותר עם ניגון וידאו באיכות גבוהה וניווט חלק.

## דפדפנים נתמכים

| דפדפן | גרסה מינימלית | גרסה מומלצת |
|-------|---------------|-------------|
| Google Chrome | 90+ | האחרונה |
| Mozilla Firefox | 88+ | האחרונה |
| Apple Safari | 14+ | האחרונה |
| Microsoft Edge | 90+ | האחרונה |

## דרישות מערכת

### דרישות מינימליות
- **מעבד:** Dual-core 2.0 GHz או שווה ערך
- **זיכרון RAM:** 4 GB
- **אינטרנט:** 5 Mbps לאיכות SD
- **מערכת הפעלה:** Windows 10, macOS 10.14, או Ubuntu 20.04

### דרישות מומלצות
- **מעבד:** Quad-core 2.5 GHz או שווה ערך
- **זיכרון RAM:** 8 GB או יותר
- **אינטרנט:** 25 Mbps לאיכות 4K Ultra HD
- **מערכת הפעלה:** Windows 11, macOS 12, או Ubuntu LTS האחרונה

## הגדרות דפדפן

לביצועים אופטימליים, ודאו שההגדרות הבאות מופעלות:

1. **JavaScript:** חייב להיות מופעל כדי שהנגן יפעל
2. **עוגיות (Cookies):** נדרשות לשמירת הפגישה וההעדפות שלכם
3. **האצת חומרה:** הפעילו לניגון וידאו חלק יותר
4. **חוסם חלונות קופצים:** הוסיפו את Bayit+ לרשימת האתרים המותרים

## טיפים לפתרון בעיות

- נקו את זיכרון המטמון של הדפדפן אם אתם חווים בעיות ניגון
- השביתו תוספי דפדפן שעלולים להפריע לסטרימינג וידאו
- ודאו שהדפדפן שלכם מעודכן לגרסה האחרונה
- בדקו שמנהלי ההתקן הגרפיים שלכם מעודכנים

## דפדפנים לא נתמכים

Internet Explorer וגרסאות ישנות של דפדפנים אינם נתמכים. לחוויה הטובה ביותר, אנא עדכנו לאחד מהדפדפנים הנתמכים המפורטים למעלה.
