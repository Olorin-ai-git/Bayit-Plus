# הגדרת אפליקציית iOS

התחילו עם Bayit+ באייפון או באייפד שלכם ותיהנו מסטרימינג עם תכונות iOS חזקות.

## התקנה

1. פתחו את **App Store** במכשיר ה-iOS שלכם
2. חפשו **Bayit+**
3. לחצו על **קבל** להורדה והתקנה
4. פתחו את האפליקציה והתחברו עם החשבון שלכם

## דרישות מערכת

- iOS 15.0 או מאוחר יותר
- iPhone 8 או חדש יותר, iPad (דור 6) או חדש יותר
- 150 MB שטח אחסון פנוי

## תכונות עיקריות

### אינטגרציה עם Siri
שלטו ב-Bayit+ ללא ידיים עם פקודות Siri:
- "Hey Siri, הפעל משהו ב-Bayit+"
- "Hey Siri, השהה את Bayit+"
- "Hey Siri, פתח את Bayit+"

### סטרימינג AirPlay
הזרימו תוכן ל-Apple TV או למכשירים תואמי AirPlay:
1. התחילו להפעיל תוכן באפליקציה
2. לחצו על סמל AirPlay בנגן
3. בחרו את מכשיר ה-AirPlay מהרשימה

### תמונה בתוך תמונה (PiP)
המשיכו לצפות תוך שימוש באפליקציות אחרות:
1. התחילו להפעיל סרטון
2. החליקו למעלה או לחצו על Home למזעור
3. הסרטון ממשיך בחלון צף
4. לחצו לחזרה למסך מלא

### הורדות לצפייה לא מקוונת
הורידו תוכן לצפייה ללא אינטרנט:
1. מצאו את התוכן שברצונכם להוריד
2. לחצו על סמל ההורדה
3. בחרו איכות וידאו
4. גשו להורדות מלשונית הספרייה

## הגדרות והעדפות

- **איכות וידאו:** התאימו איכות סטרימינג בהגדרות > הפעלה
- **נתונים סלולריים:** שלטו בשימוש בנתונים בהגדרות > הורדות
- **התראות:** נהלו התראות בהגדרות > התראות

## טיפים

- הפעילו Face ID או Touch ID לכניסה מהירה
- השתמשו במצב כהה לצפייה נוחה בלילה
- חברו AirPods למעבר אודיו אוטומטי
