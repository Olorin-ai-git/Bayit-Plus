# הגדרת אפליקציית Android

התחילו עם Bayit+ בטלפון או בטאבלט ה-Android שלכם ותיהנו מחוויית סטרימינג עשירה בתכונות.

## התקנה

1. פתחו את **Google Play Store** במכשיר שלכם
2. חפשו **Bayit+**
3. לחצו על **התקן** להורדה
4. פתחו את האפליקציה והתחברו עם החשבון שלכם

## דרישות מערכת

- Android 8.0 (Oreo) או מאוחר יותר
- 2 GB זיכרון RAM מינימום
- 150 MB שטח אחסון פנוי

## תכונות עיקריות

### אינטגרציה עם Google Assistant
שלטו ב-Bayit+ עם הקול שלכם:
- "Hey Google, פתח את Bayit+"
- "Hey Google, הפעל סרטים ב-Bayit+"
- "Hey Google, עצור"

### תמיכה ב-Chromecast
הזרימו לטלוויזיה שלכם באמצעות Chromecast:
1. ודאו שהמכשיר וה-Chromecast מחוברים לאותה רשת Wi-Fi
2. התחילו להפעיל תוכן באפליקציה
3. לחצו על סמל Cast בנגן
4. בחרו את מכשיר ה-Chromecast שלכם

### תמונה בתוך תמונה (PiP)
צפו בסרטונים תוך ביצוע משימות אחרות:
1. התחילו להפעיל סרטון
2. לחצו על כפתור Home או החווה
3. הסרטון ממשיך בחלון קטן
4. גררו למיקום מחדש או לחצו להרחבה

### הורדות לצפייה לא מקוונת
שמרו תוכן לצפייה ללא אינטרנט:
1. גלשו לתוכן הרצוי
2. לחצו על כפתור ההורדה
3. בחרו את האיכות המועדפת
4. מצאו הורדות בקטע הספרייה

## הגדרות תצורה

- **איכות סטרימינג:** הגדרות > איכות וידאו
- **מיקום הורדה:** הגדרות > אחסון
- **חיסכון בנתונים:** הגדרות > נתונים סלולריים
- **התראות:** הגדרות > התראות

## הרשאות

האפליקציה עשויה לבקש את ההרשאות הבאות:
- **אחסון:** נדרש להורדות לצפייה לא מקוונת
- **מיקרופון:** אופציונלי, לחיפוש קולי

## טיפים

- השתמשו בטביעת אצבע או זיהוי פנים לגישה מהירה
- הפעילו חריגות אופטימיזציית סוללה להורדות רצופות
- חברו אוזניות Bluetooth להאזנה פרטית
