# Atajos de Teclado

Domina el reproductor web de Bayit+ con estos atajos de teclado para una navegacion eficiente y control de reproduccion.

## Controles de Reproduccion

| Atajo | Accion |
|-------|--------|
| `Espacio` o `K` | Reproducir / Pausar |
| `M` | Silenciar / Activar sonido |
| `F` | Alternar Pantalla Completa |
| `Esc` | Salir de Pantalla Completa |

## Navegacion y Desplazamiento

| Atajo | Accion |
|-------|--------|
| `Flecha Izquierda` | Retroceder 10 segundos |
| `Flecha Derecha` | Avanzar 10 segundos |
| `J` | Retroceder 10 segundos |
| `L` | Avanzar 10 segundos |
| `Inicio` | Ir al principio |
| `Fin` | Ir al final |
| `0-9` | Saltar al 0%-90% del video |

## Controles de Volumen

| Atajo | Accion |
|-------|--------|
| `Flecha Arriba` | Aumentar volumen 5% |
| `Flecha Abajo` | Disminuir volumen 5% |
| `M` | Alternar silencio |

## Subtitulos y Configuracion

| Atajo | Accion |
|-------|--------|
| `C` | Activar/desactivar subtitulos |
| `S` | Abrir menu de configuracion |

## Velocidad de Reproduccion

| Atajo | Accion |
|-------|--------|
| `Shift + >` | Aumentar velocidad de reproduccion |
| `Shift + <` | Disminuir velocidad de reproduccion |

## Consejos para una Mejor Experiencia

- Los atajos funcionan cuando el reproductor de video esta enfocado
- Haz clic en el area del reproductor primero si los atajos no responden
- Usa `Tab` para navegar entre los controles del reproductor
- Presiona `?` para ver la superposicion de atajos mientras ves contenido

## Accesibilidad

Todos los atajos de teclado estan disenados para funcionar con lectores de pantalla. El reproductor admite funciones de accesibilidad estandar, incluyendo indicadores de enfoque y etiquetas ARIA para todos los controles.
