# Requisitos del Navegador

Bayit+ esta optimizado para navegadores web modernos para ofrecer la mejor experiencia de streaming con reproduccion de video de alta calidad y navegacion fluida.

## Navegadores Compatibles

| Navegador | Version Minima | Version Recomendada |
|-----------|----------------|---------------------|
| Google Chrome | 90+ | Ultima |
| Mozilla Firefox | 88+ | Ultima |
| Apple Safari | 14+ | Ultima |
| Microsoft Edge | 90+ | Ultima |

## Requisitos del Sistema

### Requisitos Minimos
- **Procesador:** Doble nucleo 2.0 GHz o equivalente
- **RAM:** 4 GB
- **Internet:** 5 Mbps para calidad SD
- **Sistema Operativo:** Windows 10, macOS 10.14, o Ubuntu 20.04

### Requisitos Recomendados
- **Procesador:** Cuatro nucleos 2.5 GHz o equivalente
- **RAM:** 8 GB o mas
- **Internet:** 25 Mbps para 4K Ultra HD
- **Sistema Operativo:** Windows 11, macOS 12, o ultima version LTS de Ubuntu

## Configuracion del Navegador

Para un rendimiento optimo, asegurate de que las siguientes configuraciones esten habilitadas:

1. **JavaScript:** Debe estar habilitado para que el reproductor funcione
2. **Cookies:** Requeridas para mantener tu sesion y preferencias
3. **Aceleracion por Hardware:** Habilitar para una reproduccion de video mas fluida
4. **Bloqueador de Ventanas Emergentes:** Agrega Bayit+ a tu lista de sitios permitidos

## Consejos para Solucionar Problemas

- Limpia la cache de tu navegador si experimentas problemas de reproduccion
- Deshabilita extensiones del navegador que puedan interferir con la transmision de video
- Asegurate de que tu navegador este actualizado a la ultima version
- Verifica que los controladores de graficos esten actualizados

## Navegadores No Compatibles

Internet Explorer y versiones anteriores de navegadores no son compatibles. Para la mejor experiencia, actualiza a uno de los navegadores compatibles listados arriba.
