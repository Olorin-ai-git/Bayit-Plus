# Modo Shabat

El Modo Shabat de Bayit+ proporciona configuraciones especiales para usuarios que observan el Shabat y las festividades. Esta funcion asegura que tu experiencia de Bayit+ se alinee con tus practicas religiosas.

## Acerca del Modo Shabat

El Modo Shabat ajusta automaticamente el comportamiento de la app durante Shabat y festividades para apoyar la observancia. Cuando esta activo, ciertas funciones se modifican o pausan.

## Activar el Modo Shabat

### Activacion Automatica
Configura el modo Shabat para activarse automaticamente:
1. Ve a Configuracion y selecciona "Judaismo"
2. Activa "Modo Shabat Automatico"
3. Confirma tu ubicacion para calculos precisos de horarios
4. El modo se activa al encendido de velas y desactiva despues de Havdala

### Activacion Manual
Activa el modo manualmente en cualquier momento:
1. Abre el menu de configuracion rapida
2. Toca "Activar Modo Shabat"
3. Desactiva manualmente cuando desees

## Comportamiento del Modo Shabat

Cuando el Modo Shabat esta activo:

### Notificaciones
- Las notificaciones push se pausan
- Los recordatorios se retienen hasta despues de Shabat
- Solo alertas criticas de seguridad permanecen activas

### Reproduccion
- El contenido continua reproduciendose si se inicio antes de Shabat
- Reproduccion automatica de siguiente episodio (si esta configurada)
- Los controles de reproduccion permanecen accesibles

### Descargas
- Las descargas programadas se pausan
- Las descargas se reanudan automaticamente despues de Shabat

### Actualizaciones
- Las actualizaciones de la app se posponen
- La sincronizacion de datos en segundo plano se reduce

## Configuracion Avanzada

Personaliza el comportamiento del Modo Shabat:

### Horarios Personalizados
- Ajusta el inicio del Shabat (antes del encendido de velas)
- Configura el fin del Shabat (minutos despues de la salida)
- Configura para festividades de uno o dos dias

### Contenido Pre-Shabat
- Descarga automatica de contenido antes de Shabat
- Cola de reproduccion preparada para Shabat
- Contenido de parasha listo para Shabat

### Excepciones
- Selecciona funciones que permanecen activas
- Configura perfiles especificos para el modo

## Festividades

El Modo Shabat tambien funciona para festividades:
- Se activa automaticamente en Yom Tov
- Reconoce festividades de Israel y diaspora
- Ajusta para segundos dias de festividad (Yom Tov Sheni)

## Ubicacion

Para horarios precisos, Bayit+ necesita tu ubicacion:
- Usa GPS para deteccion automatica
- O ingresa tu ciudad manualmente en configuracion
- Soporta ubicaciones en todo el mundo
