# Controles Parentales

Bayit+ ofrece controles parentales completos para ayudarte a administrar el contenido al que tu familia puede acceder. Estas herramientas te brindan tranquilidad mientras tus hijos disfrutan de entretenimiento apropiado para su edad.

## Configurar tu PIN

Un PIN de 4 digitos protege la configuracion de los controles parentales contra cambios no autorizados.

1. Ve a **Configuracion** > **Controles Parentales**
2. Selecciona **Crear PIN**
3. Ingresa un PIN de 4 digitos que tus hijos no puedan adivinar facilmente
4. Confirma tu PIN ingresandolo nuevamente

Tu PIN sera requerido para acceder a contenido restringido, modificar configuraciones o salir de los perfiles infantiles.

## Restricciones de Contenido

Puedes restringir el contenido segun las clasificaciones de edad para cada perfil:

| Nivel de Restriccion | Clasificaciones Permitidas |
|----------------------|---------------------------|
| Ninos pequenos | Solo G |
| Ninos mayores | G, PG |
| Adolescentes | G, PG, PG-13 |
| Adultos | Todas las clasificaciones |

Para configurar restricciones de contenido:

1. Ve a **Configuracion** > **Controles Parentales** > **Restricciones de Contenido**
2. Selecciona el perfil que deseas modificar
3. Elige el nivel de restriccion apropiado
4. Ingresa tu PIN para confirmar los cambios

## Bloquear Contenido Especifico

Ademas de las clasificaciones por edad, puedes bloquear titulos individuales:

1. Navega al contenido que deseas bloquear
2. Selecciona el menu **Mas Opciones**
3. Elige **Bloquear Este Titulo**
4. Ingresa tu PIN para confirmar

El contenido bloqueado no aparecera en los resultados de busqueda ni en las recomendaciones de los perfiles restringidos.

## Restablecer tu PIN

Si olvidas tu PIN, puedes restablecerlo a traves del correo electronico de tu cuenta:

1. Ve a **Configuracion** > **Controles Parentales**
2. Selecciona **Olvide mi PIN**
3. Revisa tu correo electronico para obtener un codigo de verificacion
4. Ingresa el codigo y crea un nuevo PIN
