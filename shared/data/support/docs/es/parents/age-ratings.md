# Entendiendo las Clasificaciones de Contenido

Bayit+ utiliza clasificaciones de edad estandar de la industria para ayudarte a tomar decisiones informadas sobre la visualizacion para tu familia. Cada pieza de contenido muestra su clasificacion de forma prominente en la pagina de detalles.

## Categorias de Clasificacion

| Clasificacion | Descripcion | Audiencia Tipica |
|---------------|-------------|------------------|
| G | Audiencias generales, apto para todas las edades | Todas las edades |
| PG | Se sugiere orientacion parental | 7+ anos |
| PG-13 | Puede ser inapropiado para menores de 13 | 13+ anos |
| R | Restringido, contenido para adultos | 17+ anos |
| TV-Y | Disenado para ninos pequenos | 2-6 anos |
| TV-Y7 | Disenado para ninos de 7 anos o mas | 7+ anos |
| TV-G | Apto para todas las edades | Todas las edades |
| TV-PG | Se sugiere orientacion parental | 10+ anos |
| TV-14 | Puede ser inadecuado para menores de 14 | 14+ anos |

## Descriptores de Contenido

Ademas de la clasificacion principal, los descriptores proporcionan orientacion especifica:

- **V**: Violencia o violencia de fantasia
- **S**: Contenido sexual o dialogo sugestivo
- **L**: Lenguaje vulgar o grosero
- **D**: Dialogo sugestivo
- **FV**: Violencia de fantasia (programacion infantil)

Estos descriptores aparecen debajo de la clasificacion en las paginas de detalles del contenido.

## Como se Asignan las Clasificaciones

Nuestro equipo de contenido revisa todo el material utilizando pautas establecidas por juntas de clasificacion incluyendo la MPAA y las Pautas Parentales de TV. Los factores considerados incluyen:

- Intensidad y frecuencia de la violencia
- Contenido de lenguaje y dialogo
- Elementos tematicos y temas maduros
- Representaciones de uso de sustancias

## Usar las Clasificaciones con los Controles Parentales

Las clasificaciones se integran directamente con los controles parentales:

1. Establece una clasificacion maxima para cada perfil
2. El contenido por encima de esa clasificacion se oculta automaticamente
3. Intentar acceder a contenido restringido requiere tu PIN

## Diferencias Regionales en las Clasificaciones

Algunos contenidos pueden tener diferentes clasificaciones en diferentes regiones. Bayit+ muestra la clasificacion apropiada para tu ubicacion. Si viajas, las clasificaciones pueden ajustarse segun los estandares locales.

Para preguntas sobre clasificaciones de contenido especificas, contacta a nuestro equipo de soporte a traves del Centro de Ayuda.
