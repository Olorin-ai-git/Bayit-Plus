# Administracion del Tiempo de Pantalla

Bayit+ incluye controles de tiempo de pantalla integrados para ayudarte a establecer habitos de visualizacion saludables para tu familia. Configura limites diarios, programa horarios de visualizacion y recibe alertas cuando los limites esten por alcanzarse.

## Configurar Limites de Tiempo Diarios

Configura el tiempo maximo de visualizacion por dia para cada perfil:

1. Ve a **Configuracion** > **Controles Parentales** > **Tiempo de Pantalla**
2. Selecciona el perfil a configurar
3. Activa **Limite de Tiempo Diario**
4. Establece las horas y minutos maximos
5. Ingresa tu PIN para confirmar

Cuando se alcance el limite, la reproduccion se pausa y aparece un mensaje amigable explicando que el tiempo de visualizacion ha terminado por el dia.

## Crear Horarios de Visualizacion

Restringe la visualizacion a ventanas de tiempo especificas:

1. Navega a **Tiempo de Pantalla** > **Horario de Visualizacion**
2. Selecciona los dias de la semana a configurar
3. Establece las horas de visualizacion permitidas (por ejemplo, 4:00 PM - 7:00 PM)
4. Guarda tu horario

Fuera de las horas programadas, el perfil muestra un mensaje indicando cuando la visualizacion estara disponible nuevamente.

## Modo Hora de Dormir

Detiene automaticamente la reproduccion a una hora de dormir designada:

1. Activa **Modo Hora de Dormir** en la configuracion de Tiempo de Pantalla
2. Establece la hora de dormir para cada dia
3. Elige un periodo de aviso (5, 10 o 15 minutos)

Durante el periodo de aviso, aparece un recordatorio suave. A la hora de dormir, el episodio actual termina y no puede iniciarse nuevo contenido.

## Reportes de Tiempo de Pantalla

Revisa los habitos de visualizacion a traves de reportes detallados:

- **Resumenes diarios**: Tiempo visto por perfil cada dia
- **Tendencias semanales**: Patrones en el comportamiento de visualizacion
- **Desglose de contenido**: Categorias y titulos mas vistos

Accede a los reportes desde **Configuracion** > **Controles Parentales** > **Reportes de Actividad**.

## Extender el Tiempo

Para ocasiones especiales, puedes otorgar extensiones temporales:

1. Cuando aparezca el mensaje de limite de tiempo, selecciona **Solicitar Mas Tiempo**
2. Se envia una notificacion a la cuenta del padre
3. Aprueba la solicitud e ingresa tu PIN
4. Elige la duracion de la extension (15, 30 o 60 minutos)
