# Gestion de Campanas

Crea y administra campanas de correo electronico y notificaciones push para interactuar con tu audiencia de Bayit+. Las campanas ayudan a promocionar nuevo contenido, reactivar usuarios inactivos y comunicar actualizaciones importantes.

## Tipos de Campanas

**Campanas de Correo Electronico**: Correos HTML enriquecidos para comunicaciones detalladas incluyendo anuncios de nuevos lanzamientos, boletines informativos y actualizaciones de cuenta.

**Notificaciones Push**: Mensajes cortos e inmediatos entregados a las aplicaciones moviles y de TV para promociones de contenido sensibles al tiempo.

**Mensajes en la Aplicacion**: Banners y modales mostrados dentro de la aplicacion para promociones contextuales.

## Creacion de una Campana

1. Navega a **Campanas** > **Crear Nueva**
2. Selecciona el tipo de campana
3. Define tu audiencia objetivo
4. Disena el contenido de tu mensaje
5. Programa el momento de entrega
6. Revisa y lanza

## Segmentacion de Audiencia

Segmenta tu audiencia utilizando estos criterios:

| Criterio | Ejemplos |
|----------|----------|
| Actividad | Activos, inactivos, usuarios nuevos |
| Preferencias | Intereses de genero, idioma |
| Suscripcion | Gratuita, premium, prueba |
| Comportamiento | Series completadas, abandonadas |
| Demografia | Rango de edad, ubicacion |

Combina multiples criterios para una segmentacion precisa.

## Personalizacion de Contenido

Los campos dinamicos personalizan automaticamente los mensajes:

- `{{user.name}}`: Nombre de pantalla del destinatario
- `{{content.title}}`: Titulo del contenido recomendado
- `{{content.image}}`: Imagen del poster del contenido
- `{{profile.name}}`: Nombre del perfil activo

## Pruebas A/B

Prueba variaciones para optimizar el rendimiento:

1. Crea dos o mas variantes del mensaje
2. Define el porcentaje de audiencia de prueba
3. Selecciona la metrica de exito (aperturas, clics, conversiones)
4. Lanza la prueba
5. Revisa los resultados e implementa el ganador

## Analiticas de Campanas

Rastrea el rendimiento de las campanas con metricas detalladas:

- Tasa de entrega y estadisticas de rebote
- Tasas de apertura y clics
- Seguimiento de conversiones a vistas de contenido
- Tasas de cancelacion de suscripcion

## Plantillas de Campanas

Guarda disenos utilizados frecuentemente como plantillas:

1. Disena tu campana
2. Haz clic en **Guardar como Plantilla**
3. Nombra y categoriza la plantilla
4. Reutiliza para futuras campanas

Las plantillas mantienen una marca consistente en las comunicaciones.
