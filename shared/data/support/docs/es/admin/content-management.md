# Gestion de Contenido

El sistema de Gestion de Contenido permite a los administradores agregar, editar, organizar y eliminar contenido multimedia de la biblioteca de Bayit+. Esta guia cubre el ciclo de vida completo del contenido.

## Vista General de la Biblioteca de Contenido

Accede a la biblioteca de contenido desde **Contenido** > **Biblioteca**. La biblioteca muestra todo el contenido con opciones de filtrado:

- **Tipo**: Peliculas, Series, Canales, Podcasts
- **Estado**: Publicado, Borrador, Procesando, Archivado
- **Clasificacion**: Filtrar por clasificacion de edad
- **Categoria**: Navegar por genero o categoria

## Agregar Nuevo Contenido

Para agregar contenido a la biblioteca:

1. Navega a **Contenido** > **Agregar Nuevo**
2. Selecciona el tipo de contenido (Pelicula, Serie, Canal, Podcast)
3. Carga los archivos multimedia o proporciona URLs de streaming
4. Completa el formulario de metadatos (titulo, descripcion, elenco, clasificaciones)
5. Carga las imagenes (poster, fondo, miniaturas)
6. Configura las opciones de disponibilidad y programacion
7. Envia para procesamiento

## Edicion de Contenido Existente

Modifica los detalles del contenido en cualquier momento:

1. Localiza el contenido en la biblioteca
2. Haz clic en el boton **Editar**
3. Actualiza cualquier campo de metadatos
4. Los cambios en campos que no son multimedia se publican inmediatamente
5. Los cambios en archivos multimedia activan el reprocesamiento

## Metadatos de Contenido

Metadatos requeridos para todo el contenido:

| Campo | Descripcion |
|-------|-------------|
| Titulo | Nombre de visualizacion en todos los idiomas |
| Descripcion | Sinopsis mostrada en la pagina de detalles |
| Clasificacion | Clasificacion de edad (G, PG, PG-13, R) |
| Categorias | Clasificaciones de genero y tema |
| Duracion | Duracion en minutos |
| Ano de Lanzamiento | Fecha de lanzamiento original |

## Programacion de Contenido

Controla cuando el contenido esta disponible:

- **Inmediato**: Publicar tan pronto como se complete el procesamiento
- **Programado**: Establecer una fecha y hora de publicacion especifica
- **Ventana**: Definir fechas de inicio y fin de disponibilidad

## Eliminacion de Contenido

Para eliminar contenido de la plataforma:

1. Selecciona el contenido en la biblioteca
2. Haz clic en **Acciones** > **Archivar**
3. El contenido archivado se oculta pero se preserva
4. Para eliminacion permanente, contacta a tu administrador del sistema

Archivar preserva el historial de visualizacion y permite la restauracion futura.
