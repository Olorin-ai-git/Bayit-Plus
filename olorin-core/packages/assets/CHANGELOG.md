# @olorin/assets Changelog

## [2.0.0] - 2026-01-22

### Added
- Initial release of centralized asset package
- Favicon support (8 sizes: 16x16, 32x32, 64x64, 128x128, 192x192, 512x512, apple-touch-icon, favicon.ico)
- Wizard logo variants (main, fraud, streaming, radio, omen)
- Olorin brand logos (primary and text)
- TypeScript constants for all asset paths
- Utility functions for asset loading
- Favicon metadata generator
- Complete type safety with TypeScript

### Migration
- Consolidated 11.2MB of duplicated favicons across 5 portals
- Consolidated 1.26MB of duplicated logos across 4 portals
- Single source of truth for all brand assets
