üîå API Endpoints

RESTful API contract with comprehensive request/response schemas for all investigation operations.
POST /api/investigations

Create new hybrid graph investigation

GET /api/investigations/{id}/status

Poll investigation status (2-second intervals)

GET /api/investigations/{id}/results

Fetch complete investigation results

POST /api/investigations/{id}/export

Generate export (PDF, JSON, CSV)

PATCH /api/investigations/{id}/control

Control investigation (pause/stop)

üóÑÔ∏è Database Schema

Schema-locked database design with no DDL operations, using only existing columns.

investigation_states Table

investigation_id VARCHAR(255) PRIMARY KEY user_id VARCHAR(255) NOT NULL lifecycle_stage VARCHAR(50) NOT NULL -- CREATED|SETTINGS|IN_PROGRESS|COMPLETED settings_json TEXT -- Investigation configuration progress_json TEXT -- Current progress state results_json TEXT -- Final results status VARCHAR(50) NOT NULL -- Status enum version INTEGER NOT NULL -- Optimistic locking created_at TIMESTAMP NOT NULL updated_at TIMESTAMP NOT NULL last_accessed TIMESTAMP

Schema Verification

  • Startup verification queries information_schema
  • Validates all required columns exist
  • Fails fast if schema mismatch detected
  • No DDL operations or auto-migrations
