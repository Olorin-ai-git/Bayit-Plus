# @olorin/shared-services

## 2.0.0

### Major Changes

- ## Initial Release - Olorin Ecosystem Unification

  This is the initial release of the unified Olorin.ai ecosystem packages.

  ### @olorin/design-tokens v1.0.0
  - Centralized design tokens for the Olorin ecosystem
  - Dark purple theme with glassmorphism effects
  - Multi-script typography support (Hebrew, CJK, Devanagari, Tamil, Bengali)
  - Tailwind CSS preset for easy integration
  - Colors, spacing, typography, shadows, and animations

  ### @olorin/shared-hooks v1.0.0
  - Reusable React hooks for all Olorin platforms
  - Platform-agnostic implementations
  - Includes authentication, content, and UI hooks

  ### @olorin/shared-i18n v1.0.0
  - Internationalization support for 9 languages
  - RTL support for Hebrew and Arabic
  - Platform-specific utilities for web and React Native
  - Locale management and formatting utilities

  ### @olorin/shared-services v1.0.0
  - Common service utilities for all platforms
  - API client helpers
  - Storage and caching utilities
  - Platform detection and environment helpers

  ### @olorin/shared-stores v1.0.0
  - Zustand-based state management
  - Cross-platform store implementations
  - Typed store definitions
  - Persistence middleware support
