COMPONENT REFERENCE

1. TVVoiceIndicator
   Purpose: Shows microphone listening status
   State: isListening, isProcessing
   Animation: Pulsing scale effect
   Sizes: small (60x60), medium (80x80), large (120x120)

2. TVVoiceResponseDisplay
   Purpose: Shows voice command responses
   State: lastResponse, error
   Duration: Auto-dismiss 5s (configurable)
   Types: Success, Error

3. TVVoiceCommandHistory
   Purpose: Lists recent voice commands
   State: commandHistory (last 5)
   Items: Focusable list (80pt height)
   Status: Success/failure icons

4. TVVoiceSettings
   Purpose: Configure voice system
   Controls: Language, wake word, TTS rate
   All: Fully focusable
   Languages: he, en, es, fr

5. TVProactiveSuggestionBanner
   Purpose: Shows voice command suggestions
   Items: 3 suggestions with icons
   Animation: Height visibility toggle
   Interaction: Dismissible

6. TVVoiceWaveform
   Purpose: Visualizes audio levels
   Bars: 12 animated (customizable)
   Level: 0-1 range
   Effect: Smooth interpolation

7. TVVoiceErrorAlert
   Purpose: Displays error messages
   Types: 8 error types with mappings
   Actions: Retry, Dismiss
   Size: 400x300pt minimum

8. TVVoicePermissionsScreen
   Purpose: Permission request flow
   Steps: 4 steps (intro, mic, speech, complete)
   Progress: Visual indicator
   Animation: Fade transitions
