# Configuración de Perfil

Personaliza tu experiencia de Bayit+ con la configuración de perfil.

## Administrar Perfiles

### Crear un Perfil

1. Abre la app de Bayit+
2. Toca el ícono de tu perfil
3. Toca **Agregar Perfil**
4. Ingresa nombre del perfil
5. Elige un avatar
6. Configura como perfil de Niños si es necesario
7. Guarda

### Editar un Perfil

1. Toca el ícono del perfil
2. Toca **Administrar Perfiles**
3. Selecciona el perfil a editar
4. Modifica nombre, avatar o configuración
5. Guarda cambios

### Eliminar un Perfil

1. Ve a **Administrar Perfiles**
2. Selecciona el perfil a eliminar
3. Toca **Eliminar Perfil**
4. Confirma eliminación

**Nota:** Eliminar un perfil elimina todas las preferencias e historial de visualización.

## Personalización del Perfil

### Opciones de Avatar

Elige entre:
- Avatares de personajes predefinidos
- Fondos de colores
- Subida de imagen personalizada (Premium)
- Avatares animados (Premium)

### Nombre para Mostrar

- Máximo 20 caracteres
- Caracteres especiales permitidos
- Emojis soportados
- Visible para miembros de la familia

### Configuración de Idioma

Configura idioma por perfil:

1. Abre configuración del perfil
2. Toca **Idioma**
3. Elige:
   - **Idioma de la App**: Idioma de la interfaz
   - **Idioma de Contenido**: Audio/subtítulos predeterminados
4. Guarda

Disponible: Inglés, Hebreo, Español

## Preferencias

### Preferencias de Contenido

Ayúdanos a recomendar mejor contenido:

1. Ve a **Perfil** > **Preferencias**
2. Selecciona géneros favoritos
3. Califica contenido que has visto
4. Marca contenido como "no me interesa"

### Clasificaciones de Madurez

Controla la visibilidad del contenido:

1. Ve a **Perfil** > **Configuración de Madurez**
2. Establece clasificación máxima:
   - Niños (G, TV-Y)
   - Adolescentes (PG-13, TV-14)
   - Adultos (R, TV-MA)
   - Todo el contenido
3. Ingresa PIN para confirmar

### Configuración de Reproducción Automática

| Configuración | Descripción |
|---------------|-------------|
| Vista Previa Auto | Videos se reproducen mientras navegas |
| Siguiente Episodio | Reproduce automáticamente el siguiente episodio |
| Saltar Intro | Salta automáticamente intros de series |
| Salir en Créditos | Sale del reproductor en los créditos |

## Perfiles de Niños

### Crear un Perfil de Niños

1. Agrega un nuevo perfil
2. Activa **Perfil de Niños**
3. Selecciona rango de edad
4. Elige avatar
5. Guarda

### Características del Perfil de Niños

- Solo contenido apropiado para la edad
- Interfaz simplificada
- Sin acceso a configuración
- Sin opciones de pago
- Búsqueda segura activada

### Protección con PIN

Protege perfiles de adultos:

1. Ve a configuración del perfil adulto
2. Toca **Bloqueo de Perfil**
3. Establece PIN de 4 dígitos
4. Requerido para entrar al perfil

## Configuración de Reproducción

### Calidad de Video

Preferencias de calidad por perfil:

| Configuración | Mejor Para |
|---------------|------------|
| Auto | Se adapta a la conexión |
| Bajo | Ahorrar datos |
| Medio | Equilibrado |
| Alto | Mejor imagen |
| Ultra | Pantallas 4K |

### Configuración de Audio

- Idioma de audio predeterminado
- Activar/desactivar sonido envolvente
- Normalización de volumen
- Audio descriptivo predeterminado

### Configuración de Subtítulos

- Idioma de subtítulos predeterminado
- Estilo de subtítulos:
  - Tamaño de fuente (Pequeño/Medio/Grande)
  - Color de fuente
  - Opacidad del fondo
  - Posición

## Notificaciones

### Tipos de Notificaciones

Controla qué notificaciones recibes:

| Tipo | Descripción |
|------|-------------|
| Nuevos Lanzamientos | Cuando llega nuevo contenido |
| Continuar Viendo | Recordatorios para continuar |
| Recomendaciones | Sugerencias personalizadas |
| Expira Pronto | Descargas a punto de expirar |
| Cuenta | Actualizaciones importantes de cuenta |

### Administrar Notificaciones

1. Ve a **Perfil** > **Notificaciones**
2. Activa/desactiva cada tipo de notificación
3. Configura horas silenciosas si lo deseas
4. Elige correo vs push

## Configuración de Privacidad

### Historial de Visualización

- Ve tu historial completo
- Elimina elementos individuales
- Borra todo el historial
- Pausa historial (modo privado)

### Datos y Privacidad

1. **Descarga tus Datos**: Solicita todos los datos
2. **Eliminar Cuenta**: Eliminación permanente
3. **Personalización de Anuncios**: Activa/desactiva anuncios dirigidos
4. **Analíticas**: Excluirse de analíticas

### Actividad de Visualización

Ve dispositivos y sesiones:

1. Ve a **Configuración** > **Seguridad**
2. Toca **Actividad Reciente**
3. Ve todos los dispositivos conectados
4. Cierra sesión en dispositivos específicos
5. Cierra sesión en todos los dispositivos

## Sincronización Entre Dispositivos

### Qué Se Sincroniza

- Historial de visualización
- Mi Lista
- Preferencias
- Posición de continuar viendo
- Calificaciones y reseñas

### Problemas de Sincronización

Si no sincroniza:
1. Verifica conexión a internet
2. Verifica la misma cuenta en todos los dispositivos
3. Fuerza actualización (desliza hacia abajo)
4. Cierra sesión y vuelve a iniciar

## Configuración de Cuenta

### Correo y Contraseña

1. Ve a **Configuración** > **Cuenta**
2. Actualiza dirección de correo
3. Cambia contraseña
4. Verifica por correo

### Número de Teléfono

Agrega para recuperación de cuenta:
1. Ve a **Configuración** > **Cuenta**
2. Toca **Número de Teléfono**
3. Ingresa y verifica por SMS

### Cuentas Vinculadas

Administra inicios de sesión sociales:
- Google
- Apple
- Facebook

Agrega o elimina cuentas vinculadas en configuración de Seguridad.

## Accesibilidad

### Visual

- Modo de alto contraste
- Texto más grande
- Soporte de lector de pantalla
- Movimiento reducido

### Audio

- Descripciones de audio
- Subtítulos cerrados predeterminados
- Audio mono
- Control por voz

### Navegación

- Indicadores de enfoque
- Navegación por teclado
- Comandos de voz

## Exportar tus Datos

### Solicitar Exportación de Datos

1. Ve a **Configuración** > **Privacidad**
2. Toca **Descargar tus Datos**
3. Confirma solicitud
4. Recibe correo cuando esté listo
5. Descarga dentro de 7 días

### Datos Incluidos

- Información de cuenta
- Historial de visualización
- Preferencias
- Historial de pagos
- Interacciones de soporte
