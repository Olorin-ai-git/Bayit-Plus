# Fiestas de Visualizacion

Ve contenido junto con amigos y familia aunque esten lejos. Las fiestas de visualizacion sincronizan la reproduccion para que todos vean el mismo momento juntos.

## Como Funcionan las Fiestas de Visualizacion

Las fiestas de visualizacion permiten que multiples usuarios transmitan el mismo contenido simultaneamente con reproduccion sincronizada. Los participantes pueden chatear y reaccionar mientras ven.

## Requisitos

- Todos los participantes necesitan cuentas activas de Bayit+
- El contenido debe estar disponible para todos los participantes
- Se requiere conexion estable a internet
- Disponible en dispositivos moviles, web y TV

## Iniciar una Fiesta de Visualizacion

### Crear una Sesion
1. Abre el contenido que quieres ver
2. Toca el icono de "Fiesta de Visualizacion"
3. Selecciona "Crear Fiesta"
4. Elige participantes de tu lista de amigos
5. Se envian invitaciones automaticamente

### Compartir Enlace
Alternativamente, genera un enlace para compartir:
1. Copia el enlace de la fiesta de visualizacion
2. Envia via mensaje, correo electronico o redes sociales
3. Los destinatarios hacen clic para unirse

## Unirse a una Fiesta de Visualizacion

Cuando recibes una invitacion:
1. Toca la notificacion o enlace
2. Confirma que quieres unirte
3. Tu reproduccion se sincroniza con el anfitrion
4. Comienza a ver con el grupo

## Durante la Fiesta de Visualizacion

### Reproduccion Sincronizada
- Todos los participantes ven el mismo momento
- Pausar afecta a todos los espectadores
- Saltar adelante o atras se sincroniza entre participantes

### Funciones de Chat
- Chat de texto en tiempo real durante la visualizacion
- Panel de chat redimensionable en pantalla
- Indicadores de escritura muestran cuando otros estan tipeando

### Reacciones
Envia reacciones rapidas que aparecen en la pantalla de todos:
- Emojis y emoticones
- Animaciones de reaccion
- Efectos de sonido (si estan habilitados)

## Controles del Anfitrion

El anfitrion de la fiesta de visualizacion tiene controles adicionales:
- Pausar y reanudar para todos
- Saltar a puntos especificos
- Expulsar participantes si es necesario
- Finalizar la sesion de fiesta de visualizacion

## Configuracion de Fiestas de Visualizacion

Ajusta preferencias en Configuracion:
- Permitir unirse a cualquiera vs solo amigos
- Habilitar/deshabilitar sonidos de reaccion
- Configurar nivel de madurez del contenido
- Configurar notificaciones

## Limitaciones

- Numero maximo de participantes: 8 por fiesta
- Algunos contenidos pueden tener restricciones de fiestas de visualizacion debido a licencias
- Los usuarios de prueba gratuita pueden tener acceso limitado a fiestas de visualizacion
