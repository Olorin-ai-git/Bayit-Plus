# Perfiles de Usuario

Crea perfiles personalizados para cada miembro de tu hogar. Cada perfil mantiene sus propias preferencias, historial de visualizacion y recomendaciones.

## Limites de Perfiles por Plan

El numero de perfiles depende de tu suscripcion:
- **Basico**: 1 perfil
- **Estandar**: Hasta 3 perfiles
- **Premium**: Hasta 5 perfiles
- **Familiar**: Hasta 6 perfiles

## Crear un Perfil

1. Ve a Configuracion y selecciona "Administrar Perfiles"
2. Toca "Agregar Perfil"
3. Ingresa un nombre para el perfil
4. Elige un avatar de las opciones disponibles
5. Configura las preferencias del perfil (opcional)
6. Guarda el nuevo perfil

## Configuracion del Perfil

Cada perfil puede personalizarse con:

### Avatar
Selecciona de una galeria de imagenes de avatar para personalizar el icono de tu perfil.

### Preferencia de Idioma
Configura el idioma preferido para la interfaz y recomendaciones de contenido.

### Clasificacion de Madurez
Controla la clasificacion maxima de contenido visible en este perfil.

### Reproduccion Automatica
Habilita o deshabilita la reproduccion automatica de siguientes episodios y avances.

### Configuracion de Subtitulos
Configura el idioma preferido de subtitulos y configuracion de visualizacion.

## Perfiles para Ninos

Crea perfiles dedicados para ninos con caracteristicas de seguridad mejoradas:
- Contenido limitado a clasificaciones apropiadas para la edad
- Interfaz curada solo para ninos
- Acceso restringido a configuracion de cuenta
- Navegacion simplificada

Para crear un perfil de ninos, activa "Este es un perfil de ninos" al configurar un nuevo perfil.

## PIN de Perfil

Protege los perfiles con un codigo PIN para prevenir acceso no autorizado:
1. Selecciona un perfil desde Administrar Perfiles
2. Habilita "Bloqueo de Perfil"
3. Crea un PIN de 4 digitos
4. Confirma el PIN

## Cambiar de Perfil

Cambia de perfil en cualquier momento:
- Toca el icono de tu perfil en la app
- Selecciona un perfil diferente de la lista
- Los perfiles de ninos pueden requerir PIN de padre para salir

## Eliminar un Perfil

Elimina perfiles que ya no necesitas:
1. Ve a Administrar Perfiles
2. Selecciona el perfil a eliminar
3. Confirma la eliminacion

El perfil principal de la cuenta no puede eliminarse.
