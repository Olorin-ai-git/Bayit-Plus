# Gestion de Usuarios

La seccion de Gestion de Usuarios proporciona herramientas para administrar cuentas de suscriptores, roles de administrador y permisos de acceso en toda la plataforma Bayit+.

## Directorio de Usuarios

Accede al directorio de usuarios desde **Usuarios** > **Todos los Usuarios**. El directorio soporta:

- **Busqueda**: Encuentra usuarios por correo electronico, nombre o ID de cuenta
- **Filtros**: Segmenta por estado de suscripcion, fecha de creacion o actividad
- **Acciones Masivas**: Selecciona multiples usuarios para operaciones por lotes
- **Exportacion**: Descarga listas de usuarios para analisis externo

## Visualizacion de Detalles del Usuario

Haz clic en cualquier usuario para ver su perfil completo:

- Informacion de cuenta y estado de suscripcion
- Perfiles creados bajo la cuenta
- Historial de visualizacion y preferencias
- Historial de tickets de soporte
- Informacion de pagos y facturacion

## Acciones de Cuenta

Acciones disponibles para cuentas individuales:

| Accion | Descripcion |
|--------|-------------|
| Restablecer Contrasena | Enviar correo de restablecimiento de contrasena |
| Suspender Cuenta | Deshabilitar temporalmente el acceso |
| Cancelar Suscripcion | Terminar la suscripcion de pago |
| Emitir Credito | Aplicar credito a la cuenta |
| Fusionar Cuentas | Combinar cuentas duplicadas |

## Roles de Administrador

Asigna permisos granulares a los miembros del equipo de administracion:

- **Super Administrador**: Acceso completo a la plataforma
- **Gestor de Contenido**: Solo biblioteca de contenido y cargas
- **Agente de Soporte**: Cuentas de usuario y tickets de soporte
- **Analista**: Analiticas e informes (solo lectura)
- **Marketing**: Solo gestion de campanas

## Creacion de Cuentas de Administrador

Para agregar un nuevo administrador:

1. Navega a **Usuarios** > **Administradores**
2. Haz clic en **Agregar Administrador**
3. Ingresa la direccion de correo electronico y nombre
4. Selecciona el rol apropiado
5. El nuevo administrador recibe un correo de invitacion

## Seguridad y Registros de Auditoria

Todas las acciones administrativas se registran por seguridad:

- Modificaciones de cuentas de usuario
- Cambios de contenido
- Actualizaciones de permisos
- Intentos de inicio de sesion y sesiones

Accede a los registros de auditoria desde **Configuracion** > **Seguridad** > **Registro de Auditoria**.

## Operaciones Masivas de Usuarios

Realiza acciones en multiples usuarios simultaneamente:

1. Usa filtros para segmentar los usuarios objetivo
2. Selecciona usuarios con las casillas de verificacion
3. Elige una accion masiva del menu desplegable
4. Confirma la accion con tu PIN de administrador
