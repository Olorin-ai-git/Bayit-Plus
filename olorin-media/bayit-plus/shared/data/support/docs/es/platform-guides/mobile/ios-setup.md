# Configuracion de la App para iOS

Comienza con Bayit+ en tu iPhone o iPad y disfruta de streaming con potentes funciones de iOS.

## Instalacion

1. Abre la **App Store** en tu dispositivo iOS
2. Busca **Bayit+**
3. Toca **Obtener** para descargar e instalar
4. Abre la app e inicia sesion con tu cuenta

## Requisitos del Sistema

- iOS 15.0 o posterior
- iPhone 8 o mas reciente, iPad (6ta generacion) o mas reciente
- 150 MB de espacio de almacenamiento libre

## Funciones Principales

### Integracion con Siri
Controla Bayit+ con las manos libres usando comandos de Siri:
- "Oye Siri, reproduce algo en Bayit+"
- "Oye Siri, pausa Bayit+"
- "Oye Siri, abre Bayit+"

### Streaming con AirPlay
Transmite contenido a tu Apple TV o dispositivos compatibles con AirPlay:
1. Comienza a reproducir contenido en la app
2. Toca el icono de AirPlay en el reproductor
3. Selecciona tu dispositivo AirPlay de la lista

### Picture-in-Picture (PiP)
Continua viendo mientras usas otras apps:
1. Comienza a reproducir un video
2. Desliza hacia arriba o presiona Inicio para minimizar
3. El video continua en una ventana flotante
4. Toca para volver a pantalla completa

### Descargas sin Conexion
Descarga contenido para ver sin internet:
1. Encuentra el contenido que deseas descargar
2. Toca el icono de descarga
3. Selecciona la calidad de video
4. Accede a las descargas desde tu pestana Biblioteca

## Configuracion y Preferencias

- **Calidad de Video:** Ajusta la calidad de streaming en Configuracion > Reproduccion
- **Datos Celulares:** Controla el uso de datos en Configuracion > Descargas
- **Notificaciones:** Administra alertas en Configuracion > Notificaciones

## Consejos

- Habilita Face ID o Touch ID para iniciar sesion rapidamente
- Usa el Modo Oscuro para una visualizacion nocturna comoda
- Conecta AirPods para cambio automatico de audio
