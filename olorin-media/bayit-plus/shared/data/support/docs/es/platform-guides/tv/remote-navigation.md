# Guia de Navegacion con Control Remoto

Aprende como navegar Bayit+ en tu TV usando tu control remoto para una experiencia de visualizacion fluida.

## Navegacion con D-Pad

El pad direccional (D-pad) es tu herramienta principal de navegacion en TV.

### Movimiento Basico
- **Arriba/Abajo:** Moverse entre filas y elementos del menu
- **Izquierda/Derecha:** Navegar por los carruseles de contenido
- **OK/Seleccionar:** Elegir el elemento resaltado o confirmar accion
- **Atras:** Volver a la pantalla anterior o cerrar menu

### En el Navegador de Contenido
1. Usa Arriba/Abajo para moverte entre filas de contenido
2. Usa Izquierda/Derecha para desplazarte por los elementos de una fila
3. Presiona OK para ver los detalles del contenido
4. Presiona OK nuevamente para iniciar la reproduccion

## Controles de Reproduccion

### Durante la Reproduccion de Video
| Boton | Accion |
|-------|--------|
| OK | Mostrar/ocultar controles de reproduccion |
| Reproducir/Pausar | Alternar reproduccion |
| Izquierda | Retroceder 10 segundos |
| Derecha | Avanzar 10 segundos |
| Arriba | Mostrar informacion del video |
| Abajo | Acceder a configuracion de audio y subtitulos |
| Atras | Salir a la pantalla de navegacion |

### Busqueda en el Video
1. Presiona OK para mostrar la barra de progreso
2. Usa Izquierda/Derecha para desplazarte por la linea de tiempo
3. Presiona OK para reanudar desde la posicion seleccionada

## Control por Voz

### Apple TV (Siri)
Manten presionado el boton de Siri y di tu comando:
- "Avanza rapido 5 minutos"
- "Que dijeron?" (retrocede 15 segundos con subtitulos)
- "Reproduce algo para ninos"

### Android TV (Google Assistant)
Presiona el boton del microfono y habla:
- "Salta la introduccion"
- "Muestrame documentales"
- "Activa subtitulos en hebreo"

## Navegacion del Menu

### Menu Principal
- Usa el D-pad para resaltar categorias del menu
- Presiona OK para entrar a una categoria
- Presiona Atras para volver al menu principal

### Menu de Configuracion
1. Navega al icono de tu perfil
2. Presiona Abajo para acceder a Configuracion
3. Usa el D-pad para navegar por las opciones
4. Presiona OK para alternar o seleccionar elementos

## Consejos para Navegacion Facil

- Manten presionado Izquierda/Derecha para desplazamiento continuo en filas largas
- Usa los botones numericos (si estan disponibles) para saltar a posiciones especificas durante la reproduccion
- Manten presionado Atras para volver directamente a la pantalla de inicio en algunos dispositivos
- Habilita "Reducir Movimiento" en configuracion si las animaciones dificultan la navegacion

## Accesibilidad

- El soporte para lectores de pantalla esta disponible para usuarios con discapacidad visual
- El modo de alto contraste puede habilitarse en configuracion de pantalla
- Las descripciones de audio estan disponibles para contenido seleccionado
