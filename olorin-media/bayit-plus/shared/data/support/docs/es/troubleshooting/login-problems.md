# Solución de Problemas de Inicio de Sesión

¿No puedes acceder a tu cuenta? Sigue estos pasos para resolver problemas de inicio de sesión.

## Problemas Comunes de Inicio de Sesión

### Olvidé mi Contraseña

1. Toca **Olvidé mi contraseña** en la pantalla de inicio de sesión
2. Ingresa tu dirección de correo electrónico
3. Revisa tu bandeja de entrada (y carpeta de spam)
4. Haz clic en el enlace de restablecimiento dentro de 24 horas
5. Crea una nueva contraseña
6. Inicia sesión con la nueva contraseña

**Consejos:**
- La contraseña debe tener al menos 8 caracteres
- Incluye mayúsculas, minúsculas y números
- Evita contraseñas comunes
- No reutilices contraseñas de otros sitios

### Contraseña Incorrecta

Si estás seguro de tu contraseña:

1. Verifica que Bloq Mayús esté desactivado
2. Verifica el idioma del teclado
3. Intenta escribir la contraseña en un campo de texto primero
4. Limpia el campo de contraseña y vuelve a escribir lentamente
5. Usa un gestor de contraseñas si está disponible

### Cuenta Bloqueada

Después de múltiples intentos fallidos, las cuentas se bloquean temporalmente:

1. Espera 30 minutos
2. Intenta de nuevo con la contraseña correcta
3. Usa "Olvidé mi contraseña" si no estás seguro
4. Contacta soporte si sigue bloqueada

### Correo No Reconocido

Si aparece "correo no encontrado":

1. Verifica errores de escritura en el correo
2. Prueba direcciones de correo alternativas
3. Verifica si te registraste con número de teléfono
4. Prueba inicio de sesión social (Google, Apple)
5. Crea una nueva cuenta si es necesario

## Autenticación de Dos Factores

### No Puedo Acceder al Código 2FA

**App de Autenticación:**
1. Verifica la hora correcta en el teléfono
2. Busca la entrada de Bayit+ en la app
3. Usa códigos de respaldo si están disponibles
4. Contacta soporte con verificación de identidad

**Código SMS:**
1. Verifica que el teléfono tenga señal
2. Espera hasta 5 minutos
3. Toca "Reenviar código"
4. Revisa todas las bandejas de SMS
5. Prueba opción de llamada de voz

### Perdí Acceso al Dispositivo 2FA

1. Usa uno de tus códigos de respaldo
2. Contacta soporte con:
   - Correo de la cuenta
   - Últimos 4 dígitos del método de pago
   - Historial de visualización reciente
   - Identificación oficial (si es requerida)

### Desactivar 2FA

Si tienes acceso:
1. Ve a **Configuración** > **Seguridad**
2. Toca **Autenticación de Dos Factores**
3. Ingresa el código 2FA actual
4. Confirma desactivación

## Problemas de Inicio de Sesión Social

### Inicio de Sesión con Google No Funciona

1. Asegúrate de que la cuenta de Google esté activa
2. Verifica que la app de Google esté actualizada
3. Prueba en el navegador web
4. Limpia caché de la app de Google
5. Vuelve a autenticar con Google

### Inicio de Sesión con Apple No Funciona

1. Verifica que Apple ID esté activo
2. Asegúrate de que Face ID/Touch ID funcione
3. Prueba con contraseña en su lugar
4. Verifica configuración de iCloud
5. Inicia sesión a través de web

### Conecté la Cuenta Incorrecta

1. Cierra sesión completamente
2. Limpia caché de la app
3. Inicia sesión con el método correcto
4. Contacta soporte si estás atascado

## Problemas de Dispositivo

### Sesión Expirada

1. Característica de seguridad normal
2. Simplemente inicia sesión de nuevo
3. Activa "Recordarme"
4. Verifica fecha/hora del dispositivo

### Demasiados Dispositivos

Has alcanzado tu límite de dispositivos:

1. Ve a **Configuración** > **Dispositivos**
2. Elimina dispositivos que no uses
3. O actualiza tu plan para más dispositivos

### Dispositivo No Reconocido

Después de reinstalar o resetear:
1. Inicia sesión normalmente
2. Verifica por correo si se solicita
3. Completa verificación de dispositivo
4. Esta es una característica de seguridad

## Problemas de Red

### El Inicio de Sesión Se Agota

1. Verifica conexión a internet
2. Prueba una red diferente
3. Desactiva VPN temporalmente
4. Prueba datos móviles vs Wi-Fi
5. Espera e intenta de nuevo

### "Algo Salió Mal"

1. Verifica la página de estado del servicio
2. Espera unos minutos
3. Reinicia la app
4. Busca actualizaciones de la app
5. Reinstala si persiste

## Seguridad de la Cuenta

### Actividad Sospechosa

Si notas actividad desconocida:

1. Cambia la contraseña inmediatamente
2. Activa autenticación de dos factores
3. Cierra sesión en todos los dispositivos
4. Revisa la actividad de la cuenta
5. Contacta soporte

### Cuenta Comprometida

1. Restablece la contraseña inmediatamente
2. Usa una contraseña única y fuerte
3. Revisa el correo para cambios no autorizados
4. Revisa apps conectadas
5. Activa 2FA
6. Contacta soporte

## Problemas de Perfil

### No Puedo Cambiar de Perfil

1. Asegúrate de que todos los perfiles estén configurados
2. Verifica PIN del perfil si está establecido
3. Cierra y vuelve a abrir la app
4. Cierra sesión y vuelve a iniciar

### Perfil Eliminado

1. Los perfiles no se pueden recuperar
2. Crea un nuevo perfil
3. Las preferencias no se restaurarán
4. El historial de visualización se perdió

## Problemas de Suscripción

### La Suscripción No Aparece

Después de un pago exitoso:
1. Cierra sesión y vuelve a iniciar
2. Espera 15 minutos para sincronizar
3. Revisa el correo de confirmación de pago
4. Verifica en suscripciones de la tienda de apps
5. Contacta soporte con el recibo

### Nivel de Suscripción Incorrecto

1. Verifica configuración de cuenta
2. Verifica monto del pago
3. Contacta soporte de facturación
4. Puede tomar tiempo para sincronizar

## Contactar Soporte

Si aún no puedes iniciar sesión:

1. **Usa Soporte por Voz** (desde la pantalla de inicio de sesión)
   - Di "Hey Bayit, no puedo iniciar sesión"

2. **Correo a Soporte**
   - Incluye: correo, tipo de dispositivo, mensajes de error
   - Respuesta esperada: 24 horas

3. **Crear Ticket**
   - Accede vía web: bayitplus.com/support
   - No se requiere inicio de sesión para problemas de inicio de sesión

**Información a Incluir:**
- Dirección de correo utilizada
- Tipo y modelo del dispositivo
- Versión de la app
- Captura de pantalla del error
- Pasos que has intentado
