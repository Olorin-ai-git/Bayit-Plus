# Search Functionality

Find exactly what you want to watch with powerful search capabilities. Bayit+ offers multiple ways to discover content across movies, shows, live TV, podcasts, and radio.

## Text Search

### Basic Search
1. Tap the search icon from any screen
2. Enter your search term using the keyboard
3. Results appear as you type
4. Select from suggestions or press search

### What You Can Search
- Movie and show titles
- Actor and director names
- Genre keywords
- Channel names
- Podcast titles and hosts
- Radio station names

## Voice Search

Speak your search query for hands-free discovery. Voice search is available on:
- iOS and Android mobile apps
- Apple TV using Siri Remote
- Android TV using voice remote
- Web browsers with microphone access

### Using Voice Search
1. Tap the microphone icon in the search bar
2. Speak your query clearly
3. Review the transcription
4. Results display automatically

Voice search supports natural language queries such as "action movies from the 1990s" or "cooking shows with celebrity chefs."

## Search Filters

Refine your results using filters:

### Content Type
- Movies
- TV Series
- Live Channels
- Podcasts
- Radio Stations

### Genre
Select from available genres to narrow results by category.

### Release Year
Filter movies and shows by release date or decade.

### Language
Find content in specific languages or with particular subtitle options.

### Rating
Filter by age rating for family-appropriate content.

## Search Results

Results are organized by relevance and grouped by content type. Each result displays:
- Thumbnail image
- Title
- Year and rating
- Brief description
- Quick action buttons (play, add to watchlist)

## Recent Searches

Your search history is saved for quick access to previous queries. Clear search history from the Settings menu for privacy.
