# Subtitles and Accessibility

Bayit+ provides comprehensive subtitle options and accessibility features to ensure everyone can enjoy our content.

## Available Subtitle Languages

Subtitles are available in multiple languages depending on the content. Common options include:
- English
- Hebrew
- Spanish
- Arabic
- Russian
- French

## Enabling Subtitles

### During Playback
1. Tap the screen to show playback controls
2. Select the subtitle icon (CC or speech bubble)
3. Choose your preferred language
4. Subtitles appear immediately

### Default Settings
Set a default subtitle language in your profile settings:
1. Go to Settings and select your profile
2. Choose "Subtitle Preferences"
3. Select your preferred language
4. Enable "Always show subtitles" if desired

## Subtitle Customization

Adjust subtitle appearance to improve readability:

### Text Size
Choose from small, medium, large, or extra-large text.

### Font Style
Select from available font options including sans-serif and serif styles.

### Text Color
Choose white, yellow, or other high-contrast colors.

### Background
Add a background behind text for better visibility:
- No background
- Semi-transparent
- Solid background

### Edge Style
Apply outline or drop shadow effects to text edges.

## Closed Captions

Some content offers closed captions (CC) that include non-dialogue audio descriptions such as sound effects and music cues. Look for the CC indicator when browsing.

## Audio Description

Select titles include audio description tracks that narrate visual elements for viewers who are blind or have low vision. Available tracks appear in the audio selection menu during playback.

## Hearing Assistance

Enable visual notifications for viewers who are deaf or hard of hearing. This feature provides on-screen alerts for important audio cues.

## Accessibility Settings

Access all accessibility options from Settings:
1. Go to Settings
2. Select "Accessibility"
3. Configure subtitle, caption, and audio description preferences
4. Changes apply to your profile across all devices
