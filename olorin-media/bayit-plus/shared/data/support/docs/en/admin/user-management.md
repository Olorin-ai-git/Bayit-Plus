# User Management

The User Management section provides tools for managing subscriber accounts, administrator roles, and access permissions across the Bayit+ platform.

## User Directory

Access the user directory from **Users** > **All Users**. The directory supports:

- **Search**: Find users by email, name, or account ID
- **Filters**: Segment by subscription status, creation date, or activity
- **Bulk Actions**: Select multiple users for batch operations
- **Export**: Download user lists for external analysis

## Viewing User Details

Click any user to view their complete profile:

- Account information and subscription status
- Profiles created under the account
- Viewing history and preferences
- Support ticket history
- Payment and billing information

## Account Actions

Available actions for individual accounts:

| Action | Description |
|--------|-------------|
| Reset Password | Send password reset email |
| Suspend Account | Temporarily disable access |
| Cancel Subscription | End paid subscription |
| Issue Credit | Apply account credit |
| Merge Accounts | Combine duplicate accounts |

## Administrator Roles

Assign granular permissions to admin team members:

- **Super Admin**: Full platform access
- **Content Manager**: Content library and uploads only
- **Support Agent**: User accounts and support tickets
- **Analyst**: Analytics and reports (read-only)
- **Marketing**: Campaign management only

## Creating Admin Accounts

To add a new administrator:

1. Navigate to **Users** > **Administrators**
2. Click **Add Administrator**
3. Enter email address and name
4. Select the appropriate role
5. The new admin receives an invitation email

## Security and Audit Logs

All administrative actions are logged for security:

- User account modifications
- Content changes
- Permission updates
- Login attempts and sessions

Access audit logs from **Settings** > **Security** > **Audit Log**.

## Bulk User Operations

Perform actions on multiple users simultaneously:

1. Use filters to segment the target users
2. Select users with checkboxes
3. Choose a bulk action from the dropdown
4. Confirm the action with your admin PIN
