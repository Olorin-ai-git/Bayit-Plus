# Notification Settings

Customize how and when Bayit+ communicates with you through email, push notifications, and in-app alerts.

## Managing Notifications

### Accessing Notification Settings

1. Go to **Settings** > **Account** > **Notifications**
2. Review all notification categories
3. Toggle options on or off
4. Save your preferences

## Email Notifications

### Available Email Notifications

| Category | Description | Default |
|----------|-------------|---------|
| **New Releases** | Content matching your interests | On |
| **Watchlist Updates** | When saved content becomes available | On |
| **Recommendations** | Personalized suggestions weekly | On |
| **Account Alerts** | Security and billing updates | Always On |
| **Newsletter** | News and feature announcements | Off |
| **Promotions** | Special offers and discounts | Off |

### Configuring Email Preferences

1. Go to **Settings** > **Notifications** > **Email**
2. Toggle each category as desired
3. Set frequency for applicable options:
   - Immediately
   - Daily digest
   - Weekly summary

### Email Frequency

Control how often you receive emails:
1. Go to **Settings** > **Notifications** > **Email Frequency**
2. Choose your preference:
   - **Real-time** - As events occur
   - **Daily** - One summary per day
   - **Weekly** - One summary per week

## Push Notifications

### Enabling Push Notifications

**iOS:**
1. Open device Settings
2. Find Bayit+ in app list
3. Tap Notifications
4. Enable Allow Notifications
5. Configure alert style preferences

**Android:**
1. Open device Settings
2. Go to Apps > Bayit+
3. Tap Notifications
4. Enable notifications
5. Configure channels as desired

### Push Notification Categories

| Category | Description |
|----------|-------------|
| **Now Playing** | Reminders for live events |
| **New Episodes** | When series you follow update |
| **Downloads Ready** | Download completion alerts |
| **Continue Watching** | Resume unfinished content |
| **Live Events** | Sports and special broadcasts |

### Managing Push Preferences

1. Go to **Settings** > **Notifications** > **Push**
2. Enable or disable each category
3. Set quiet hours if desired
4. Configure critical alerts separately

### Quiet Hours

Prevent notifications during specific times:
1. Go to **Settings** > **Notifications** > **Quiet Hours**
2. Enable Quiet Hours
3. Set start time and end time
4. Choose which notifications can bypass:
   - Security alerts (recommended)
   - Live event reminders

## In-App Notifications

### Notification Center

Access your notification history:
1. Tap the bell icon in the app
2. View recent notifications
3. Tap to take action
4. Swipe to dismiss

### In-App Alert Types

- **Content Updates** - New releases and additions
- **System Messages** - Maintenance and outages
- **Recommendations** - Suggested content
- **Social Updates** - Friend activity (if enabled)
- **Achievement Badges** - Viewing milestones

### Clearing Notifications

1. Open the Notification Center
2. Tap **Clear All** to remove all
3. Or swipe individual notifications to dismiss

## Profile-Specific Notifications

### Per-Profile Settings

Each profile can have unique notification preferences:
1. Switch to the desired profile
2. Go to **Settings** > **Notifications**
3. Adjust settings for that profile
4. Settings apply only to that profile

### Kids Profile Notifications

Kids profiles have limited notifications:
- No promotional content
- No social notifications
- Parents control all settings
- Only content reminders enabled

## Device-Specific Settings

### Managing by Device

Control notifications per device:
1. Go to **Settings** > **Notifications** > **Devices**
2. Select a device
3. Configure which notifications it receives
4. Useful for TV vs mobile preferences

### TV Notifications

Smart TV notifications are limited to:
- Live event starting soon
- Download ready to watch
- Account security alerts

## Unsubscribing

### Quick Unsubscribe

From any email:
1. Scroll to email footer
2. Click **Unsubscribe** link
3. Confirm your choice
4. Updates take effect within 24 hours

### Complete Opt-Out

To stop all non-essential communications:
1. Go to **Settings** > **Notifications**
2. Tap **Notification Preferences**
3. Select **Essential Only**
4. Confirm your choice

Essential notifications include:
- Security alerts
- Billing confirmations
- Required legal notices
