# Device Management

Manage the devices connected to your Bayit+ account, control remote access, and understand device limits.

## Viewing Connected Devices

### Accessing Device List

1. Go to **Settings** > **Account** > **Devices**
2. View all registered devices
3. See device details including:
   - Device name and type
   - Last active date
   - Location (approximate)
   - App version installed

### Device Information

Each device entry shows:
- **Device Name** - Model or custom name
- **Type** - Phone, tablet, TV, or web browser
- **Status** - Active, idle, or offline
- **Last Used** - Date and time of last activity
- **Downloads** - Number of downloaded titles

## Device Limits by Plan

| Plan | Registered Devices | Simultaneous Streams |
|------|-------------------|---------------------|
| **Basic** | 2 devices | 1 stream |
| **Standard** | 4 devices | 2 streams |
| **Premium** | 6 devices | 4 streams |
| **Family** | 10 devices | 6 streams |

### Registered vs Active Devices

- **Registered** - Devices that have signed into your account
- **Active** - Devices currently streaming content
- Limits apply to simultaneous active streams

## Managing Devices

### Renaming a Device

1. Go to **Settings** > **Devices**
2. Find the device to rename
3. Tap the device entry
4. Select **Rename Device**
5. Enter a descriptive name
6. Save changes

### Removing a Device

1. Go to **Settings** > **Devices**
2. Find the device to remove
3. Tap the device entry
4. Select **Remove Device**
5. Confirm removal

Removing a device:
- Signs out that device immediately
- Deletes downloaded content on that device
- Frees up a device slot
- Requires re-authentication to add back

## Signing Out Remotely

### Sign Out Single Device

1. Go to **Settings** > **Devices**
2. Select the device
3. Tap **Sign Out This Device**
4. Confirm your choice

### Sign Out All Devices

1. Go to **Settings** > **Account** > **Security**
2. Tap **Sign Out All Devices**
3. Enter your password
4. Confirm sign out

Use this feature if:
- You suspect unauthorized access
- You lost a device
- You want to reset all sessions

## Trusted Devices

### Marking as Trusted

Trusted devices skip some security checks:
1. Go to **Settings** > **Devices**
2. Select a device
3. Toggle **Trusted Device** on
4. Verify with password or 2FA

### Trusted Device Benefits

- Skip 2FA on subsequent logins
- Faster sign-in experience
- Still protected by other security measures

### Removing Trust

1. Go to **Settings** > **Devices**
2. Select the trusted device
3. Toggle **Trusted Device** off
4. Device will require 2FA again

## Download Management

### Viewing Downloads by Device

1. Go to **Settings** > **Devices**
2. Select a device
3. View **Downloads** section
4. See titles downloaded to that device

### Deleting Remote Downloads

1. Go to **Settings** > **Devices**
2. Select the device
3. Tap **Manage Downloads**
4. Select titles to delete
5. Confirm deletion

Downloads will be removed when the device connects to the internet.

## Device Troubleshooting

### Device Not Appearing

If a device is missing from your list:
1. Open Bayit+ on that device
2. Sign out and sign back in
3. Check internet connection
4. Update the app to latest version

### Reaching Device Limit

If you cannot add more devices:
1. Remove devices you no longer use
2. Upgrade your subscription plan
3. Wait 24 hours after removing a device

### Device Shows Wrong Location

Location is approximate and based on IP address:
- VPN usage affects displayed location
- Mobile networks may show carrier location
- This is normal and not a security concern
