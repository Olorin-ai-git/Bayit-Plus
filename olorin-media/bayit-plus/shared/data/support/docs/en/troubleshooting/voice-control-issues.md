# Troubleshooting Voice Control Issues

Having trouble with voice commands? This guide helps resolve microphone permissions, voice recognition, and wake word problems.

## Microphone Permissions

### Checking Microphone Access

Voice control requires microphone permission to function.

**iOS:**
1. Open device Settings
2. Scroll down to Bayit+
3. Enable **Microphone** toggle
4. Return to Bayit+ and try again

**Android:**
1. Open device Settings
2. Go to Apps > Bayit+
3. Tap Permissions
4. Enable **Microphone**
5. Return to Bayit+ and try again

**Web Browser:**
1. Click the lock icon in the address bar
2. Find Microphone permission
3. Set to **Allow**
4. Refresh the page

### Permission Denied Error

If you see "Microphone access denied":
1. Follow the steps above for your device
2. Restart the Bayit+ app
3. Try voice control again
4. Grant permission when prompted

### Microphone In Use by Another App

If another app is using the microphone:
1. Close other voice-enabled apps
2. End any active calls
3. Stop any recordings in progress
4. Return to Bayit+ and try again

## Voice Recognition Problems

### Voice Not Recognized

If the app does not understand your commands:

1. **Speak clearly** - Use a normal speaking pace
2. **Reduce background noise** - Move to a quieter area
3. **Get closer** - Speak toward the microphone
4. **Check microphone** - Ensure it is not blocked

### Improving Recognition Accuracy

1. Go to **Settings** > **Voice Control** > **Language**
2. Select your preferred language
3. Enable **Voice Training** if available
4. Complete the training session

### Supported Languages

Voice control supports:
- English (US, UK, Australian)
- Hebrew
- Spanish (Latin American, European)

Select your language in Settings > Voice Control.

### Common Voice Commands

| Command | Action |
|---------|--------|
| "Play [title]" | Starts playing content |
| "Pause" | Pauses current playback |
| "Resume" | Resumes playback |
| "Go back 30 seconds" | Rewinds playback |
| "Skip forward" | Jumps ahead |
| "Turn on subtitles" | Enables subtitles |
| "Search for [query]" | Searches content |
| "Open settings" | Goes to settings |

## Wake Word Issues

### Wake Word Not Detected

The wake word "Hey Bayit" activates voice control.

If the wake word is not detected:
1. Go to **Settings** > **Voice Control**
2. Verify **Wake Word Detection** is enabled
3. Check microphone permission is granted
4. Ensure device is not in power-saving mode

### Adjusting Wake Word Sensitivity

1. Go to **Settings** > **Voice Control** > **Wake Word**
2. Adjust **Sensitivity** level:
   - **Low** - Fewer false activations
   - **Medium** - Balanced (recommended)
   - **High** - More responsive

### False Activations

If voice control activates unintentionally:
1. Lower wake word sensitivity
2. Disable wake word and use button activation
3. Check for similar-sounding words nearby

### Changing Activation Method

If you prefer not to use the wake word:
1. Go to **Settings** > **Voice Control**
2. Disable **Wake Word Detection**
3. Use the microphone button instead
4. Press and hold to speak commands

## Device-Specific Issues

### Smart TV Voice Control

**Apple TV:**
1. Hold the Siri button on the remote
2. Speak your command
3. Wake word not supported on Apple TV

**Android TV:**
1. Press the microphone button on remote
2. Or say "Hey Google, open Bayit+"
3. Use Google Assistant integration

**Amazon Fire TV:**
1. Hold the Alexa button on remote
2. Say "Alexa, open Bayit+"
3. Use Alexa for voice commands

### Mobile Voice Control

1. Tap the microphone icon in the app
2. Or use the wake word when app is open
3. Background listening requires app to be active

## Troubleshooting Steps

### Voice Control Not Working at All

1. Restart the Bayit+ app
2. Check microphone permissions
3. Verify internet connection
4. Update the app to latest version
5. Restart your device

### Voice Control Previously Worked

1. Check for recent permission changes
2. Verify no new apps are blocking microphone
3. Check for system updates
4. Clear app cache and try again

### Testing Your Microphone

1. Go to **Settings** > **Voice Control**
2. Tap **Test Microphone**
3. Speak when prompted
4. Verify your voice is detected
5. Adjust microphone sensitivity if needed

## Privacy and Voice Data

### Voice Recording Policy

- Voice data processed in real-time
- Commands not stored permanently
- No third-party sharing of voice data
- Disable voice history in privacy settings

### Disabling Voice Features

1. Go to **Settings** > **Voice Control**
2. Toggle **Voice Control** off
3. Voice features are completely disabled
4. Use manual controls only

## Contact Support

For persistent voice control issues:
1. **In-App:** Settings > Help > Voice Control Issue
2. Include: device type, command attempted, and error message
