# Keyboard Shortcuts

Master the Bayit+ web player with these keyboard shortcuts for efficient navigation and playback control.

## Playback Controls

| Shortcut | Action |
|----------|--------|
| `Space` or `K` | Play / Pause |
| `M` | Mute / Unmute |
| `F` | Toggle Fullscreen |
| `Esc` | Exit Fullscreen |

## Seeking and Navigation

| Shortcut | Action |
|----------|--------|
| `Left Arrow` | Rewind 10 seconds |
| `Right Arrow` | Forward 10 seconds |
| `J` | Rewind 10 seconds |
| `L` | Forward 10 seconds |
| `Home` | Go to beginning |
| `End` | Go to end |
| `0-9` | Jump to 0%-90% of video |

## Volume Controls

| Shortcut | Action |
|----------|--------|
| `Up Arrow` | Increase volume by 5% |
| `Down Arrow` | Decrease volume by 5% |
| `M` | Toggle mute |

## Captions and Settings

| Shortcut | Action |
|----------|--------|
| `C` | Toggle captions on/off |
| `S` | Open settings menu |

## Playback Speed

| Shortcut | Action |
|----------|--------|
| `Shift + >` | Increase playback speed |
| `Shift + <` | Decrease playback speed |

## Tips for Better Experience

- Shortcuts work when the video player is focused
- Click on the player area first if shortcuts are not responding
- Use `Tab` to navigate between player controls
- Press `?` to view the shortcuts overlay while watching

## Accessibility

All keyboard shortcuts are designed to work with screen readers. The player supports standard accessibility features including focus indicators and ARIA labels for all controls.
