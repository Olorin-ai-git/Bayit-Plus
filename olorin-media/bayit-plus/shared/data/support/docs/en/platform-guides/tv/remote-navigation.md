# Remote Navigation Guide

Learn how to navigate Bayit+ on your TV using your remote control for a smooth viewing experience.

## D-Pad Navigation

The directional pad (D-pad) is your primary navigation tool on TV.

### Basic Movement
- **Up/Down:** Move between rows and menu items
- **Left/Right:** Browse through content carousels
- **OK/Select:** Choose highlighted item or confirm action
- **Back:** Return to previous screen or close menu

### In Content Browser
1. Use Up/Down to move between content rows
2. Use Left/Right to scroll through items in a row
3. Press OK to view content details
4. Press OK again to start playback

## Playback Controls

### During Video Playback
| Button | Action |
|--------|--------|
| OK | Show/hide playback controls |
| Play/Pause | Toggle playback |
| Left | Rewind 10 seconds |
| Right | Forward 10 seconds |
| Up | Show video information |
| Down | Access audio and subtitle settings |
| Back | Exit to browse screen |

### Seeking
1. Press OK to show the progress bar
2. Use Left/Right to scrub through the timeline
3. Press OK to resume from selected position

## Voice Control

### Apple TV (Siri)
Hold the Siri button and speak your command:
- "Fast forward 5 minutes"
- "What did they say?" (rewinds 15 seconds with subtitles)
- "Play something for kids"

### Android TV (Google Assistant)
Press the microphone button and speak:
- "Skip intro"
- "Show me documentaries"
- "Turn on Hebrew subtitles"

## Menu Navigation

### Main Menu
- Use D-pad to highlight menu categories
- Press OK to enter a category
- Press Back to return to the main menu

### Settings Menu
1. Navigate to your profile icon
2. Press Down to access Settings
3. Use D-pad to navigate options
4. Press OK to toggle or select items

## Tips for Easy Navigation

- Hold Left/Right for continuous scrolling through long rows
- Use number buttons (if available) to jump to specific positions during playback
- Long-press Back to return directly to the home screen on some devices
- Enable "Reduce Motion" in settings if animations cause navigation difficulty

## Accessibility

- Screen reader support is available for visually impaired users
- High contrast mode can be enabled in display settings
- Audio descriptions are available for select content
