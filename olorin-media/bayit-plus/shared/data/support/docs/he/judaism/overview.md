# סעיף היהדות

סעיף היהדות ב-Bayit+ מספק מרכז מקיף לתוכן יהודי, משאבי לימוד וחיבור קהילתי.

## מה תמצאו

סעיף היהדות כולל:

- סרטוני לימוד תורה ושיעורי אודיו
- תוכן דף יומי ופרשת שבוע יומי
- לוח עברי עם מידע על חגים
- זמנים (זמנים הלכתיים) למיקום שלכם
- מדריך קהילתי למשאבים מקומיים
- מצב שבת לתזמון אוטומטי

## גישה לסעיף היהדות

לגישה לתוכן יהודי:

1. פתחו את התפריט הראשי
2. בחרו יהדות
3. עיינו בקטגוריות הזמינות

הסעיף מתאים את עצמו להעדפות ולמיקום שלכם להצגת תוכן רלוונטי.

## תוכן מותאם אישית

Bayit+ לומדת את תחומי העניין שלכם לאורך זמן:

- שיעורים מרבנים שאתם עוקבים אחריהם מופיעים ראשונים
- תוכן המתאים לרמת הלימוד שלכם מודגש
- תוכן ספציפי לחגים מופיע לפני כל חג
- זמנים ומשאבים קהילתיים מבוססי מיקום

## תכונות יומיות

התחילו כל יום עם תוכן יהודי:

- טקס בוקר עם מזג אוויר, חדשות והשראה
- תזכורות תהילים ולימוד יומי
- התראות יארצייט לבני משפחה
- טיימרים לספירה לאחור לשבת וחגים

## מסלולי לימוד

עקבו אחר תוכניות לימוד מובנות:

- דף יומי עם מגידי שיעור מרובים
- פרשת שבוע עם פרשנות
- משנה יומית והלכה יומית
- קורסי הכנה לחגים

## חיבור קהילתי

התחברו לקהילה היהודית דרך:

- מאתרי בתי כנסת ומניינים
- מדריך מסעדות כשרות
- רשימות ארגונים יהודיים
- לוחות אירועים לאזור שלכם
