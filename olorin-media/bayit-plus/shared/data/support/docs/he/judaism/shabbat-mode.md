# מצב שבת

מצב שבת ב-Bayit+ מתאים אוטומטית את חוויית הסטרימינג שלכם בהתאם לזמנים ההלכתיים למיקומכם.

## איך מצב שבת עובד

כשמופעל, מצב שבת:

- משהה את כל הסטרימינג בזמן הדלקת נרות
- משבית התראות במהלך השבת
- מחדש פונקציונליות לאחר זמן הבדלה
- חל על כל המכשירים בחשבון

המצב מופעל ומושבת אוטומטית ללא צורך בפעולה מצדכם.

## הפעלת מצב שבת

להפעלת מצב שבת אוטומטי:

1. עברו להגדרות
2. בחרו העדפות יהדות
3. הפעילו מצב שבת אוטומטי
4. אשרו את המיקום שלכם לזמנים מדויקים

הרשאות מיקום מאפשרות חישוב זמנים מדויק.

## הגדרות מיקום

מצב שבת משתמש במיקום שלכם לקביעת זמנים:

- הגדירו את המיקום הראשי שלכם ידנית אם מעדיפים
- מצב נסיעות מתאים כשאתם מבקרים באזורים אחרים
- בחרו את המנהג שלכם לחישובי זמנים
- בחרו דקות הדלקת נרות מוקדמת אם רצוי

## מצב חג

מצב שבת מופעל גם ליום טוב:

- חל על כל חגים דו-יומיים מחוץ לישראל
- יישום יום אחד בישראל
- מכסה יום כיפור וצומות אחרים
- סטרימינג חול המועד נשאר זמין

## תזכורות לפני שבת

קבלו התראות מועילות לפני שבת:

- תזכורת שעה לפני הדלקת נרות
- תזכורת בזמן פלג המנחה
- תזכורת אחרונה 18 דקות לפני השקיעה
- זמני תזכורת מותאמים אישית זמינים בהגדרות

## תור תוכן

הכינו תור תוכן לצפייה לאחר שבת:

1. הוסיפו תוכניות או סרטים לתור השבת שלכם
2. התוכן מוכן כששבת נגמרת
3. המשיכו לצפות מיד לאחר הבדלה

תכונה זו עוזרת לכם לתכנן את הצפייה למוצאי שבת.

## עקיפה ידנית

במידת הצורך, תוכלו להשבית זמנית את מצב שבת למצבים ספציפיים דרך תפריט ההגדרות.
