# בקרת הורים

Bayit+ מספקת בקרת הורים מקיפה כדי לעזור לכם לנהל לאיזה תוכן המשפחה שלכם יכולה לגשת. כלים אלה נותנים לכם שקט נפשי בזמן שילדיכם נהנים מבידור מותאם גיל.

## הגדרת קוד PIN

קוד PIN בן 4 ספרות מגן על הגדרות בקרת ההורים שלכם משינויים לא מורשים.

1. נווטו ל**הגדרות** > **בקרת הורים**
2. בחרו **צור PIN**
3. הזינו קוד PIN בן 4 ספרות שילדיכם לא ינחשו בקלות
4. אשרו את ה-PIN על ידי הזנתו שוב

קוד ה-PIN שלכם יידרש לגישה לתוכן מוגבל, לשינוי הגדרות או ליציאה מפרופילי ילדים.

## הגבלות תוכן

ניתן להגביל תוכן על בסיס דירוגי גיל לכל פרופיל:

| רמת הגבלה | דירוגים מותרים |
|-----------|----------------|
| ילדים קטנים | G בלבד |
| ילדים גדולים | G, PG |
| נוער | G, PG, PG-13 |
| מבוגרים | כל הדירוגים |

להגדרת הגבלות תוכן:

1. עברו ל**הגדרות** > **בקרת הורים** > **הגבלות תוכן**
2. בחרו את הפרופיל שברצונכם לשנות
3. בחרו את רמת ההגבלה המתאימה
4. הזינו את קוד ה-PIN שלכם לאישור השינויים

## חסימת תוכן ספציפי

מעבר לדירוגי גיל, ניתן לחסום כותרים בודדים:

1. נווטו לתוכן שברצונכם לחסום
2. בחרו בתפריט **אפשרויות נוספות**
3. בחרו **חסום כותר זה**
4. הזינו את קוד ה-PIN שלכם לאישור

תוכן חסום לא יופיע בתוצאות חיפוש או בהמלצות עבור פרופילים מוגבלים.

## איפוס קוד PIN

אם שכחתם את קוד ה-PIN, ניתן לאפס אותו דרך כתובת הדוא"ל של החשבון:

1. עברו ל**הגדרות** > **בקרת הורים**
2. בחרו **שכחתי PIN**
3. בדקו את הדוא"ל שלכם לקבלת קוד אימות
4. הזינו את הקוד וצרו PIN חדש
