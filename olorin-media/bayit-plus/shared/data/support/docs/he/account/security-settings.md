# הגדרות אבטחה

הגנו על חשבון Bayit+ שלכם עם ניהול סיסמאות, אימות דו-שלבי וניטור כניסות.

## ניהול סיסמאות

### שינוי הסיסמה שלכם

1. גשו ל**הגדרות** > **חשבון** > **אבטחה**
2. לחצו **שנה סיסמה**
3. הזינו את הסיסמה הנוכחית שלכם
4. הזינו את הסיסמה החדשה פעמיים
5. לחצו **שמור שינויים**

### דרישות סיסמה

הסיסמה שלכם חייבת לכלול:
- לפחות 8 תווים
- אות גדולה אחת (A-Z)
- אות קטנה אחת (a-z)
- מספר אחד (0-9)
- תו מיוחד אחד (!@#$%^&*)

### שיטות עבודה מומלצות לסיסמאות

- השתמשו בסיסמה ייחודית שלא משמשת במקום אחר
- שקלו להשתמש במנהל סיסמאות
- אל תשתפו את הסיסמה שלכם עם אף אחד
- שנו את הסיסמה כל 6 חודשים

## אימות דו-שלבי

### הפעלת אימות דו-שלבי

1. גשו ל**הגדרות** > **חשבון** > **אבטחה**
2. לחצו **אימות דו-שלבי**
3. בחרו את השיטה שלכם:
   - **אפליקציית אימות** (מומלץ)
   - **קוד SMS**
4. עקבו אחר הוראות ההגדרה
5. שמרו את קודי הגיבוי שלכם במקום בטוח

### שימוש באפליקציית אימות

1. הורידו Google Authenticator או Authy
2. סרקו את קוד ה-QR המוצג ב-Bayit+
3. הזינו את הקוד בן 6 הספרות לאימות
4. אחסנו קודי גיבוי במקום בטוח

### קודי גיבוי

כאשר אתם מפעילים אימות דו-שלבי, אתם מקבלים 10 קודי גיבוי:
- כל קוד ניתן לשימוש פעם אחת בלבד
- אחסנו אותם בבטחה במצב לא מקוון
- השתמשו בהם אם איבדתם גישה למכשיר האימות
- צרו קודים חדשים בהגדרות אבטחה אם צריך

## היסטוריית כניסות

### צפייה בכניסות אחרונות

1. גשו ל**הגדרות** > **חשבון** > **אבטחה**
2. לחצו **היסטוריית כניסות**
3. צפו בכל הכניסות האחרונות עם:
   - תאריך ושעה
   - סוג מכשיר
   - מיקום (משוער)
   - כתובת IP

### זיהוי פעילות חשודה

חפשו כניסות ש:
- התרחשו ממיקומים לא מוכרים
- השתמשו במכשירים לא מוכרים
- קרו בזמנים לא רגילים

### תגובה לגישה לא מורשית

1. שנו את הסיסמה שלכם מיידית
2. הפעילו אימות דו-שלבי
3. התנתקו מכל המכשירים (ראו למטה)
4. סקרו אפליקציות מחוברות
5. צרו קשר עם התמיכה אם אתם מודאגים

## התנתקות מכל המכשירים

### איך להתנתק מכל מקום

1. גשו ל**הגדרות** > **חשבון** > **אבטחה**
2. לחצו **התנתק מכל המכשירים**
3. אשרו את הסיסמה שלכם
4. כל ההפעלות מסתיימות

פעולה זו:
- מתנתקת מכל מכשיר חוץ מהנוכחי שלכם
- דורשת אימות מחדש בכל המכשירים
- לא משפיעה על ההורדות או ההעדפות שלכם

## התראות אבטחה

### ניהול התראות

1. גשו ל**הגדרות** > **חשבון** > **אבטחה**
2. לחצו **התראות אבטחה**
3. בחרו אילו התראות לקבל:
   - כניסות ממכשיר חדש
   - שינויי סיסמה
   - שינויי אימות דו-שלבי
   - שינויי מנוי

### משלוח התראות

התראות אבטחה נשלחות דרך:
- אימייל (תמיד מופעל)
- התראת פוש (אופציונלי)
- התראה באפליקציה (אופציונלי)

## שחזור חשבון

### הגדרת אפשרויות שחזור

1. גשו ל**הגדרות** > **חשבון** > **אבטחה**
2. לחצו **אפשרויות שחזור**
3. הוסיפו כתובת אימייל לשחזור
4. הוסיפו מספר טלפון לשחזור
5. אמתו כל שיטה

### אם איבדתם גישה

1. השתמשו באיפוס סיסמה דרך אימייל
2. השתמשו בקודי גיבוי לאימות דו-שלבי
3. צרו קשר עם התמיכה עם אימות זהות
