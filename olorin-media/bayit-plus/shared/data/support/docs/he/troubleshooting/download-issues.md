# פתרון בעיות הורדה

נתקלים בבעיות בהורדת תוכן לצפייה לא מקוונת? מדריך זה עוזר לפתור כשלי הורדה, בעיות אחסון והורדות שפג תוקפן.

## כשלי הורדה

### הורדה לא מתחילה

1. **בדקו חיבור אינטרנט** - הורדות דורשות חיבור יציב
2. **ודאו מנוי** - התוכנית שלכם חייבת לתמוך בהורדות
3. **בדקו זכויות תוכן** - לא כל הכותרים ניתנים להורדה
4. **הפעילו מחדש את האפליקציה** - סגרו לחלוטין ופתחו מחדש

### בדיקת זמינות הורדה

1. עברו לדף פרטי התוכן
2. חפשו את כפתור **הורדה**
3. אם אפור, התוכן לא ניתן להורדה
4. לחלק מהכותרים יש הגבלות אזוריות

### דרישות מנוי

| תוכנית | הורדות מותרות |
|--------|---------------|
| בסיסי | ללא הורדות |
| סטנדרט | ללא הורדות |
| פרימיום | הורדות ללא הגבלה |
| משפחה | הורדות ללא הגבלה |

שדרגו את התוכנית אם הורדות לא זמינות.

### הורדה נקטעה

אם הורדה נעצרת באמצע:
1. בדקו יציבות חיבור Wi-Fi
2. לחצו על ההורדה להמשך
3. אם תקועה, בטלו והתחילו מחדש
4. עברו לאזור עם אות חזק יותר

## בעיות אחסון

### שגיאת אחסון לא מספיק

אם אתם רואים "אין מספיק אחסון":

**iOS:**
1. עברו להגדרות > כללי > אחסון iPhone
2. בחנו שימוש באחסון
3. מחקו אפליקציות או תוכן לא בשימוש
4. נקו מטמון Bayit+ אם נדרש

**Android:**
1. עברו להגדרות > אחסון
2. בחנו פירוט אחסון
3. נקו נתונים מאוחסנים
4. הסירו קבצים או אפליקציות לא בשימוש

### בדיקת אחסון הורדות

1. עברו ל**הגדרות** > **הורדות**
2. צפו ב**אחסון בשימוש** על ידי Bayit+
3. ראו פירוט לפי סוג תוכן
4. מחקו תוכן לפינוי מקום

### ניהול תוכן שהורד

1. עברו ל**הורדות** באפליקציה
2. בחנו את כל הכותרים שהורדו
3. החליקו או לחצו למחיקת כותרים בודדים
4. השתמשו ב**מחק הכל** לניקוי מלא

### איכות הורדה וגודל

איכות גבוהה יותר משמעותה קובץ גדול יותר:

| איכות | גודל משוער (לשעה) |
|-------|-------------------|
| נמוכה | 300 MB |
| בינונית | 700 MB |
| גבוהה | 1.5 GB |
| Ultra | 3 GB |

### שינוי איכות הורדה

1. עברו ל**הגדרות** > **הורדות**
2. בחרו **איכות הורדה**
3. בחרו רמת איכות מועדפת
4. איכות נמוכה חוסכת מקום אחסון

### מיקום הורדה (Android)

העבירו הורדות לכרטיס SD:
1. עברו ל**הגדרות** > **הורדות**
2. לחצו על **מיקום הורדה**
3. בחרו **כרטיס SD** אם זמין
4. הורדות חדשות נשמרות בכרטיס SD

## הורדות שפג תוקפן

### הבנת תפוגת הורדות

תוכן שהורד פג תוקפו אחרי:
- **30 יום** מתאריך ההורדה
- **48 שעות** אחרי הפעלה ראשונה
- או כאשר המנוי מסתיים

### בדיקת סטטוס תפוגה

1. עברו ל**הורדות**
2. צפו בתאריך תפוגה מתחת לכל כותר
3. כותרים מציגים "פג תוקף בעוד X ימים"
4. תוכן שפג תוקפו מוסר אוטומטית

### חידוש הורדות שפג תוקפן

לצפייה בתוכן שפג תוקפו שוב:
1. ודאו חיבור אינטרנט
2. מחקו את ההורדה שפג תוקפה
3. הורידו מחדש את התוכן
4. תקופת תפוגה חדשה מתחילה

### הורדות אחרי סיום מנוי

אם המנוי שלכם פג:
- כל ההורדות הופכות לבלתי ניתנות להפעלה
- התוכן נשאר במכשיר זמנית
- הפעילו מחדש את המנוי לשחזור גישה
- או שהורדות יימחקו אוטומטית

## מגבלות הורדה

### מקסימום הורדות

מגבלות תלויות בתוכנית:
- **פרימיום:** 25 כותרים למכשיר
- **משפחה:** 25 כותרים למכשיר, לכל פרופיל

### מגבלות מכשיר להורדות

הורדות קשורות למכשירים ספציפיים:
- לא ניתן להעביר הורדות בין מכשירים
- לכל מכשיר יש מגבלה משלו
- הסרת מכשיר מוחקת את ההורדות שלו

### הגעה למגבלת הורדות

אם לא ניתן להוריד עוד:
1. עברו ל**הורדות**
2. מחקו כותרים שצפיתם בהם
3. מחקו כותרים שאינכם צריכים עוד
4. נסו להוריד שוב

## הורדה לא מתנגנת

### תוכן שהורד לא מתנגן

1. **בדקו תפוגה** - התוכן עשוי להיות פג תוקף
2. **ודאו חשבון** - ודאו שאתם מחוברים
3. **בדקו תאריך/שעה** - תאריך המכשיר חייב להיות מדויק
4. **הפעילו מחדש אפליקציה** - סגרו ופתחו את Bayit+

### שגיאת הפעלה בתוכן שהורד

1. מחקו את ההורדה הבעייתית
2. הורידו מחדש את התוכן
3. ודאו אחסון מספיק
4. נסו הגדרת איכות אחרת

### תאריך/שעה שגויים במכשיר

הורדות מאמתות מול שעת המכשיר:
1. עברו להגדרות מכשיר > תאריך ושעה
2. הפעילו **הגדר אוטומטית**
3. הפעילו מחדש את אפליקציית Bayit+
4. נסו להפעיל תוכן שוב

## הורדות Wi-Fi בלבד

### הפעלת מצב Wi-Fi בלבד

למניעת שימוש בנתונים סלולריים:
1. עברו ל**הגדרות** > **הורדות**
2. הפעילו **Wi-Fi בלבד**
3. הורדות מתקדמות רק ב-Wi-Fi
4. חוסך מכסת נתונים סלולריים

### הורדה בנתונים סלולריים

אם צריכים להוריד בסלולרי:
1. עברו ל**הגדרות** > **הורדות**
2. השביתו **Wi-Fi בלבד**
3. היו מודעים לשימוש בנתונים
4. הורידו איכות להפחתת צריכת נתונים

## פניה לתמיכה

לבעיות הורדה מתמשכות:
1. **באפליקציה:** הגדרות > עזרה > בעיית הורדה
2. **קולית:** "היי בית, אני לא יכול להוריד"
3. כללו: סוג מכשיר, שם תוכן והודעת שגיאה
