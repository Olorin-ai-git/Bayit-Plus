# דרישות רשת

ודאו שחיבור האינטרנט שלכם עומד בדרישות לביצועי סטרימינג אופטימליים של Bayit+.

## דרישות מהירות אינטרנט

### מהירויות מינימום לפי איכות

| איכות וידאו | מהירות הורדה נדרשת |
|-------------|-------------------|
| SD (480p) | 3 Mbps |
| HD (720p) | 5 Mbps |
| Full HD (1080p) | 10 Mbps |
| 4K Ultra HD | 25 Mbps |
| 4K עם HDR | 40 Mbps |

### בדיקת המהירות שלכם

1. היכנסו ל-speedtest.net או fast.com
2. הריצו בדיקת מהירות
3. רשמו את מהירות ההורדה
4. השוו מול הדרישות
5. בדקו בזמנים שונים ביום

### המלצות מהירות

לחוויה הטובה ביותר, אנו ממליצים:
- **צופה יחיד:** 1.5 כפול מהמהירות המינימלית
- **צופים מרובים:** חברו את הדרישות יחד
- **דוגמה:** שני זרמי 4K צריכים 50+ Mbps

## אופטימיזציית Wi-Fi

### מיקום הנתב

1. מקמו את הנתב במיקום מרכזי
2. הרימו את הנתב מהרצפה
3. הרחיקו מקירות ועצמים מתכתיים
4. הימנעו ממיקום ליד מיקרוגל או טלפונים אלחוטיים

### תדרי Wi-Fi

| תדר | מתאים ל |
|-----|---------|
| 2.4 GHz | טווח ארוך יותר, יותר הפרעות |
| 5 GHz | מהירויות גבוהות יותר, טווח קצר יותר |

**המלצה:** השתמשו ב-5 GHz לסטרימינג כשאפשר.

### הפחתת הפרעות

1. שנו ערוץ Wi-Fi בהגדרות הנתב
2. עדכנו קושחת נתב
3. הסירו מכשירים מחוברים לא בשימוש
4. שקלו Wi-Fi mesh לבתים גדולים

### חיבור קווי

לסטרימינג הכי יציב בטלוויזיה:
1. חברו כבל Ethernet למכשיר הסטרימינג
2. חברו את הקצה השני לנתב
3. חיבורים קוויים מבטלים בעיות Wi-Fi
4. מומלץ לסטרימינג 4K

## הגדרות חומת אש

### פורטים נדרשים

Bayit+ דורש שפורטים אלה יהיו פתוחים:

| פורט | פרוטוקול | מטרה |
|------|----------|------|
| 80 | TCP | תעבורת HTTP |
| 443 | TCP | תעבורת HTTPS |
| 53 | UDP | פענוח DNS |

### הגדרת חומת האש

1. גשו ללוח הניהול של הנתב
2. מצאו הגדרות חומת אש או אבטחה
3. ודאו שהפורטים למעלה לא חסומים
4. הוסיפו את Bayit+ לאפליקציות מותרות

### רשתות ארגוניות

לרשתות ארגוניות או בתי ספר:
1. פנו למנהל ה-IT
2. בקשו גישה לשירותי סטרימינג
3. ספקו דומיין נדרש: *.bayitplus.com
4. עשוי לדרוש הגדרת פרוקסי

## בעיות VPN

### תאימות VPN

שימוש ב-VPN עשוי לגרום לבעיות:
- מהירות סטרימינג מופחתת
- התנגשויות הגבלה גיאוגרפית
- ניתוקי חיבור במהלך הפעלה
- חלק מה-VPNים חסומים

### הגדרות VPN מומלצות

אם חייבים להשתמש ב-VPN:
1. התחברו לשרת בארץ המגורים
2. השתמשו בשרת מותאם לסטרימינג
3. הפעילו split tunneling ל-Bayit+
4. השתמשו בפרוטוקול UDP למהירות טובה יותר

### השבתת VPN לסטרימינג

לחוויה הטובה ביותר:
1. השביתו VPN בעת צפייה ב-Bayit+
2. או החריגו את Bayit+ ממנהרת ה-VPN
3. בדקו אצל ספק ה-VPN ל-split tunneling

### שגיאת VPN חסום

אם אתם רואים שגיאה הקשורה ל-VPN:
1. התנתקו מה-VPN
2. נקו מטמון אפליקציה
3. הפעילו מחדש את האפליקציה
4. נסו שוב ללא VPN

## סטרימינג בנתונים סלולריים

### הערכות שימוש בנתונים

| איכות | נתונים לשעה |
|-------|-------------|
| נמוכה | 0.3 GB |
| בינונית | 0.7 GB |
| גבוהה | 1.5 GB |
| Ultra | 3 GB |

### הפחתת שימוש בנתונים סלולריים

1. עברו ל**הגדרות** > **הפעלה**
2. הגדירו **איכות נתונים סלולריים** לנמוכה או בינונית
3. הפעילו **Wi-Fi בלבד** להורדות
4. השתמשו בתוכן שהורד כשאפשר

### דרישות רשת סלולרית

- 4G LTE מומלץ לסטרימינג HD
- 5G מספק את החוויה הניידת הטובה ביותר
- 3G עשוי לתמוך רק באיכות SD
- עוצמת אות משפיעה על הביצועים

## פתרון בעיות רשת

### ניתוקי חיבור במהלך הפעלה

1. בדקו נקודות מתות של Wi-Fi
2. התקרבו לנתב
3. הפעילו מחדש נתב ומכשיר
4. הורידו הגדרת איכות וידאו
5. נסו חיבור קווי

### בעיות באפרינג

1. הריצו בדיקת מהירות
2. סגרו אפליקציות סטרימינג אחרות
3. נתקו מכשירים לא בשימוש
4. הורידו איכות בהגדרות > הפעלה
5. הפעילו מחדש ציוד רשת

### אבחון רשת

1. עברו ל**הגדרות** > **עזרה** > **בדיקת רשת**
2. הריצו את האבחון המובנה
3. בחנו תוצאות והמלצות
4. עקבו אחר התיקונים המוצעים

### בעיות DNS

אם תוכן לא נטען:
1. נסו לשנות שרתי DNS
2. עברו להגדרות הנתב
3. הגדירו DNS ל-Google (8.8.8.8) או Cloudflare (1.1.1.1)
4. הפעילו מחדש מכשירים לאחר השינוי

## שיקולי ספק אינטרנט

### חניקת מהירות

חלק מספקי האינטרנט חונקים תעבורת סטרימינג:
1. הריצו בדיקת מהירות לשרתי Bayit+
2. השוו עם בדיקת מהירות כללית
3. פנו לספק אם יש הבדל משמעותי
4. שקלו ספק שלא חונק

### מגבלות נתונים

היו מודעים למגבלות נתונים:
1. בדקו מגבלת הנתונים אצל הספק
2. עקבו אחר שימוש חודשי
3. הורידו איכות לחיסכון בנתונים
4. שקלו תוכניות ללא הגבלה לסטרימינג כבד

## פניה לתמיכה

לעזרה הקשורה לרשת:
1. **באפליקציה:** הגדרות > עזרה > בעיית רשת
2. הריצו אבחון רשת לפני פנייה
3. כללו: שם ספק אינטרנט, תוצאות בדיקת מהירות, דגם נתב
