# ניהול קמפיינים

צרו ונהלו קמפייני אימייל והתראות Push כדי לעורר מעורבות בקרב קהל Bayit+. קמפיינים עוזרים לקדם תוכן חדש, לעורר מחדש משתמשים לא פעילים ולתקשר עדכונים חשובים.

## סוגי קמפיינים

**קמפייני אימייל**: אימיילי HTML עשירים לתקשורת מפורטת הכוללת הודעות על יציאות חדשות, ניוזלטרים ועדכוני חשבון.

**התראות Push**: הודעות קצרות ומיידיות הנשלחות לאפליקציות מובייל וטלוויזיה לקידומי תוכן רגישים לזמן.

**הודעות באפליקציה**: באנרים ומודלים המוצגים בתוך האפליקציה לקידומים הקשריים.

## יצירת קמפיין

1. נווטו אל **קמפיינים** > **צור חדש**
2. בחרו את סוג הקמפיין
3. הגדירו את קהל היעד שלכם
4. עצבו את תוכן ההודעה
5. תזמנו את מועד המשלוח
6. סקרו והשיקו

## מיקוד קהל

פלחו את הקהל שלכם באמצעות קריטריונים אלה:

| קריטריון | דוגמאות |
|----------|---------|
| פעילות | פעילים, לא פעילים, משתמשים חדשים |
| העדפות | תחומי עניין בז'אנרים, שפה |
| מנוי | חינמי, פרימיום, ניסיון |
| התנהגות | השלימו סדרה, נטשו |
| דמוגרפיה | טווח גילאים, מיקום |

שלבו מספר קריטריונים למיקוד מדויק.

## התאמה אישית של תוכן

שדות דינמיים מתאימים אישית הודעות אוטומטית:

- `{{user.name}}`: שם התצוגה של הנמען
- `{{content.title}}`: כותרת תוכן מומלץ
- `{{content.image}}`: תמונת פוסטר התוכן
- `{{profile.name}}`: שם הפרופיל הפעיל

## בדיקות A/B

בדקו וריאציות לאופטימיזציה של ביצועים:

1. צרו שתי וריאציות הודעה או יותר
2. הגדירו את אחוז קהל הבדיקה
3. בחרו את מדד ההצלחה (פתיחות, לחיצות, המרות)
4. השיקו את הבדיקה
5. סקרו תוצאות ופרסו את המנצח

## אנליטיקת קמפיינים

עקבו אחר ביצועי קמפיין עם מדדים מפורטים:

- שיעור משלוח וסטטיסטיקות חזרה
- שיעורי פתיחה ושיעורי לחיצה
- מעקב המרות לצפיות בתוכן
- שיעורי הסרה מרשימת תפוצה

## תבניות קמפיין

שמרו עיצובים בשימוש תכוף כתבניות:

1. עצבו את הקמפיין שלכם
2. לחצו על **שמור כתבנית**
3. תנו שם וקטגוריה לתבנית
4. השתמשו מחדש לקמפיינים עתידיים

תבניות שומרות על מיתוג עקבי בכל התקשורת.
