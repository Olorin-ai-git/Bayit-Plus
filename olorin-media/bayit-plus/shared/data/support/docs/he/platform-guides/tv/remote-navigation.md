# מדריך ניווט בשלט

למדו כיצד לנווט ב-Bayit+ בטלוויזיה באמצעות השלט לחוויית צפייה חלקה.

## ניווט D-Pad

משטח הכיוונים (D-pad) הוא כלי הניווט העיקרי בטלוויזיה.

### תנועה בסיסית
- **למעלה/למטה:** מעבר בין שורות ופריטי תפריט
- **שמאלה/ימינה:** גלישה בקרוסלות תוכן
- **OK/Select:** בחירת פריט מודגש או אישור פעולה
- **Back:** חזרה למסך הקודם או סגירת תפריט

### בדפדפן התוכן
1. השתמשו בלמעלה/למטה למעבר בין שורות תוכן
2. השתמשו בשמאלה/ימינה לגלילה בפריטים בשורה
3. לחצו OK לצפייה בפרטי תוכן
4. לחצו OK שוב להתחלת הפעלה

## פקדי הפעלה

### בזמן הפעלת וידאו
| כפתור | פעולה |
|--------|-------|
| OK | הצגה/הסתרה של פקדי הפעלה |
| Play/Pause | מעבר בין הפעלה להשהייה |
| שמאלה | אחורה 10 שניות |
| ימינה | קדימה 10 שניות |
| למעלה | הצגת מידע על הווידאו |
| למטה | גישה להגדרות אודיו וכתוביות |
| Back | יציאה למסך הגלישה |

### דילוג
1. לחצו OK להצגת סרגל ההתקדמות
2. השתמשו בשמאלה/ימינה לגלישה בציר הזמן
3. לחצו OK להמשך מהמיקום הנבחר

## שליטה קולית

### Apple TV (Siri)
החזיקו את כפתור Siri ואמרו את הפקודה:
- "קדימה מהיר 5 דקות"
- "מה הם אמרו?" (מריץ אחורה 15 שניות עם כתוביות)
- "הפעל משהו לילדים"

### Android TV (Google Assistant)
לחצו על כפתור המיקרופון ואמרו:
- "דלג על הפתיחה"
- "הראה לי דוקומנטריות"
- "הפעל כתוביות בעברית"

## ניווט בתפריטים

### תפריט ראשי
- השתמשו ב-D-pad להדגשת קטגוריות תפריט
- לחצו OK להיכנס לקטגוריה
- לחצו Back לחזרה לתפריט הראשי

### תפריט הגדרות
1. נווטו לסמל הפרופיל שלכם
2. לחצו למטה לגישה להגדרות
3. השתמשו ב-D-pad לניווט באפשרויות
4. לחצו OK למעבר או לבחירת פריטים

## טיפים לניווט קל

- החזיקו שמאלה/ימינה לגלילה רציפה בשורות ארוכות
- השתמשו בכפתורי מספרים (אם זמינים) לקפיצה למיקומים ספציפיים בזמן הפעלה
- לחיצה ארוכה על Back מחזירה ישירות למסך הבית בחלק מהמכשירים
- הפעילו "הפחת תנועה" בהגדרות אם אנימציות מקשות על הניווט

## נגישות

- תמיכה בקוראי מסך זמינה למשתמשים עם לקות ראייה
- ניתן להפעיל מצב ניגודיות גבוהה בהגדרות תצוגה
- תיאורי אודיו זמינים לתוכן נבחר
