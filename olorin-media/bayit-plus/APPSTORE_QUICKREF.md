## VERSION MANAGEMENT

### When Incrementing Versions:
- **Patch (1.0.X):** Bug fixes, minor improvements
- **Minor (1.X.0):** New features, significant improvements
- **Major (X.0.0):** Major overhaul, complete redesign

### Version Code/Build Number:
- **iOS:** CFBundleVersion (auto-increment)
- **Android:** versionCode (must increment each release)
- **Format:** Always increment by 1, never use same number twice
